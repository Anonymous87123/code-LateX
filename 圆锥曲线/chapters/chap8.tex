\chapter{基本空间与元素}

射影几何是一门常常在中学课堂之外才被正式介绍的几何学，但它所揭示的思想，实际上一直潜伏在解析几何、圆锥曲线、透视图法以及大量“定点定线”问题的背后。许多在欧氏几何视角下看来彼此分离的对象，例如平行线、无穷远方向、椭圆、抛物线、双曲线、切线结构、焦点准线关系、调和点列与完全四边形，在射影几何中都能被放入一个统一且高度对称的框架里。

本讲义的写作目标并不是简单堆砌结论，而是希望把一整套逻辑链条梳理清楚。我们会看到：在仿射空间中加入无穷远元素之后，原本“不相交”的平行线将获得交点；把点看成齐次坐标的比例类之后，有限点与无穷远点将被统一对待；一旦把“共线”“共点”“交比”看成核心不变量，我们就能理解为什么射影变换会成为研究图形本质的自然工具；而当二次曲线被看作射影平面中的二次对象时，极点极线、配极原则、自极三角形以及焦点准线互为极点极线等现象，就不再是零散技巧，而会呈现出一种几乎不可避免的内在统一。

对于许多读者而言，最先接触射影几何，往往是因为极点极线在圆锥曲线题中十分高效。确实如此：它能迅速识别某些“过定点”“点在定直线”“定值”结论的深层背景，帮助我们看穿题目结构。但是，若只把它当成一项解题术，就会错过它真正的价值。极点极线之所以有力，并不是因为它神秘，而是因为它背后站着更普遍的理论：交比、调和共轭、完全四边形、对偶原理、二次型与射影变换。理解这些，才能真正理解为什么“某点在某线的极线上”会自动推出“另一点在前一点的极线上”，也才能理解为什么许多圆锥曲线题的答案会呈现如此整齐的点线对称性。

本讲义将尽量在以下三种视角之间来回切换，并保持彼此贯通：

\begin{enumerate}
    \item \textbf{直观几何视角}：从平行线、透视、无穷远点与投影出发，建立对射影空间的形象理解；
    \item \textbf{代数坐标视角}：用齐次坐标、矩阵、线性变换与二次型，把射影几何写成可以严密操作的代数体系；
    \item \textbf{圆锥曲线应用视角}：把极点极线、配极原则、自极三角形、焦点准线等问题置于统一框架下理解。
\end{enumerate}

需要提醒的是：射影几何在严格展开时，会自然触及高等几何、线性代数、射影代数几何甚至拓扑的一些思想。因此本文采取的策略是：在保证结构完整的前提下，优先把“概念为什么自然”“结论为什么统一”“公式为什么长成这样”讲透。对于需要更高阶背景才能完全形式化的内容，我们会尽量给出几何解释与代数骨架，让读者在中间层次上也能建立起可靠理解。

最后，还想特别强调一点：学射影几何，不应把它理解成“另起炉灶的一门怪学问”，而应把它看作是对已有解析几何知识的一次升维。你会发现，原来熟悉的椭圆、双曲线、抛物线只是二次曲线在仿射坐标中的不同面貌；原来熟悉的弦、切线、焦点、准线，也都能在射影语言中得到更统一的解释；原来看似偶然的“定点”“定线”结论，往往只是配极原则的投影。若能在这些地方建立起贯通感，那么本讲义的使命也就完成了一半。

\begin{conclusion}{}{}
射影几何最值得学习的，不是若干技巧，而是一种组织几何事实的方式：\\
把无穷远元素补进来，把点与直线放到对称地位，把交比视作核心不变量，把二次曲线看作点线对偶结构的载体。这样一来，许多原本分散的结论会自动汇成一条主线。
\end{conclusion}

\section{为什么要从仿射空间走向射影空间}
在普通解析几何中，我们研究的是仿射空间。例如平面直角坐标系所对应的就是二维仿射空间 $\A^2(\R)$。在这个空间里，我们熟悉的很多结论都十分自然：两点确定一条直线，直线可以写成一次方程，圆锥曲线可以写成二次方程。然而，一旦进入更深一点的几何问题，就会遇到一个极其典型的障碍：\textbf{平行}。

在仿射平面中，两条不重合的平行直线没有交点。这个现象在初等几何里看起来非常正常，但从结构上看，它意味着平面中的“直线相交”这件事并不封闭。也就是说，“两条不同直线相交于一点”这一命题不是无条件成立的，它必须额外排除平行情形。许多投影问题、透视问题、透视图法以及圆锥曲线在无穷远处的行为，都因此显得不够统一。

射影几何的第一个核心思想就是：\textbf{把平行线的交点也补上去。} 更准确地说，对于仿射平面中每一个方向，我们补加一个无穷远点，使得所有该方向上的平行直线都经过这个点。于是，不同方向对应不同的无穷远点，而所有无穷远点又共同组成一条新的直线，称为无穷远直线。

这样一来，原来的仿射平面便被“封闭”为射影平面。此时：

\begin{itemize}
    \item 任意两条不同直线都相交于唯一一点；
    \item 平行不再是特殊情形，而只是“交点落在无穷远直线上”的普通情形；
    \item 椭圆、抛物线、双曲线在无穷远处的区别，变成了它们与无穷远直线交点个数的区别。
\end{itemize}

从这个意义上讲，射影空间不是在仿射空间之外另造一个世界，而是在仿射空间的基础上补足原本缺失的极限结构，使整个几何体系变得更封闭、更统一、更对称。

\begin{definition}{}{}
设 $\F$ 是一个域。考虑向量空间 $\F^{n+1}$ 中所有非零向量的集合
\[
\F^{n+1}\setminus\{ 0\}.
\]
在其上定义等价关系
\[
(x_0,x_1,\dots,x_n)\sim (y_0,y_1,\dots,y_n)
\quad \Longleftrightarrow \quad
\exists \lambda\in \F^\times,\ (y_0,\dots,y_n)=\lambda(x_0,\dots,x_n).
\]
这些等价类所构成的集合称为 \textbf{$n$维射影空间}，记作
\[
\PP^n(\F).
\]
\end{definition}

这个定义的本质是：射影空间中的一个点，并不是向量空间里的某个具体向量，而是\textbf{一条过原点的直线}。因为同一条直线上的所有非零向量都只差一个非零倍数，它们在射影空间中应被视为同一点。

\section{射影空间、射影平面与射影直线}

\subsection{实射影空间与复射影空间}
当底层域 $\F=\R$ 时，得到的是\textbf{实射影空间}：
\[
\RP^n=\PP^n(\R).
\]
当底层域 $\F=\C$ 时，得到的是\textbf{复射影空间}：
\[
\CP^n=\PP^n(\C).
\]

对大多数解析几何与中学背景下的射影几何问题而言，实射影空间已经足够。但从理论完整性与代数几何角度看，复射影空间往往更自然，因为许多在实数域下“看不见”的交点，在复域中会自动补齐。例如两条平面二次曲线的交点总数在复射影平面中会满足非常整齐的代数规律。

\subsection{射影平面}
\begin{definition}{}{}
二维射影空间 $\PP^2(\F)$ 称为 \textbf{射影平面}。特别地，$\RP^2$ 称为实射影平面，$\CP^2$ 称为复射影平面。
\end{definition}

射影平面是本文最主要的研究对象。它可以被理解为：

\[
\text{射影平面}=\text{仿射平面}+\text{一条无穷远直线}.
\]

在仿射平面中，每个方向补上一个无穷远点；所有方向对应的无穷远点合在一起，就得到无穷远直线。于是，原来的仿射平面被自然嵌入射影平面中，而射影平面又比仿射平面多出一整条新的直线。

\subsection{射影直线}
\begin{definition}{}{}
一维射影空间 $\PP^1(\F)$ 称为 \textbf{射影直线}。
\end{definition}

从最直观的角度看，射影直线就是“普通直线再加上一个无穷远点”。对于实数域，可以粗略写作
\[
\RP^1 \cong \R\cup\{\infty\}.
\]
这并不是严格意义上的集合恒等，而是一种非常有帮助的直观模型：普通数轴向左右延伸到无穷远，最终在射影直线中会闭合到同一个无穷远点上。

因此，射影直线本质上已经不再是一条“有两个方向无穷延伸的开直线”，而是一个封闭的一维射影对象。后文中所有关于点列、交比、调和点列、一维射影变换的理论，都将首先发生在射影直线上。

\section{点、直线、无穷远点与无穷远直线}

\subsection{点：射影空间中的基本元素}
\begin{definition}{}{}
射影空间中的\textbf{点}是向量空间中一条过原点的一维线性子空间，常用一组不全为零的比例坐标
\[
(x_0:x_1:\cdots:x_n)
\]
来表示。
\end{definition}

这里的冒号不是普通有序数组的符号，而是在强调：这些坐标只在比例意义下确定。例如
\[
(1:2:3)=(2:4:6)=(-1:-2:-3).
\]
它们都表示同一个射影点。

这正是射影坐标系统最重要的思想：\textbf{点不是由一个固定坐标表示，而是由一整类成比例的坐标共同表示。}

\subsection{齐次坐标}
\begin{definition}{}{}
设 $P\in \PP^n(\F)$。若 $P$ 由非零向量 $(x_0,x_1,\dots,x_n)$ 所对应的一维子空间表示，则称
\[
(x_0:x_1:\cdots:x_n)
\]
为点 $P$ 的 \textbf{齐次坐标}。
\end{definition}

齐次坐标的最大优势在于：有限点与无穷远点被完全统一了。以实射影平面 $\RP^2$ 为例：

\begin{itemize}
    \item 若点的第三坐标 $z\neq 0$，则可化成 $(x:y:1)$ 的形式，对应仿射平面中的有限点 $(x,y)$；
    \item 若点满足 $z=0$，则它位于无穷远直线上，是一个无穷远点。
\end{itemize}

因此，齐次坐标使得我们不必额外为“无穷远点”单独设立一套例外规则，它们只是普通射影点中的一类而已。

\subsection{直线：射影平面中的一维子空间}
在射影平面中，一条直线可以由一个齐次一次方程来表示。

\begin{definition}{}{}
在射影平面 $\PP^2(\F)$ 中，一个不全为零的齐次线性方程
\[
ax+by+cz=0
\]
所确定的点集称为一条 \textbf{直线}。
\end{definition}

从线性代数角度看，这条直线对应于向量空间中的一个二维子空间；从坐标角度看，它是所有满足齐次一次方程的比例类点所组成的集合。

一个极其重要的事实是：在射影平面里，\textbf{点与直线在代数形式上非常对称}。点写成比例类 $(x:y:z)$，直线写成系数比例类 $[a:b:c]$，二者之间的关联关系就是
\[
ax+by+cz=0.
\]
这种对称性正是后文对偶原理的根源。

\subsection{无穷远点}
\begin{definition}{}{}
仿射空间中同一方向的一族平行直线，在射影化之后所共有的交点称为该方向的 \textbf{无穷远点}。
\end{definition}

例如，在仿射平面中，所有斜率为 $k$ 的直线
\[
y=kx+b
\]
两两平行。在射影平面中，它们会交于同一个无穷远点
\[
(1:k:0).
\]
而所有形如 $x=c$ 的竖直直线，也会交于无穷远点
\[
(0:1:0).
\]

这说明无穷远点并不是唯一的，而是“每个方向一个”。方向不同，无穷远点也不同。

\subsection{无穷远直线}
\begin{definition}{}{}
射影平面中所有无穷远点所组成的直线称为 \textbf{无穷远直线}，通常记作 $\ell_\infty$。
\end{definition}

在标准齐次坐标下，无穷远直线通常写作
\[
z=0.
\]

于是，仿射平面恰好可以被看作射影平面去掉无穷远直线以后剩下的部分：
\[
\A^2(\F)\cong \PP^2(\F)\setminus \ell_\infty.
\]

这条公式非常重要。它告诉我们：仿射几何并不是和射影几何完全无关的另一套体系，而只是射影几何中“看不见无穷远直线”的那一部分。

\begin{property}{}{}
在射影平面中，任意两条不同直线都相交于唯一一点；其中，仿射平面中的平行线在射影平面中的交点恰好落在无穷远直线上。
\end{property}

\section{射影坐标与仿射坐标的关系}

\subsection{从仿射坐标到齐次坐标}
在二维情形下，仿射平面中的一个点 $(u,v)$ 可以嵌入射影平面为
\[
(u,v)\longmapsto (u:v:1).
\]
也可以写作 $(1:u:v)$，这取决于采用哪一种约定。本文统一采用
\[
(x,y)\mapsto (x:y:1)
\]
这一写法。

这种嵌入说明：仿射点只是射影点中第三坐标不为零的一类特殊点。而一旦第三坐标为零，就进入无穷远部分。

\subsection{齐次化}
很多仿射方程在射影几何中需要改写成齐次方程，这个过程称为\textbf{齐次化}。例如：

\begin{itemize}
    \item 仿射直线方程
    \[
    ax+by+c=0
    \]
    的齐次形式为
    \[
    ax+by+cz=0.
    \]
    \item 仿射二次曲线方程
    \[
    Ax^2+Bxy+Cy^2+Dx+Ey+F=0
    \]
    的齐次形式为
    \[
    Ax^2+Bxy+Cy^2+Dxz+Eyz+Fz^2=0.
    \]
\end{itemize}

齐次化的好处在于：曲线在无穷远处的行为不再被忽略，而是被统一编码进同一个方程中。正因为如此，椭圆、抛物线、双曲线与无穷远直线的交点情况，才能被自然拿来作为分类标准。

\subsection{射影坐标}
\begin{definition}{}{}
在射影空间中，选定一组参考点或参考标架后所建立的比例坐标系统，称为 \textbf{射影坐标}。
\end{definition}

射影坐标与仿射坐标最大的区别在于：它不依赖于某个绝对原点和长度单位，而只依赖于投影结构和比例结构。因此，它更接近射影几何本身的本质。

从一维情形看，射影直线上只要固定三个互不重合的基准点，就可以把其他点参数化；从二维情形看，射影平面只要选定四个一般位置点，就能建立对应的射影坐标系。

\vspace{0.5em}
\begin{remark}
许多初学者会觉得“射影坐标”和“齐次坐标”是同一个概念。严格来说，齐次坐标强调的是“用比例类表示点”的方式，而射影坐标强调的是“在射影空间中建立坐标系统”的过程。二者密切相关，但侧重点不同。
\end{remark}

\chapter{变换与映射}

\section{射影变换的基本思想}
在欧氏几何中，我们研究刚体运动，因此长度和角度保持不变；在仿射几何中，我们允许更大的变换，于是平行性与某些比值成为核心不变量；而在射影几何中，我们进一步允许中心投影与更一般的射影映射，研究对象便不再是长度和角度，而是共线、共点、交比等更本质的结构。

射影变换的直观来源，就是透视。想象空间中有一个投影中心，把一个平面上的图形投影到另一个平面上。这样的映射通常不会保长度，不会保角，也不会把圆仍然送成圆，但它会保直线性：直线仍映为直线。这正是射影几何的基本精神。

\begin{definition}{}{}
设 $T:\F^{n+1}\to\F^{n+1}$ 是一个非奇异线性变换，则它在射影空间 $\PP^n(\F)$ 上诱导一个映射
\[
[x]\longmapsto [Tx].
\]
由此得到的点到点的一一对应称为 \textbf{射影变换} 或 \textbf{射影映射}。
\end{definition}

这里 $[x]$ 表示向量 $x$ 所对应的比例类。由于如果 $x$ 与 $\lambda x$ 表示同一点，则 $Tx$ 与 $\lambda Tx$ 仍表示同一点，所以这个定义是良定的。

\begin{property}{}{}
射影变换具有如下基本性质：
\begin{enumerate}
    \item 点映点，直线映直线；
    \item 共线点仍共线；
    \item 共点线仍共点；
    \item 交比保持不变；
    \item 二次曲线映为二次曲线。
\end{enumerate}
\end{property}

这些性质告诉我们：射影变换不是“任意扭曲图形”，而是在保持某些深层结构不变的前提下，对图形进行最广泛的线性投影变换。

\section{透视对应}
\begin{definition}{}{}
若两个点列、或两个线束、或一个点列与一个线束之间，通过某个固定中心或某条固定轴建立起对应关系，则称这种对应为 \textbf{透视对应}。
\end{definition}

最常见的情形是：取平面上一点 $O$，用过 $O$ 的一束直线去截两条不同的直线 $l_1,l_2$。这样，$l_1$ 上的每个点都会通过与 $O$ 的连线对应到 $l_2$ 上的一个点。于是，两个点列之间便形成了透视对应。

透视对应之所以重要，是因为它正是射影变换最几何化的来源。许多一维射影几何中的结论，例如交比保持不变、调和结构保持不变、点列与线束之间的对应，都可以通过透视来理解。

\begin{conclusion}{}{}
从本质上说，射影几何中的一维对象（点列与线束）是可以互相投影、互相截影的，而这种“中心投影—截影”机制正是射影不变量产生的来源。
\end{conclusion}

\section{直射变换与对射变换}

\subsection{直射变换}
\begin{definition}{}{}
将点映为点、将直线映为直线，并保持关联关系的射影变换，称为 \textbf{直射变换}。
\end{definition}

通常我们说“射影变换”，默认指的就是这种点到点、线到线的变换。它是平面图形在投影下最自然的变化方式。

\subsection{对射变换}
\begin{definition}{}{}
若一个映射把点映为直线，把直线映为点，并保持“点在直线上”这一关联关系的对偶形式，则称其为 \textbf{对射变换}。
\end{definition}

对射变换在初看时会显得有些抽象，但它实际上是后文极点极线理论的核心语言。因为极点极线正是一种典型的“点 $\leftrightarrow$ 直线”的对应机制。

如果说直射变换是在“同类对象”之间移动，那么对射变换就是在“点世界”和“线世界”之间搭桥。射影几何的对偶原理，就是在这种思想上建立起来的。

\section{对合}
\begin{definition}{}{}
一个变换 $f$ 若满足
\[
f\circ f=\mathrm{id},
\]
则称 $f$ 为一个 \textbf{对合}。
\end{definition}

对合的意思是：做一次变换得到某个像，再做一次同样的变换，就回到原对象。在线性代数中，关于某条直线的对称、关于某个点的中心对称都是对合；在射影几何中，一维射影直线上的许多重要变换也是对合。

对合在射影几何中的特殊意义在于：它常常与“成对出现”的几何结构相关。例如，在一条直线上，把每个点映到它关于某种几何构型的“共轭点”，若这个对应是互逆的，那么它往往就是一个对合。极点极线、二次曲线上的共轭关系、完全四边形诱导出的调和对应，都与对合有非常深刻的联系。

\chapter{不变量与度量}

\section{什么是射影不变量}
几何学从根本上说，就是在某类允许的变换下研究保持不变的对象。不同几何学所研究的不变量不同：

\begin{center}
\begin{tabular}{>{\centering\arraybackslash}m{3cm}>{\centering\arraybackslash}m{4cm}>{\centering\arraybackslash}m{5cm}}
\hline
几何类型 & 允许变换 & 主要不变量 \\
\hline
欧氏几何 & 刚体运动 & 长度、角度 \\
相似几何 & 相似变换 & 角度、长度比 \\
仿射几何 & 仿射变换 & 共线、平行、单比 \\
射影几何 & 射影变换 & 共线、共点、交比 \\
\hline
\end{tabular}
\end{center}

因此，射影几何最核心的数值型不变量不是长度，也不是角度，而是\textbf{交比}。而像“共线”“共点”“点在线上”“线过某点”这样的关联性质，也都属于射影不变量。

\section{交比（复比）}
\begin{definition}{}{}
设直线上有四个有序点 $A,B,C,D$，则它们的 \textbf{交比}（或复比）定义为
\[
(A,B;C,D)=\frac{AC/BC}{AD/BD},
\]
其中线段采用有向线段的意义。
\end{definition}

若用仿射坐标表示，设四点坐标分别为 $a,b,c,d$，则
\[
(A,B;C,D)=\frac{(c-a)(d-b)}{(c-b)(d-a)}.
\]

交比并不是一个偶然拼凑出来的公式，而是一维射影几何中最核心的数值不变量。它的意义在于：在射影变换之下，四个点的交比保持不变。

\begin{theorem}{}{}
在一维射影变换下，四个点的交比保持不变。
\end{theorem}

\begin{myproof}
设射影直线上的变换为
\[
x\longmapsto \frac{ax+b}{cx+d},\qquad ad-bc\neq 0.
\]
把四个点的坐标代入交比公式进行化简，可以验证变换前后的交比完全相同。故交比是射影变换下的不变量。
\end{myproof}

这一定理的意义极大。因为在射影几何中，长度、角度都不是稳定量，而交比却始终保留，所以它成为我们衡量射影结构最可靠的数值工具。

\section{调和共轭与调和点列}
\begin{definition}{}{}
若四个共线点 $A,B,C,D$ 的交比满足
\[
(A,B;C,D)=-1,
\]
则称这四点构成 \textbf{调和点列}，称 $C,D$ 关于 $A,B$ 互为 \textbf{调和共轭}。
\end{definition}

调和点列是交比最重要的特例之一。它之所以格外重要，是因为：

\begin{enumerate}
    \item 它在射影变换下保持不变；
    \item 它会自然出现在完全四边形、极点极线和二次曲线的各种构造中；
    \item 它是“中点”“调和平均”“角平分线”等许多初等几何现象在射影几何中的统一表达。
\end{enumerate}

若用有向线段写，调和点列还可改写为
\[
\frac{AB}{BC}=\frac{AD}{DC}.
\]
进一步还能得到
\[
\frac{1}{AB}+\frac{1}{AD}=\frac{2}{AC},
\]
这也是“调和”一词的来源之一。

\vspace{0.5em}
\begin{property}{}{}
调和点列在透视对应下保持不变；换言之，若一点列是调和点列，则把它从某中心投影到另一条直线后，得到的对应四点仍构成调和点列。
\end{property}

这条性质看似简单，实则是后续调和线束、完全四边形调和性质、极点极线几何定义以及配极原则的底层支柱。

\section{射影度量}
纯粹的射影几何本身并不讨论长度和角度，因为这些量在一般射影变换下都不保持。但这并不意味着射影几何与度量几何毫无关系。恰恰相反，通过在射影空间中额外引入一个固定的“绝对形”，我们可以在射影框架下重新定义距离和角度，这便形成了所谓的\textbf{射影度量}。

\begin{definition}{}{}
在射影空间中，若额外指定一个固定的二次曲线或二次曲面作为 \textbf{绝对形}，并利用交比来定义点间距离或线间角度，则由此得到的度量结构称为 \textbf{射影度量}。
\end{definition}

这一定义背后的思想由克莱因系统化提出：欧氏几何、双曲几何、椭圆几何都可以看作射影几何加上不同绝对形之后得到的子几何。

例如，在平面双曲几何的克莱因模型中，绝对形是一条实二次曲线（通常取单位圆）。圆内部的点被看作双曲点，圆弦被看作双曲直线，而两点之间的双曲距离可由它们与边界圆的交点构成的交比来定义。这说明，交比不仅是射影不变量，而且还能进一步承担“距离”的角色。

对欧氏几何而言，情况略微抽象一些：其绝对形不再是实可见的圆，而是在无穷远直线上一对共轭虚点，也就是所谓的\textbf{虚圆点}。正是借助它们，欧氏角度也可以被重新解释为某种交比的函数。

\begin{remark}
因此，射影几何并不是与欧氏几何、非欧几何并列的“另一种几何”，而更像是一个总框架。许多几何体系都可以被看成“射影几何 + 某种额外绝对结构”。
\end{remark}

\chapter{二次曲线与配极}

\section{二次曲线的射影观点}
在中学解析几何中，我们习惯把椭圆、抛物线、双曲线分别看成三类不同曲线。但在射影几何中，这种分裂的观点会被统一。因为在射影平面里，它们全都是\textbf{二次曲线}，区别只在于它们与无穷远直线的交点情况不同。

\begin{definition}{}{}
在射影平面 $\PP^2(\F)$ 中，一个二次齐次方程
\[
Q(x,y,z)=0
\]
的零点集称为一条 \textbf{二次曲线}。
\end{definition}

一般地，二次曲线可写成
\[
Ax^2+Bxy+Cy^2+Dxz+Eyz+Fz^2=0.
\]

如果记
\[
 X=
\begin{pmatrix}
x\\ y\\ z
\end{pmatrix},
\]
那么这条方程还可以写成矩阵形式
\[
 X^{\mathrm T}M X=0,
\]
其中 $M$ 是一个对称矩阵。

这种写法有两个巨大优点：

\begin{enumerate}
    \item 它把几何问题转化为二次型与线性代数问题；
    \item 它为极点极线的定义提供了最自然的代数平台。
\end{enumerate}

\section{二次曲线的射影分类}
射影几何中最漂亮的一点，在于椭圆、抛物线、双曲线的区别不再是“形状看起来不一样”，而是“它们与无穷远直线相交方式不同”。

\begin{definition}{}{}
设一条实射影平面中的非退化二次曲线与无穷远直线 $\ell_\infty$ 的实交点情况如下：
\begin{enumerate}
    \item 若无实交点，则称为 \textbf{椭圆型}；
    \item 若有一个重实交点（相切），则称为 \textbf{抛物型}；
    \item 若有两个不同实交点，则称为 \textbf{双曲型}。
\end{enumerate}
\end{definition}

下面分别来看三种经典曲线的齐次化结果。

\subsection{椭圆型}
椭圆
\[
\frac{x^2}{a^2}+\frac{y^2}{b^2}=1
\]
齐次化后为
\[
\frac{x^2}{a^2}+\frac{y^2}{b^2}=z^2.
\]
令 $z=0$，得到
\[
\frac{x^2}{a^2}+\frac{y^2}{b^2}=0.
\]
在实数域中，这个方程没有非零解，因此椭圆与无穷远直线没有实交点，故它是椭圆型二次曲线。

\subsection{抛物型}
抛物线
\[
y^2=2px
\]
齐次化后为
\[
y^2=2pxz.
\]
令 $z=0$，得到
\[
y^2=0.
\]
故在无穷远直线上只有一个重交点 $(1:0:0)$，因此抛物线是抛物型二次曲线。

\subsection{双曲型}
双曲线
\[
\frac{x^2}{a^2}-\frac{y^2}{b^2}=1
\]
齐次化后为
\[
\frac{x^2}{a^2}-\frac{y^2}{b^2}=z^2.
\]
令 $z=0$，得到
\[
\frac{x^2}{a^2}-\frac{y^2}{b^2}=0,
\]
也即
\[
\frac{x}{a}=\pm \frac{y}{b}.
\]
所以双曲线与无穷远直线有两个实交点，这两个无穷远点正对应双曲线两条渐近线的方向。

\begin{conclusion}{}{}
在实射影平面中，椭圆、抛物线、双曲线三者的本质区别，仅仅在于它们与无穷远直线的交点数分别为 $0,1,2$。
\end{conclusion}

这一观点非常重要。它告诉我们，在更高层次上，椭圆、抛物线、双曲线并不是三类互不相干的曲线，而是同一个射影二次对象在实仿射部分中的三种不同表现。

\section{极点与极线}

\subsection{代数定义}
极点极线是二次曲线理论中最核心的结构之一。它之所以重要，是因为它建立了一种天然的“点—直线”对应，而这种对应恰好是后文对偶原理和配极原则的具体实现。

设二次曲线由对称矩阵 $M$ 给出：
\[
 X^{\mathrm T}M X=0.
\]
若取一点
\[
P=[ p],
\]
那么由双线性形式
\[
 p^{\mathrm T}M X=0
\]
所给出的直线，便定义为点 $P$ 的极线。

\begin{definition}{}{}
设非退化二次曲线
\[
Q( X)= X^{\mathrm T}M X=0.
\]
对任一点 $P=[ p]$，由方程
\[
 p^{\mathrm T}M X=0
\]
所确定的直线称为点 $P$ 关于该二次曲线的 \textbf{极线}；反过来，给定一条直线，也可唯一对应一个 \textbf{极点}。
\end{definition}

这一定义乍看抽象，但它其实正是“切线方程形式推广到任意点”的自然结果。

\subsection{仿射形式下的极线方程}
若仿射二次曲线写成
\[
Ax^2+2Bxy+Cy^2+2Dx+2Ey+F=0,
\]
则点 $P(x_0,y_0)$ 的极线方程是
\[
Axx_0+B(xy_0+x_0y)+Cyy_0+D(x+x_0)+E(y+y_0)+F=0.
\]

这就是常说的“代一半”公式。也就是说：

\begin{itemize}
    \item $x^2$ 换成 $xx_0$；
    \item $y^2$ 换成 $yy_0$；
    \item $2xy$ 换成 $xy_0+x_0y$；
    \item $2x$ 换成 $x+x_0$；
    \item $2y$ 换成 $y+y_0$；
    \item 常数项保持不变。
\end{itemize}

\begin{criterion}{}{}
对于一般二次曲线
\[
Ax^2+2Bxy+Cy^2+2Dx+2Ey+F=0,
\]
点 $P(x_0,y_0)$ 的极线可用“代一半”快速写成
\[
Axx_0+B(xy_0+x_0y)+Cyy_0+D(x+x_0)+E(y+y_0)+F=0.
\]
\end{criterion}

\subsection{几种常见二次曲线的极线公式}
\subsubsection{圆}
对于圆
\[
x^2+y^2=r^2,
\]
点 $P(x_0,y_0)$ 的极线为
\[
x_0x+y_0y=r^2.
\]

\subsubsection{椭圆}
对于椭圆
\[
\frac{x^2}{a^2}+\frac{y^2}{b^2}=1,
\]
点 $P(x_0,y_0)$ 的极线为
\[
\frac{x_0x}{a^2}+\frac{y_0y}{b^2}=1.
\]

\subsubsection{双曲线}
对于双曲线
\[
\frac{x^2}{a^2}-\frac{y^2}{b^2}=1,
\]
点 $P(x_0,y_0)$ 的极线为
\[
\frac{x_0x}{a^2}-\frac{y_0y}{b^2}=1.
\]

\subsubsection{抛物线}
对于抛物线
\[
y^2=2px,
\]
点 $P(x_0,y_0)$ 的极线为
\[
y_0y=p(x+x_0).
\]

\begin{example}{}{}
求椭圆
\[
\frac{x^2}{9}+\frac{y^2}{4}=1
\]
上点 $P(3\cos\theta,2\sin\theta)$ 处的切线方程。
\end{example}

\begin{solution}
由于点 $P$ 在椭圆上，极线退化为该点切线。直接由极线公式可得
\[
\frac{(3\cos\theta)x}{9}+\frac{(2\sin\theta)y}{4}=1,
\]
化简为
\[
\frac{x\cos\theta}{3}+\frac{y\sin\theta}{2}=1.
\]
\end{solution}

\subsection{极线的几何意义}
极线的几何意义取决于极点与二次曲线的位置关系：

\begin{enumerate}
    \item 若极点在曲线外，则极线是过该点所作两条切线的切点连线；
    \item 若极点在曲线上，则极线退化为该点处的切线；
    \item 若极点在曲线内，则极线通常与曲线没有实交点，但射影意义下仍完全成立。
\end{enumerate}

所以极线不是“切线的小修小补”，而是把切线概念推广到平面任意点后的统一结果。

\section{配极原则}
极点极线理论中最核心的结论，就是配极原则。它既有极其简洁的代数证明，也有深刻的几何含义。

\begin{theorem}{}{}
设 $Q=0$ 是一条非退化二次曲线。若点 $P$ 的极线通过点 $Q$，则点 $Q$ 的极线也通过点 $P$。
\end{theorem}

\begin{myproof}
设 $P=[ p]$，$Q=[ q]$。由定义，点 $Q$ 在点 $P$ 的极线上，当且仅当
\[
 p^{\mathrm T}M q=0.
\]
由于矩阵 $M$ 对称，所以
\[
 p^{\mathrm T}M q= q^{\mathrm T}M p.
\]
这就说明点 $P$ 也在点 $Q$ 的极线上。故命题成立。
\end{myproof}

\begin{conclusion}{}{}
“$P$ 在 $Q$ 的极线上”与“$Q$ 在 $P$ 的极线上”是完全对称的命题。极点极线对应不是单向的，而是一种自反的点线对偶关系。
\end{conclusion}

\subsection{配极原则的一个重要推论}
如果一点 $P$ 在某条固定直线 $d$ 上运动，那么 $P$ 的极线会恒过某个固定点 $D$。这个点正是直线 $d$ 的极点。

这条结论对于大量“证明某直线恒过定点”的问题极其有力。因为它告诉我们：

\begin{quote}
只要识别出一个动点始终落在定直线上，那么它对应的极线就自动过定点。
\end{quote}

同理，若一条动直线始终过某个定点，那么它们的极点就会落在某条定直线上。这正是点线对偶性的直接体现。

\section{焦点与准线互为极点极线}
这一结论在中学解析几何中往往不会被正式介绍，但在射影几何看来却非常自然。

\begin{theorem}{}{}
对于标准椭圆、双曲线与抛物线，其焦点与对应准线互为极点极线。
\end{theorem}

\begin{myproof}
以椭圆
\[
\frac{x^2}{a^2}+\frac{y^2}{b^2}=1,\qquad c^2=a^2-b^2
\]
为例。右焦点为
\[
F(c,0),
\]
右准线为
\[
x=\frac{a^2}{c}.
\]
把准线写成极线标准形式：
\[
\frac{cx}{a^2}+\frac{0\cdot y}{b^2}=1.
\]
于是可直接读出其极点为 $(c,0)$，即焦点 $F$。故焦点与准线互为极点极线。
\end{myproof}

这一定理说明，焦点准线结构并不是与极点极线完全独立的另一套理论，它只是后者在某些标准坐标下的显性表现。

\section{自极三角形}
\begin{definition}{}{}
若三角形 $\triangle ABC$ 满足：点 $A$ 的极线是边 $BC$，点 $B$ 的极线是边 $CA$，点 $C$ 的极线是边 $AB$，则称 $\triangle ABC$ 为该二次曲线的 \textbf{自极三角形}。
\end{definition}

自极三角形之所以重要，在于它把三组点线对应闭合成一个完整结构。对每一个顶点而言，它的对边都是自己的极线；反过来，对每一条边而言，它的极点又是对应的顶点。

\begin{property}{}{}
自极三角形中的三个顶点两两共轭，三条边也两两构成对应的共轭线结构。
\end{property}

在理论上，自极三角形常用于：

\begin{enumerate}
    \item 化简二次曲线方程；
    \item 构造极点极线；
    \item 组织完全四边形与二次曲线的关系；
    \item 在解析几何中选择更自然的坐标系。
\end{enumerate}

\chapter{对偶原理}

\section{点与直线的对称地位}
射影平面与欧氏平面最显著的区别之一，在于点和直线几乎拥有完全对称的地位。在欧氏几何叙述中，我们往往习惯把点当成基础对象，认为直线是“由点连出来的”。但在射影几何中，这种偏见会被打破：点与直线都可以用齐次对象表示，而且“点在线上”这一关系本身就是点与直线的双向配合。

\begin{definition}{}{}
在射影平面中，将命题中的“点”与“直线”互换，“共线”与“共点”互换，“连接两点”与“求两线交点”互换，若所得命题仍然成立，则称这一现象为 \textbf{对偶原理}。
\end{definition}

对偶原理并不是一种巧合，而是射影平面代数结构的直接反映。因为：

\begin{itemize}
    \item 点可写为比例类 $(x:y:z)$；
    \item 直线可写为比例类 $[a:b:c]$；
    \item 点在直线上当且仅当
    \[
    ax+by+cz=0.
    \]
\end{itemize}

这种对称性意味着：点与直线分别属于彼此对偶的空间，而几何命题往往也因此自动成对出现。

\section{典型对偶关系}
下面列出射影平面中最常见的几组对偶对象：

\begin{center}
\begin{tabular}{>{\centering\arraybackslash}m{4cm}>{\centering\arraybackslash}m{4cm}}
\hline
原对象 & 对偶对象 \\
\hline
点 & 直线 \\
共线 & 共点 \\
连接两点成线 & 两线相交得点 \\
点列 & 线束 \\
内接多边形 & 外切多边形 \\
二次点列 & 二级曲线（切线包络） \\
\hline
\end{tabular}
\end{center}

\begin{conclusion}{}{}
在射影几何中，一个纯粹由点、直线、共点、共线和关联关系叙述的命题，往往都会自动伴随一个对偶命题。
\end{conclusion}

\section{德萨格定理的自对偶性}
射影几何中的很多经典定理都展现出对偶美感，其中最著名的就是德萨格定理。

\begin{theorem}{}{}
若两个三角形对应顶点连线共点，则对应边交点共线。
\end{theorem}

把其中的“点”和“直线”互换，“共点”和“共线”互换，就得到：

\begin{theorem}{}{}
若两个三角形对应边交点共线，则对应顶点连线共点。
\end{theorem}

这正是德萨格定理的逆命题，因此德萨格定理呈现出鲜明的自对偶性。它不仅说明某一条具体结论成立，更说明射影平面内部存在一种全局的点线镜像对称。

\section{对偶图形}
\begin{definition}{}{}
在射影平面中，通过点线互换而得到的对应图形称为 \textbf{对偶图形}。
\end{definition}

例如：

\begin{itemize}
    \item 一条直线上的点集构成点列，它的对偶对象是过某点的直线集，即线束；
    \item 一条非退化二次曲线上的点集，其对偶对象是这条曲线的全体切线所形成的包络，也是一条二次对象。
\end{itemize}

特别地，极点极线正是“点图形”和“线图形”之间最具体的一种对偶对应。它不是抽象的口号，而是一套可以写成方程、可以直接计算、可以在题目中高效应用的对偶机制。

\chapter{几何结构：点列、线束与二次点列}

\section{点列}
\begin{definition}{}{}
一条直线上的所有点组成的集合称为 \textbf{点列}。
\end{definition}

点列并不只是“若干个排成一行的点”，而是一个完整的一维射影对象。实际上，一条射影直线本身就是一个点列。因此，一维射影几何的全部内容——交比、调和点列、对合、射影变换——都可以在点列上展开。

在学习射影几何时，一个非常重要的观念是：不要把点列看成“二维平面里的特殊情况”，而应把它看成一类本身就十分丰富的几何对象。

\section{线束}
\begin{definition}{}{}
过同一点的所有直线组成的集合称为 \textbf{线束}。
\end{definition}

线束是点列的对偶对象。它也构成一个一维射影空间，因为每一条过定点的直线都可由一个方向参数来刻画。故点列上的交比理论、调和理论、对合理论，都可以转译到线束上去。

例如，若四条共点直线 $a,b,c,d$ 的截线像构成调和点列，那么我们就说这四条线构成调和线束。

\section{点列与线束的透视关系}
一点列可以通过连接到某个中心，变成一个线束；一线束又可以通过与某条截线相交，变成一个点列。因此，点列与线束之间天然存在往返通道。

这种“投影—截影”机制是射影几何非常根本的思想。很多一维结论之所以能在点列与线束之间自由切换，正是因为它们本质上都生活在同一个射影结构中。

\section{二次点列}
\begin{definition}{}{}
二次曲线上的点集，从射影结构角度看，称为 \textbf{二次点列}。
\end{definition}

虽然二次点列中的点不再都落在同一条直线上，但它依然拥有类似一维射影几何的结构。特别是，一条非退化二次曲线可以通过适当参数化与射影直线建立对应，因此二次点列上的很多概念都可以看作普通点列概念的推广。

例如，可以把二次曲线上四点的交比定义为：任选曲线上一点 $P$，考虑线束
\[
(PA,PB;PC,PD),
\]
并把这个值作为四点 $A,B,C,D$ 的交比。由于它与点 $P$ 的选择无关，这个定义是良定的。

\begin{property}{}{}
二次曲线上的四点交比可以借助任何一点对这四点所引射线的交比来定义，因此二次点列同样具有一维射影几何中的交比结构。
\end{property}

这说明，二次曲线并不仅仅是一条“弯曲的图形”，而是某种射影意义下的“一维对象”。

\section{射影直线上的复比}
在射影直线 $\PP^1(\F)$ 上，若四个点的齐次坐标分别为
\[
P_1=(u_1:v_1),\quad
P_2=(u_2:v_2),\quad
P_3=(u_3:v_3),\quad
P_4=(u_4:v_4),
\]
则它们的交比可以写成
\[
(P_1,P_2;P_3,P_4)
=
\frac{
\begin{vmatrix}
u_3 & v_3\\
u_1 & v_1
\end{vmatrix}
\begin{vmatrix}
u_4 & v_4\\
u_2 & v_2
\end{vmatrix}
}{
\begin{vmatrix}
u_3 & v_3\\
u_2 & v_2
\end{vmatrix}
\begin{vmatrix}
u_4 & v_4\\
u_1 & v_1
\end{vmatrix}
}.
\]

当取仿射坐标 $x_i=u_i/v_i$ 时，这个式子就退化成熟悉的
\[
(P_1,P_2;P_3,P_4)
=
\frac{(x_3-x_1)(x_4-x_2)}{(x_3-x_2)(x_4-x_1)}.
\]

因此，复比并不是某种偶然依赖长度的公式，而是齐次坐标意义下最自然的一维射影不变量。

\chapter{著名定理}

\section{德萨格定理}
\begin{theorem}{}{}
设有两个三角形
\[
\triangle ABC,\qquad \triangle A'B'C'.
\]
若对应顶点连线
\[
AA',\ BB',\ CC'
\]
共点，则对应边交点
\[
AB\cap A'B',\quad BC\cap B'C',\quad CA\cap C'A'
\]
共线。
\end{theorem}

德萨格定理是射影几何最经典的基本定理之一。它揭示了“共点透视”与“共线透视”之间的深刻联系。其几何背景非常直观：若两个三角形位于空间中的不同平面，并且都是由同一个点向这两个平面投影得到，那么它们所在平面的交线恰好就是对应边交点所在的那条直线。

\section{帕普斯定理}
\begin{theorem}{}{}
设两条直线 $\ell,m$ 上分别取三点
\[
A,B,C\in \ell,\qquad A',B',C'\in m.
\]
令
\[
X=AB'\cap A'B,\qquad
Y=AC'\cap A'C,\qquad
Z=BC'\cap B'C.
\]
则三点 $X,Y,Z$ 共线。
\end{theorem}

帕普斯定理是早期射影几何中最重要的点线构型定理之一。它说明，仅由两条直线上的六个点和若干交线，就能导出一个高度非平凡的共线结论。这也从侧面表明：射影几何中的“关联关系”本身已经非常强大。

\section{帕斯卡定理}
\begin{theorem}{}{}
设六个点
\[
A,B,C,D,E,F
\]
按顺序位于同一条非退化二次曲线上。则三对对边交点
\[
X=AB\cap DE,\qquad
Y=BC\cap EF,\qquad
Z=CD\cap FA
\]
共线。
\end{theorem}

这条共线的直线称为帕斯卡线。帕斯卡定理可以被视为帕普斯定理在二次曲线背景下的推广：当二次曲线退化为两条直线时，帕斯卡定理就会退化成帕普斯定理。

\section{布里昂雄定理}
\begin{theorem}{}{}
若一个六边形的六条边都与同一条非退化二次曲线相切，则其三条对顶点连线共点。
\end{theorem}

布里昂雄定理是帕斯卡定理的对偶命题。一个讨论“内接六边形对边交点共线”，另一个讨论“外切六边形对顶点连线共点”。二者共同体现了二次曲线理论中最鲜明的对偶对称。

\section{射影几何基本定理}
\begin{theorem}{}{}
在射影直线 $\PP^1$ 上，给定三个互不相同的点
\[
A,B,C
\]
及另外三个互不相同的点
\[
A',B',C',
\]
则存在唯一的射影变换使得
\[
A\mapsto A',\qquad B\mapsto B',\qquad C\mapsto C'.
\]
\end{theorem}

高维版本的结论是：在 $\PP^n$ 中，一个射影变换由一般位置的 $n+2$ 个点的像唯一确定。这说明射影变换具有很强的刚性，并不是任意的图形扭曲。

\chapter{射影平面模型}

\section{球面模型}
考虑单位球面
\[
S^2=\{(x,y,z)\in \R^3:x^2+y^2+z^2=1\}.
\]
把球面上任意一对对径点视为同一点，所得空间就可看作实射影平面 $\RP^2$ 的一个模型。

其原因很简单：过原点的每一条直线恰好与球面交于一对对径点，而射影点本质上正是一条过原点的直线。因此，球面模型非常形象地说明了“射影点是方向类”这一事实。

\section{齐次坐标模型}
齐次坐标模型是最标准的代数模型：
\[
\RP^2=\{(x:y:z)\neq (0:0:0)\}/\sim,
\qquad
(x:y:z)\sim(\lambda x:\lambda y:\lambda z),\ \lambda\neq 0.
\]

它的优点在于：

\begin{enumerate}
    \item 点和直线都能统一写成代数形式；
    \item 射影变换可以直接由矩阵表示；
    \item 二次曲线就是二次齐次方程；
    \item 极点极线可以由二次型与双线性型直接定义。
\end{enumerate}

因此，在严密推导和解析计算中，齐次坐标模型几乎是不可替代的。

\section{仿射平面加无穷远直线模型}
这是最适合直观理解射影平面的模型。它把射影平面写成
\[
\RP^2=\A^2\cup \ell_\infty,
\]
其中 $\ell_\infty$ 是无穷远直线。

在这个模型下：

\begin{itemize}
    \item 普通有限点仍然是仿射平面中的点；
    \item 每个方向对应一个无穷远点；
    \item 所有无穷远点组成无穷远直线；
    \item 平行线在无穷远处相交。
\end{itemize}

这个模型对理解抛物线有一个无穷远点、双曲线有两个无穷远点、椭圆没有实无穷远点尤其有效。

\section{实射影平面的浸入表示}
实射影平面不能无自交地嵌入三维欧氏空间，但可以浸入。经典可视化对象包括交叉帽（cross-cap）和 Boy 曲面等。

需要特别说明的是，虽然克莱因瓶与实射影平面都属于非定向曲面家族，且都不能在 $\R^3$ 中无自交嵌入，但它们并不是同一个拓扑对象。交叉帽常被用作实射影平面的浸入模型，而克莱因瓶则是另一种独立的非定向闭曲面。

\chapter{总结}

\section{主线回顾}
为了让整篇讲义的逻辑更加清楚，最后把全文再压缩为几条最核心的主线：

\begin{enumerate}
    \item \textbf{从仿射到射影：} 给每个方向补上无穷远点，把所有无穷远点组织成无穷远直线，于是几何空间变得封闭；
    \item \textbf{从长度到交比：} 在射影变换下，长度与角度不再稳定，但交比、共线、共点、调和结构稳定；
    \item \textbf{从点线分裂到点线对偶：} 射影平面中点和直线地位对称，对偶原理使许多定理自动成对出现；
    \item \textbf{从圆锥曲线分裂到二次曲线统一：} 椭圆、抛物线、双曲线只是同一射影二次曲线与无穷远直线相交方式不同的结果；
    \item \textbf{从二次曲线到极点极线：} 二次型自然诱导点线对应，从而产生极线、极点、配极原则、自极三角形等一整套结构。
\end{enumerate}

\section{为什么极点极线是核心而不是技巧}
很多人最先接触射影几何，是因为极点极线在圆锥曲线题中非常有效。确实，它可以迅速识别“动点在定直线上 $\Rightarrow$ 极线过定点”这类结构，从而秒杀一些繁琐运算。但若只把它理解为一项“高级技巧”，就太低估它了。

极点极线之所以强大，是因为它是射影几何几条主线的汇合点：

\begin{itemize}
    \item 它依赖于二次曲线的射影统一；
    \item 它依赖于交比和调和点列的几何背景；
    \item 它体现了点与直线的对偶关系；
    \item 它通过配极原则把“定点”和“定线”问题联系起来；
    \item 它还能把焦点准线、自极三角形、完全四边形等结构统一组织。
\end{itemize}

所以，极点极线不是某一章中的附属内容，而是整个二次曲线射影理论的中枢之一。

\section{学习建议}
如果把这份讲义当作系统学习射影几何的路线图，那么建议按以下顺序反复咀嚼：

\begin{enumerate}
    \item 先真正理解无穷远点、无穷远直线与齐次坐标；
    \item 然后掌握交比、调和点列、透视对应；
    \item 再去看射影变换、对偶原理、点列与线束；
    \item 接着理解二次曲线的射影分类；
    \item 最后集中学习极点极线、配极原则、自极三角形与经典定理。
\end{enumerate}

如果这一顺序颠倒，例如直接从极点极线公式入手，而不理解交比、对偶和二次型背景，那么会觉得这一切只是“奇怪的公式与怪异结论”；而一旦前面的结构打通，极点极线会显得近乎理所当然。

\begin{conclusion}{}{}
射影几何最迷人的地方，不在于它会给你更多结论，而在于它会让许多原本分散的结论突然连成一体。\\
平行线为什么要交于无穷远点？交比为什么比长度更重要？椭圆、抛物线、双曲线为什么其实是一家？焦点与准线为什么能互为极点极线？\\
这些问题一旦放进射影几何中，就会呈现出出人意料的统一。
\end{conclusion}

\section{结语}
如果读完这份讲义后，你逐渐形成了以下直觉，那么这份讲义就算达到了目的：

\begin{itemize}
    \item 看到平行，会想到无穷远点；
    \item 看到四点四线，会想到交比与调和结构；
    \item 看到二次曲线，会想到它不仅是一条曲线，更是一种点线对应装置；
    \item 看到“过定点”“在定直线”“共点共线”问题，会本能怀疑背后是否存在配极或对偶背景。
\end{itemize}

一旦这种直觉建立起来，射影几何就不再是一章额外知识，而会成为你理解解析几何、圆锥曲线乃至更高阶几何的一种新语言。
