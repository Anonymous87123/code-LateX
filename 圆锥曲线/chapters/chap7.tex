\chapter{包络问题}
\section{反演背景}
\begin{example}{}{}
如图点 \(C\) 为圆 \(O: x^2 + y^2 = 1\) 上异于 \(A(1,0), B(0,1)\) 的一动点；
直线 \(AC\) 交 \(y\) 轴于点 \(D\)，直线 \(BC\) 交 \(x\) 轴于点 \(E\)，记以 \(DE\) 为直径的圆为圆 \(M\)； 求证：圆 \(M\) 与曲线 \(G: (x^2 + y^2 - x - y)^2 = 8xy\) 相切。
\end{example}
\begin{solution}
首先，求解直线 $AC$ 与 $BC$ 在坐标轴上的截距，并推导其满足的代数恒等关系。
设圆 $O: x^2 + y^2 = 1$ 上的动点 $C$ 的坐标为 $(x_0, y_0)$。由于点 $C$ 异于 $A(1,0)$ 与 $B(0,1)$，必有 $x_0 \neq 1$ 且 $y_0 \neq 1$。
直线 $AC$ 经过点 $A(1,0)$ 与 $C(x_0, y_0)$，其斜率为 $\frac{y_0}{x_0 - 1}$，直线方程可表示为：
\[ y = \frac{y_0}{x_0 - 1}(x - 1). \]
令 $x = 0$，可得直线 $AC$ 与 $y$ 轴的交点 $D$ 的纵坐标 $y_D = \frac{-y_0}{x_0 - 1} = \frac{y_0}{1 - x_0}$。
记 $b = \frac{y_0}{1 - x_0}$，则点 $D$ 的坐标为 $(0, b)$。
同理，直线 $BC$ 经过点 $B(0,1)$ 与 $C(x_0, y_0)$，其斜率为 $\frac{y_0 - 1}{x_0}$，直线方程可表示为：
\[ y - 1 = \frac{y_0 - 1}{x_0}x. \]
令 $y = 0$，可得直线 $BC$ 与 $x$ 轴的交点 $E$ 的横坐标满足 $-1 = \frac{y_0 - 1}{x_0}x_E$，解得 $x_E = \frac{-x_0}{y_0 - 1} = \frac{x_0}{1 - y_0}$。
记 $a = \frac{x_0}{1 - y_0}$，则点 $E$ 的坐标为 $(a, 0)$。\\
探究截距参数 $a$ 与 $b$ 满足的代数恒等关联。分别计算 $a - 1$ 与 $b - 1$：
\[ a - 1 = \frac{x_0}{1 - y_0} - 1 = \frac{x_0 + y_0 - 1}{1 - y_0}, \]
\[ b - 1 = \frac{y_0}{1 - x_0} - 1 = \frac{x_0 + y_0 - 1}{1 - x_0}. \]
将两式相乘，可得：
\[ (a - 1)(b - 1) = \frac{(x_0 + y_0 - 1)^2}{(1 - x_0)(1 - y_0)} = \frac{x_0^2 + y_0^2 + 1 + 2x_0 y_0 - 2x_0 - 2y_0}{1 - x_0 - y_0 + x_0 y_0}. \]
由于点 $C(x_0, y_0)$ 位于圆 $O$ 上，满足约束 $x_0^2 + y_0^2 = 1$。将其代入分子中：
\[ 1 + 1 + 2x_0 y_0 - 2x_0 - 2y_0 = 2(1 - x_0 - y_0 + x_0 y_0). \]
由于 $x_0 \neq 1$ 且 $y_0 \neq 1$，分母 $1 - x_0 - y_0 + x_0 y_0 = (1 - x_0)(1 - y_0) \neq 0$。
约去非零公因式，得到参数 $a$ 与 $b$ 的代数恒等式：
\[ (a - 1)(b - 1) = 2. \]
将其展开可得 $ab - a - b + 1 = 2$，即 $ab - a - b = 1$。\\
其次，运用内积算子建立以 $DE$ 为直径的圆 $M$ 的方程，并进行算子恒等重构。
平面内动点 $(x, y)$ 在以 $D(0, b)$ 与 $E(a, 0)$ 为直径的圆 $M$ 上，等价于该点到 $D$ 与 $E$ 的向量相互垂直。
由内积公式展开可得圆 $M$ 的方程：
\[ x(x - a) + y(y - b) = 0 \implies x^2 + y^2 - ax - by = 0. \]
定义圆 $M$ 的代数算子为 $F(x, y) = x^2 + y^2 - ax - by$。
考察目标曲线 $G$ 的方程 $(x^2 + y^2 - x - y)^2 - 8xy = 0$。
将括号内部的二次多项式依圆 $M$ 的方程结构进行拆分：
\[ x^2 + y^2 - x - y = (x^2 + y^2 - ax - by) + (a - 1)x + (b - 1)y = F(x, y) + (a - 1)x + (b - 1)y. \]
定义一次线性算子 $L_1(x, y) = (a - 1)x + (b - 1)y$。
将曲线 $G$ 的方程代数重构为：
\[ \left[F(x, y) + L_1(x, y)\right]^2 - 8xy = F(x, y)^2 + 2F(x, y)L_1(x, y) + L_1(x, y)^2 - 8xy = 0. \]
展开尾部的二次齐次多项式项：
\begin{align*}
L_1(x, y)^2 - 8xy &= \left[(a - 1)x + (b - 1)y\right]^2 - 8xy \\
&= (a - 1)^2 x^2 + 2(a - 1)(b - 1)xy + (b - 1)^2 y^2 - 8xy.
\end{align*}
代入恒等式 $(a - 1)(b - 1) = 2$，交叉项化简为：
\[ 2(2)xy - 8xy = 4xy - 8xy = -4xy = -2(a - 1)(b - 1)xy. \]
从而该多项式完全配方：
\[ (a - 1)^2 x^2 - 2(a - 1)(b - 1)xy + (b - 1)^2 y^2 = \left[(a - 1)x - (b - 1)y\right]^2. \]
定义一次算子 $L_2(x, y) = (a - 1)x - (b - 1)y$，曲线 $G$ 的隐函数方程 $J(x, y) = 0$ 全局等价于：
\[ J(x, y) \equiv F(x, y)^2 + 2F(x, y)L_1(x, y) + L_2(x, y)^2 = 0. \]
另一方面，基于隐函数梯度法（偏导数向量）分析圆 $M$ 与曲线 $G$ 在交点处的相切属性。
设圆 $M$ 与曲线 $G$ 相交于公共点 $P(x_P, y_P)$。
将 $F(P) = 0$ 与 $J(P) = 0$ 代入上述全局恒等式，得 $0 + 0 + L_2(P)^2 = 0$，即 $L_2(P) = 0$。
这表明两曲线的所有公共交点均分布在直线 $L_2(x, y) = 0$ 上。
通过将 $x = (b - 1)t, y = (a - 1)t$ 代入 $F(x, y) = 0$ 求解 $t$ 以确定交点：
\[ (b - 1)^2 t^2 + (a - 1)^2 t^2 - a(b - 1)t - b(a - 1)t = 0 \implies t^2\left[(a - 1)^2 + (b - 1)^2\right] = t(2ab - a - b). \]
由 $ab - a - b = 1$ 可将右侧括号化为 $ab + 1$。
由于 $(a - 1)(b - 1) = 2$，实数参数 $a, b$ 无法同时为 $1$，故 $(a - 1)^2 + (b - 1)^2 > 0$ 恒成立。
解得 $t = 0$ 或 $t = \frac{ab + 1}{(a - 1)^2 + (b - 1)^2}$。\\
当 $ab + 1 \neq 0$ 时，对应存在异于原点的交点 $P$（即 $t \neq 0$）。
为验证相切，计算曲线 $G$ 的隐函数梯度 $\nabla J(x, y)$。对恒等式应用多元链式求导法则：
\[ \nabla J = 2F \nabla F + 2L_1 \nabla F + 2F \nabla L_1 + 2L_2 \nabla L_2. \]
在交点 $P$ 处，因 $F(P) = 0$ 且 $L_2(P) = 0$，上述梯度表达式完全坍缩为：
\[ \nabla J(P) = 2L_1(P) \nabla F(P). \]
这表明在交点 $P$ 处，曲线 $G$ 的法线向量与圆 $M$ 的法线向量 $\nabla F(P)$ 严格共线。
计算交点 $P$ 处的标量因子 $L_1(P)$：
\[ L_1(P) = (a - 1)x_P + (b - 1)y_P = (a - 1)(b - 1)t + (b - 1)(a - 1)t = 4t. \]
因 $t \neq 0$，标量因子 $4t \neq 0$。同时，圆 $M$ 是非退化的光滑曲线且不过圆心，其边界点梯度 $\nabla F(P) \neq \mathbf{0}$。
故 $\nabla J(P)$ 与 $\nabla F(P)$ 为严格平行的非零向量，两曲线在非原点交点 $P$ 处拥有公共切线，符合相切定义。\\
最后，论证当交点重合于原点时的相切状态以确保逻辑闭环。
若 $ab + 1 = 0$，则 $t = 0$ 是方程的二重根，此时原点 $(0,0)$ 是唯一的交点。
由 $ab = -1$ 与 $ab - a - b = 1$，可推导得 $a + b = -2$。
曲线 $G$ 的方程为 $(x^2 + y^2 - x - y)^2 - 8xy = 0$，其最低次多项式项构成原点处的切线系：
\[ (-x - y)^2 - 8xy = 0 \implies x^2 - 6xy + y^2 = 0. \]
这说明曲线 $G$ 在原点处自交并具有两条切线分支。
圆 $M$ 的方程为 $x^2 + y^2 - ax - by = 0$，其在原点处的切线由一次项决定，方程为 $ax + by = 0$，即 $y = -\frac{a}{b}x$。
将该切线代入曲线 $G$ 的切线系方程进行验证：
\[ x^2 - 6x\left(-\frac{a}{b}x\right) + \left(-\frac{a}{b}x\right)^2 = x^2\left(1 + \frac{6a}{b} + \frac{a^2}{b^2}\right) = x^2\left(\frac{b^2 + 6ab + a^2}{b^2}\right). \]
利用 $a + b = -2$ 与 $ab = -1$ 计算特征常数分子：
\[ a^2 + 6ab + b^2 = (a + b)^2 + 4ab = (-2)^2 + 4(-1) = 0. \]
验证结果严格为 $0$，这证明圆 $M$ 在原点处的切线与曲线 $G$ 在原点处的两支切线之一完全重合。
在解析几何中，曲线分支共享切线严格等价于两曲线在此奇点相切。
综上所述，无论交点状态如何，圆 $M$ 恒与曲线 $G$ 相切。结论成立。
\end{solution}

\begin{example}{}{}
点 $P$ 为等轴双曲线 $E: x^2 - y^2 = a^2, (a > 0)$ 上一动点，双曲线 $E$ 在点 $P$ 处的切线分别交其渐近线于 $M, N$ 两点，记以 $MN$ 为直径的圆为圆 $P$；求证：圆 $P$ 与双纽线 $G: (x^2 + y^2)^2 = 4a^2(x^2 - y^2)$ 相切；
\end{example}
\begin{solution}
首先，建立等轴双曲线的极化算子代数系统，推导切线与渐近线交点的坐标以确定圆 $P$ 的算子方程。
已知等轴双曲线 $E: x^2 - y^2 = a^2\ (a>0)$。定义双曲极化算子 $E(X,Y) = x_X x_Y - y_X y_Y$，以及平面欧氏内积算子 $I(X,Y) = x_X x_Y + y_X y_Y$。
设动点 $P(x_0, y_0)$ 位于双曲线 $E$ 上，其坐标满足 $E(P,P) = a^2$，即 $x_0^2 - y_0^2 = a^2$。
双曲线在点 $P$ 处的切线算子方程为 $E(P,X) = a^2$，即 $x_0 x - y_0 y = a^2$。
双曲线的渐近线方程满足齐次算子 $E(X,X) = 0$，即 $x^2 - y^2 = 0$，解得 $y = x$ 与 $y = -x$。
将切线分别与两条渐近线联立以求解交点 $M, N$ 的坐标：
联立 $y = x$ 与 $x_0 x - y_0 y = a^2$，提取公因式得 $x(x_0 - y_0) = a^2$。因 $x_0^2 - y_0^2 = a^2$，化简得 $x = x_0 + y_0$。解得点 $M$ 的坐标为 $(x_0 + y_0, x_0 + y_0)$。
联立 $y = -x$ 与 $x_0 x - y_0 y = a^2$，提取公因式得 $x(x_0 + y_0) = a^2$，化简得 $x = x_0 - y_0$。解得点 $N$ 的坐标为 $(x_0 - y_0, -x_0 + y_0)$。
计算线段 $MN$ 的中点坐标：
横坐标为 $\frac{(x_0 + y_0) + (x_0 - y_0)}{2} = x_0$；
纵坐标为 $\frac{(x_0 + y_0) + (-x_0 + y_0)}{2} = y_0$。
由此可得，线段 $MN$ 的中点严格为点 $P(x_0, y_0)$。
以 $MN$ 为直径的圆 $P$，其圆心即为点 $P$。其半径的平方 $r^2$ 等于 $|PM|^2$：
\[ r^2 = (x_0 + y_0 - x_0)^2 + (x_0 + y_0 - y_0)^2 = y_0^2 + x_0^2 = I(P,P) \]
基于此，圆 $P$ 的算子方程可表述为 $I(X-P, X-P) = I(P,P)$。展开并消去常数项 $I(P,P)$，得到圆 $P$ 的多项式算子方程：
\[ C(X) = I(X,X) - 2I(P,X) = 0 \]
其次，构造双纽线方程与圆方程的代数恒等式，实现曲线降维转换。
双纽线 $G: (x^2 + y^2)^2 = 4a^2(x^2 - y^2)$ 可重写为内积算子形式：
\[ G(X) = I(X,X)^2 - 4a^2 E(X,X) = 0 \]
将圆 $P$ 的算子恒等变形 $I(X,X) \equiv C(X) + 2I(P,X)$ 整体代入双纽线函数的第一项中展开：
\begin{align*}
G(X) &= \big[ C(X) + 2I(P,X) \big]^2 - 4a^2 E(X,X) \\
&= C(X)^2 + 4C(X)I(P,X) + 4I(P,X)^2 - 4a^2 E(X,X) \\
&= C(X) \big[ C(X) + 4I(P,X) \big] + 4 \big[ I(P,X)^2 - a^2 E(X,X) \big]
\end{align*}
针对余项函数 $\Delta(X) = I(P,X)^2 - a^2 E(X,X)$ 进行展开。代入 $a^2 = x_0^2 - y_0^2$ 处理：
\begin{align*}
\Delta(X) &= (x_0 x + y_0 y)^2 - (x_0^2 - y_0^2)(x^2 - y^2) \\
&= (x_0^2 x^2 + 2x_0 y_0 x y + y_0^2 y^2) - (x_0^2 x^2 - x_0^2 y^2 - y_0^2 x^2 + y_0^2 y^2) \\
&= y_0^2 x^2 + 2x_0 y_0 x y + x_0^2 y^2 = (y_0 x + x_0 y)^2
\end{align*}
定义一次线性交叉算子 $L(X) = y_0 x + x_0 y$，则原双纽线方程被严格分解为代数恒等式：
\[ G(X) \equiv C(X) \big[ C(X) + 4I(P,X) \big] + 4L(X)^2 \]
最后，基于恒等式与二维梯度算子，确立交点并严格论证两曲线相切。
设存在点 $Q(x_Q, y_Q)$，同时满足圆 $P$ 的方程 $C(Q) = 0$ 以及线性算子 $L(Q) = 0$。
联立 $x^2 + y^2 - 2x_0 x - 2y_0 y = 0$ 与 $y_0 x + x_0 y = 0$。
由第二式得 $y = -\frac{y_0}{x_0}x$（因 $x_0^2 - y_0^2 = a^2 > 0$，故 $x_0 \neq 0$）。代入第一式中提取公因式：
\[ x^2 + \left(-\frac{y_0}{x_0}x\right)^2 - 2x_0 x - 2y_0\left(-\frac{y_0}{x_0}x\right) = 0 \]
\[ x \left( x\frac{x_0^2 + y_0^2}{x_0^2} - \frac{2(x_0^2 - y_0^2)}{x_0} \right) = 0 \]
将 $x_0^2 - y_0^2 = a^2$ 代入一次项，解得非零根 $x_Q = \frac{2a^2 x_0}{x_0^2 + y_0^2}$，对应纵坐标 $y_Q = -\frac{2a^2 y_0}{x_0^2 + y_0^2}$。
由于 $a > 0$，点 $Q$ 确切存在且不为坐标原点。将点 $Q$ 代入双纽线恒等式，有 $G(Q) = 0 \cdot (0 + 4I(P,Q)) + 4 \cdot 0^2 = 0$，表明点 $Q$ 为圆 $P$ 与双纽线 $G$ 的一个公共交点。
对多项式恒等式两端施加二维梯度算子 $\nabla = \left(\frac{\partial}{\partial x}, \frac{\partial}{\partial y}\right)$，应用乘积求导法则展开：
\[ \nabla G(X) = \nabla C(X) \big[ C(X) + 4I(P,X) \big] + C(X) \nabla \big[ C(X) + 4I(P,X) \big] + 8 L(X) \nabla L(X) \]
在交点 $Q$ 处进行求值，由于 $C(Q) = 0$ 且 $L(Q) = 0$，上式后两项严格归零：
\[ \nabla G(Q) = 4I(P,Q) \nabla C(Q) \]
计算标量常数系数 $4I(P,Q)$ 的值：
\[ 4I(P,Q) = 4(x_0 x_Q + y_0 y_Q) = 4 \left( x_0 \frac{2a^2 x_0}{x_0^2 + y_0^2} + y_0 \frac{-2a^2 y_0}{x_0^2 + y_0^2} \right) = \frac{8a^2(x_0^2 - y_0^2)}{x_0^2 + y_0^2} = \frac{8a^4}{x_0^2 + y_0^2} \]
由于 $a > 0$，该标量系数 $4I(P,Q) > 0$ 恒成立。
同时，验证圆在点 $Q$ 处的梯度 $\nabla C(Q) = 2(Q - P)$ 不为零向量。若 $Q = P$，则 $y_Q = y_0$，即 $\frac{-2a^2 y_0}{x_0^2 + y_0^2} = y_0$。若 $y_0 \neq 0$，化简得 $-2a^2 = x_0^2 + y_0^2 \ge 0$，与 $a>0$ 矛盾；若 $y_0 = 0$，则 $x_0 = \pm a$，此时 $x_Q = \frac{2a^2(\pm a)}{a^2} = \pm 2a \neq x_0$，亦矛盾。故 $Q \neq P$，即 $\nabla C(Q) \neq \mathbf{0}$。
这表明双纽线方程 $G(X) = 0$ 在点 $Q$ 处的法向梯度严格等于圆 $P$ 方程 $C(X) = 0$ 在同一点法向梯度的非零正数倍。两曲线在该非奇点处的法向量完全平行，进而说明它们在此处共享同一切线。
综上所述，以 $MN$ 为直径的圆 $P$ 恒与双纽线 $G$ 相切，结论成立。
\end{solution}



\begin{example}{}{}
如图点 $P$ 为圆 $O: x^2 + y^2 = a^2, (a > 0)$ 上一动点，圆 $O$ 在点 $P$ 处的切线分别交 $x$ 轴和 $y$ 轴于 $A, B$ 两点，记以 $AB$ 为直径的圆为圆 $M$；求证：圆 $M$ 与曲线 $G: (x^2 + y^2)(x^2 + y^2 - a^2)^3 = 27a^4x^2y^2$ 相切；
\end{example}
\begin{solution}
首先，应用内积算子建立以线段 $AB$ 为直径的动圆 $M$ 的代数方程。
定义平面向量的欧氏内积算子 $I(U,V) = x_U x_V + y_U y_V$，以及模长平方算子 $I(X, X) = x^2 + y^2$。
设点 $P(x_0, y_0)$ 为圆 $O: x^2 + y^2 = a^2$ 上的动点，满足约束方程 $x_0^2 + y_0^2 = a^2$。为保证切线在坐标轴上的截距存在，设定 $x_0 y_0 \neq 0$。
根据极化算子理论，圆 $O$ 在点 $P$ 处的切线方程为 $x_0 x + y_0 y = a^2$。
分别令 $y = 0$ 和 $x = 0$，解得切线与坐标轴的交点分别为 $A\left(\frac{a^2}{x_0}, 0\right)$ 与 $B\left(0, \frac{a^2}{y_0}\right)$。
平面上以线段 $AB$ 为直径的圆 $M$，其上任意动点 $X(x,y)$ 满足直径端点向量正交条件，即内积极化式 $I(X-A, X-B) = 0$。展开得：
$$ \left(x - \frac{a^2}{x_0}\right)x + y\left(y - \frac{a^2}{y_0}\right) = 0 $$
化简得到圆 $M$ 的算子方程：
$$ I(X, X) - \frac{a^2 x}{x_0} - \frac{a^2 y}{y_0} = 0 $$
在等式两边同乘 $x_0 y_0$，将曲线族方程转化为多项式形式：
$$ F(x_0, y_0) = x_0 y_0 I(X, X) - a^2 y_0 x - a^2 x_0 y = 0 $$
其次，应用代数包络理论推演圆 $M$ 曲线族的边界轨迹。
将圆 $M$ 视作依赖于连续参数 $(x_0, y_0)$ 的曲线系。若点 $X(x, y)$ 位于该曲线系构成的包络线上，则在该点处方程 $F$ 对参量的偏导数向量必须与约束方程 $\Phi(x_0, y_0) = x_0^2 + y_0^2 - a^2 = 0$ 的偏导数向量线性相关。
计算对应的偏导数：
$$ \frac{\partial F}{\partial x_0} = y_0 I(X, X) - a^2 y, \quad \frac{\partial F}{\partial y_0} = x_0 I(X, X) - a^2 x $$
$$ \frac{\partial \Phi}{\partial x_0} = 2x_0, \quad \frac{\partial \Phi}{\partial y_0} = 2y_0 $$
线性相关的代数等价条件为二阶雅可比行列式归零：
$$ \frac{\partial F}{\partial x_0} \frac{\partial \Phi}{\partial y_0} - \frac{\partial F}{\partial y_0} \frac{\partial \Phi}{\partial x_0} = 0 $$
代入偏导数表达式：
$$ \big(y_0 I(X, X) - a^2 y\big)(2y_0) - \big(x_0 I(X, X) - a^2 x\big)(2x_0) = 0 $$
提取公因式并化简，得到相切条件约束方程：
$$ I(X, X)(y_0^2 - x_0^2) - a^2 y y_0 + a^2 x x_0 = 0 $$
接下来，联立系统方程并通过代数消元显式求出包络条件下的特征参数。
联合包络系统必须满足的两个多项式方程：
$$ x_0 y_0 I(X, X) - a^2 y_0 x - a^2 x_0 y = 0 $$
$$ y_0^2 I(X, X) - x_0^2 I(X, X) - a^2 y y_0 + a^2 x x_0 = 0 $$
将第一式两端同乘 $x_0$，第二式两端同乘 $y_0$，展开得：
$$ x_0^2 y_0 I(X, X) - a^2 x_0 y_0 x - a^2 x_0^2 y = 0 $$
$$ y_0^3 I(X, X) - x_0^2 y_0 I(X, X) - a^2 y y_0^2 + a^2 x x_0 y_0 = 0 $$
将上述两式相加，含有 $x_0^2 y_0 I(X, X)$ 与 $a^2 x_0 y_0 x$ 的项均对消，得到：
$$ y_0^3 I(X, X) - a^2 x_0^2 y - a^2 y y_0^2 = 0 \implies y_0^3 I(X, X) - a^2 y (x_0^2 + y_0^2) = 0 $$
代入参数约束 $x_0^2 + y_0^2 = a^2$，解得：
$$ y_0^3 = \frac{a^4 y}{I(X, X)} $$
同理，将第一式两端同乘 $y_0$，第二式两端同乘 $-x_0$，展开得：
$$ x_0 y_0^2 I(X, X) - a^2 y_0^2 x - a^2 x_0 y_0 y = 0 $$
$$ x_0^3 I(X, X) - x_0 y_0^2 I(X, X) + a^2 y x_0 y_0 - a^2 x x_0^2 = 0 $$
将两式相加，含有 $x_0 y_0^2 I(X, X)$ 与 $a^2 x_0 y_0 y$ 的项均对消，得到：
$$ x_0^3 I(X, X) - a^2 y_0^2 x - a^2 x x_0^2 = 0 \implies x_0^3 I(X, X) - a^2 x (x_0^2 + y_0^2) = 0 $$
代入参数约束 $x_0^2 + y_0^2 = a^2$，解得：
$$ x_0^3 = \frac{a^4 x}{I(X, X)} $$
最后，通过连续代数操作重构包络多项式并验证相切关系。
将解得的 $x_0, y_0$ 表达式代回参数约束方程 $x_0^2 + y_0^2 = a^2$：
$$ \left( \frac{a^4 x}{I(X, X)} \right)^{\frac{2}{3}} + \left( \frac{a^4 y}{I(X, X)} \right)^{\frac{2}{3}} = a^2 $$
提取公因式 $a^{\frac{8}{3}} I(X, X)^{-\frac{2}{3}}$ 分离变量，化为：
$$ a^{\frac{8}{3}} \left(x^{\frac{2}{3}} + y^{\frac{2}{3}}\right) = a^2 I(X, X)^{\frac{2}{3}} \implies a^{\frac{2}{3}} \left(x^{\frac{2}{3}} + y^{\frac{2}{3}}\right) = I(X, X)^{\frac{2}{3}} $$
对等式两边整体立方，应用恒等式 $(u+v)^3 = u^3 + v^3 + 3uv(u+v)$：
$$ a^2 \left[ x^2 + y^2 + 3 x^{\frac{2}{3}} y^{\frac{2}{3}} \left(x^{\frac{2}{3}} + y^{\frac{2}{3}}\right) \right] = I(X, X)^2 $$
利用 $I(X, X) = x^2 + y^2$ 以及 $x^{\frac{2}{3}} + y^{\frac{2}{3}} = a^{-\frac{2}{3}} I(X, X)^{\frac{2}{3}}$ 替换括号内的各项：
$$ a^2 I(X, X) + 3 a^{\frac{4}{3}} x^{\frac{2}{3}} y^{\frac{2}{3}} I(X, X)^{\frac{2}{3}} = I(X, X)^2 $$
移项以隔离分数次幂项：
$$ 3 a^{\frac{4}{3}} x^{\frac{2}{3}} y^{\frac{2}{3}} I(X, X)^{\frac{2}{3}} = I(X, X)^2 - a^2 I(X, X) = I(X, X) \big( I(X, X) - a^2 \big) $$
对等式两边再次立方：
$$ 27 a^4 x^2 y^2 I(X, X)^2 = I(X, X)^3 \big( I(X, X) - a^2 \big)^3 $$
在平面上 $I(X, X) \neq 0$ 的非退化区域，等式两端同除以 $I(X, X)^2$，多项式化简为：
$$ 27 a^4 x^2 y^2 = I(X, X) \big( I(X, X) - a^2 \big)^3 $$
将内积算子还原为坐标形式：
$$ 27 a^4 x^2 y^2 = (x^2 + y^2) (x^2 + y^2 - a^2)^3 $$
该多项式方程与题设曲线 $G$ 的方程一致。
根据代数包络理论，参数化曲线族的包络线必然与该曲线族内的每一个成员曲线在对应的特征点处相切。由于动圆 $M$ 曲线族生成的包络线即为曲线 $G$，动圆 $M$ 必然与曲线 $G$ 始终相切。结论成立。
\end{solution}


\begin{example}{}{}
过 $x$ 轴上一定点 $A(a,0)$ 的直线分别交直线 $y=x$ 及 $y=-x$ 于 $B,C$ 两点，记以 $B,C$ 为直径的圆为圆 $D$；求证: 圆 $D$ 与曲线 $G: (x^2 + y^2 - ax)^2 = a^2(x^2 - y^2)$ 相切；
\end{example}
\begin{solution}
为避免复杂的有理参数化运算所导致的多项式次幂膨胀，本题基于扩展的算子法进行纯代数推导。首先扩充算子理论，定义平面上任意两点（或位置向量）$M, N$ 的四种基础算子：
\begin{itemize}
    \item 标准内积算子：$I(M,N) = x_M x_N + y_M y_N$，满足自内积 $I(X,X) = x^2 + y^2$；
    \item 标准外积算子：$W(M,N) = x_M y_N - x_N y_M$，表征有向面积的代数倍数；
    \item 共轭内积算子：$S(M,N) = x_M x_N - y_M y_N$，满足 $S(X,X) = x^2 - y^2$；
    \item 交叉内积算子：$V(M,N) = x_M y_N + x_N y_M$。
\end{itemize}
基于上述定义，推导平面解析几何的共轭拉格朗日恒等式：
\begin{align*}
I(U,X)^2 - V(U,X)^2 &= (x_U x + y_U y)^2 - (x_U y + y_U x)^2 \\
&= x_U^2 x^2 + 2x_U y_U xy + y_U^2 y^2 - (x_U^2 y^2 + 2x_U y_U xy + y_U^2 x^2) \\
&= x_U^2(x^2 - y^2) - y_U^2(x^2 - y^2) \\
&= (x_U^2 - y_U^2)(x^2 - y^2) = S(U,U) S(X,X).
\end{align*}
已知定点 $A(a,0)$，对于任意动点 $X(x,y)$，有 $I(A,X) = S(A,X) = ax$ 且 $I(A,A) = S(A,A) = a^2$。
\begin{itemize}
\item 过点 $A$ 的直线分别交直线 $y=x$ 与 $y=-x$ 于 $B, C$ 两点。
\item 对于点 $B$，因其在 $y=x$ 上，故 $S(B,B) = x_B^2 - y_B^2 = 0$。
\item 对于点 $C$，因其在 $y=-x$ 上，故 $S(C,C) = x_C^2 - y_C^2 = 0$。
\item 计算内积与共轭内积：$I(B,C) = x_B x_C + x_B(-x_C) = 0$，$S(B,C) = x_B x_C - (-x_B x_C) = 2x_B x_C$。
\end{itemize}
借助外积算子表示面积的性质，点 $A,B,C$ 共线等价于 $W(B-A, C-A) = 0$。
利用双线性分配律展开可得：
\[
W(B,C) - W(A,C) - W(B,A) = 0.
\]
代入坐标计算各项外积：
$W(B,C) = x_B(-x_C) - x_C(x_B) = -2x_B x_C$；
$W(A,C) = a(-x_C) - 0 = -a x_C$；
$W(B,A) = 0 - a x_B = -a x_B$。
代入共线方程得 $-2x_B x_C - (-a x_C) - (-a x_B) = 0$，化简得到关于横坐标的代数约束：
\[
2x_B x_C = a(x_B + x_C).
\]
设向量 $H = B+C$，考察点 $H$ 的共轭内积与标准内积性质。
由双线性性质展开 $S(H,H)$：
\[
S(H,H) = S(B+C, B+C) = S(B,B) + 2S(B,C) + S(C,C) = 0 + 2(2x_B x_C) + 0 = 4x_B x_C.
\]
计算内积 $I(A,H)$：
\[
I(A,H) = I(A, B+C) = a(x_B + x_C).
\]
结合共线等式 $4x_B x_C = 2a(x_B + x_C)$，得到代数关系：
\[
S(H,H) = 2 I(A,H).
\]
进而计算差分向量 $H-A$ 的共轭内积：
\[
S(H-A, H-A) = S(H,H) - 2S(A,H) + S(A,A).
\]
由于 $S(A,H) = a x_H = I(A,H)$，整体代入可得：
\[
S(H-A, H-A) = 2I(A,H) - 2I(A,H) + I(A,A) = I(A,A) = a^2.
\]
建立圆 $D$ 与曲线 $G$ 的算子多项式方程。
圆 $D$ 以 $BC$ 为直径，故圆上任意动点 $X$ 满足正交条件 $I(X-B, X-C) = 0$。
展开得 $I(X,X) - I(X, B+C) + I(B,C) = 0$。因 $I(B,C) = 0$ 且 $H = B+C$，圆 $D$ 的算子方程可表示为：
\[
C(X) = I(X,X) - I(X, H) = 0.
\]
曲线 $G: (x^2+y^2-ax)^2 = a^2(x^2-y^2)$ 可重写为纯算子方程：
\[
G(X) = (I(X,X) - I(A,X))^2 - I(A,A) S(X,X) = 0.
\]
利用圆的代换式 $I(X,X) = C(X) + I(X,H)$ 对 $G(X)$ 进行多项式降阶变形：
\begin{align*}
G(X) &= (C(X) + I(X, H) - I(A,X))^2 - I(A,A) S(X,X) \\
&= (C(X) + I(H-A, X))^2 - I(A,A) S(X,X) \\
&= C(X)^2 + 2C(X)I(H-A, X) + I(H-A, X)^2 - I(A,A) S(X,X).
\end{align*}
根据已证明的共轭拉格朗日恒等式，有 $I(H-A, X)^2 - V(H-A, X)^2 = S(H-A, H-A) S(X,X)$。
代入已验证的 $S(H-A, H-A) = I(A,A)$，得到：
\[
I(H-A, X)^2 - I(A,A) S(X,X) = V(H-A, X)^2.
\]
将其代回，得到曲线 $G$ 的结构恒等分解式：
\[
G(X) = C(X)^2 + 2C(X)I(H-A, X) + V(H-A, X)^2.
\]
为论证圆 $D$ 与曲线 $G$ 相切，设 $P$ 为二次代数方程 $C(X) = 0$ 与一次线性方程 $V(H-A, X) = 0$ 异于原点 $O$ 的实数交点。
由于 $C(P) = 0$ 且 $V(H-A, P) = 0$，代入结构恒等式有 $G(P) = 0$，即点 $P$ 必然位于曲线 $G$ 上。
对结构多项式 $G(X)$ 整体求多元偏导生成梯度向量 $\nabla$，考察公共交点处的法线方向：
\[
\nabla G(X) = 2C(X)\nabla C(X) + 2\nabla C(X)I(H-A,X) + 2C(X)\nabla I(H-A,X) + 2V(H-A,X)\nabla V(H-A,X).
\]
在交点 $P$ 处，由于 $C(P) = 0$ 且 $V(H-A, P) = 0$，上述梯度向量方程瞬间坍缩为：
\[
\nabla G(P) = 2 I(H-A, P) \nabla C(P).
\]
最后验证标量比例系数及法向量的非退化性。将 $V(H-A, P) = (x_H - a)y_P + y_H x_P = 0$ 变形为 $y_P = -\frac{y_H}{x_H - a} x_P$。
代入圆的坐标展开方程 $C(P) = x_P^2 + y_P^2 - x_H x_P - y_H y_P = 0$ 中，提取公共因子 $x_P$：
\[
x_P \left[ x_P \frac{(x_H - a)^2 + y_H^2}{(x_H - a)^2} - \frac{x_H(x_H - a) - y_H^2}{x_H - a} \right] = 0.
\]
由前述计算 $S(H-A, H-A) = (x_H - a)^2 - y_H^2 = a^2$ 得 $y_H^2 = (x_H - a)^2 - a^2$。
若 $x_H - a = 0$，则 $y_H^2 = -a^2 < 0$，与实数范围发生矛盾，故 $x_H - a \neq 0$ 恒成立。
代入一次项分子中化简得：$x_H(x_H - a) - y_H^2 = x_H^2 - a x_H - ((x_H - a)^2 - a^2) = a x_H$。
对于非 $x$ 轴的非退化截线（此时 $x_H \neq 0$），解得唯一非零横坐标 $x_P = \frac{a x_H (x_H - a)}{(x_H - a)^2 + y_H^2}$。由于 $a \neq 0$ 且 $x_H \neq 0$，从而保证 $x_P \neq 0$。
据此计算切点法向比例因子 $I(H-A, P)$：
\[
I(H-A, P) = (x_H - a)x_P + y_H y_P = (x_H - a)x_P - \frac{y_H^2}{x_H - a} x_P = x_P \frac{(x_H - a)^2 - y_H^2}{x_H - a} = \frac{a^2 x_P}{x_H - a}.
\]
因定点参数 $a \neq 0$ 且 $x_P \neq 0$，必有 $I(H-A, P) \neq 0$。同时非退化圆方程保证了 $\nabla C(P) \neq \vec{0}$。
此结果严格证明，在非平凡交点 $P$ 处，曲线 $G$ 的法向向量存在，且为圆 $D$ 法向向量的非零实数倍。两曲线在公共点处法向共线且共享切线，圆 $D$ 恒与曲线 $G$ 相切。逻辑闭环，结论成立。
\end{solution}


\begin{example}{}{}
如图过 $x$ 轴上一定点 $M(a,0)$ 的直线交圆 $O: x^2 + y^2 = 1$ 于 $A,B$ 两点，记以 $A,B$ 为直径的圆为圆 $N$；求证: 圆 $N$ 与曲线 $G: (x^2 + y^2 - ax - 1)^2 + a^2(x^2 + y^2 - 2) = 0$ 相切；
\end{example}
\begin{solution}
建立平面解析几何的算子代数结构。定义圆 $O: x^2 + y^2 = 1$ 的多项式算子为 $T_O(X, X) = x^2 + y^2 - 1$。平面内的点 $X$ 位于圆 $O$ 上等价于 $T_O(X, X) = 0$。
设经过定点 $M(a,0)$ 的动直线交圆 $O$ 于 $A, B$ 两点。以线段 $AB$ 为直径的圆记为圆 $N$，设其圆心为 $N(x_0, y_0)$。
根据圆的几何性质，弦 $AB$ 的中点 $N$ 与圆心 $O$ 的连线垂直于该弦，即 $\overrightarrow{ON} \cdot \overrightarrow{MN} = 0$。将坐标代入可得圆心 $N$ 满足的约束方程：
\[ x_0(x_0 - a) + y_0^2 = 0 \implies x_0^2 + y_0^2 = ax_0. \]
在直角 $\triangle ONA$ 中，由勾股定理可知圆 $N$ 的半径平方为 $R_N^2 = |OA|^2 - |ON|^2 = 1 - (x_0^2 + y_0^2)$。
代入圆心约束，得到 $R_N^2 = 1 - ax_0$。
圆 $N$ 的多项式算子方程可表示为：
\[ F_N(X) = (x-x_0)^2 + (y-y_0)^2 - (1 - ax_0) = x^2 + y^2 - 2x_0 x - 2y_0 y + 2ax_0 - 1 = 0. \]
为寻求圆 $N$ 与圆 $O$ 的算子联系，将上式重写为：
\[ T_O(X, X) = F_N(X) + 2x_0 x + 2y_0 y - 2ax_0 = F_N(X) - 2x_0(a-x) + 2y_0 y. \]
考察目标曲线 $G: (x^2 + y^2 - ax - 1)^2 + a^2(x^2 + y^2 - 2) = 0$。
将其转化为关于极化算子 $T_O(X, X)$ 的代数表达式：
\[ E_G(X) = \big(T_O(X, X) - ax\big)^2 + a^2\big(T_O(X, X) - 1\big) = 0. \]
展开上式并合并含 $T_O(X, X)$ 的同类项：
\begin{align*}
E_G(X) &= T_O(X, X)^2 - 2ax T_O(X, X) + a^2 x^2 + a^2 T_O(X, X) - a^2 \\
&= T_O(X, X)^2 + a(a - 2x)T_O(X, X) + a^2(x^2 - 1).
\end{align*}
利用代数恒等式 $a^2(x^2 - 1) = a^2(x^2 + y^2 - 1) - a^2 y^2 = a^2 T_O(X, X) - a^2 y^2$，代入替换常数项部分：
\begin{align*}
E_G(X) &= T_O(X, X)^2 + a(a - 2x)T_O(X, X) + a^2 T_O(X, X) - a^2 y^2 \\
&= T_O(X, X)^2 + 2a(a - x)T_O(X, X) - a^2 y^2.
\end{align*}
定义关于动点 $X$ 的一次线性泛函 $L_1(X) = 2x_0(a - x) - 2y_0 y$。根据前面的推导，有 $T_O(X, X) = F_N(X) - L_1(X)$。
将其代入化简后的 $E_G(X)$，完全展开并分离包含 $F_N(X)$ 的项与不含 $F_N(X)$ 的项：
\begin{align*}
E_G(X) &= \big(F_N(X) - L_1(X)\big)^2 + 2a(a - x)\big(F_N(X) - L_1(X)\big) - a^2 y^2 \\
&= F_N(X)^2 - 2F_N(X)L_1(X) + L_1(X)^2 + 2a(a - x)F_N(X) - 2a(a - x)L_1(X) - a^2 y^2 \\
&= F_N(X)\big[F_N(X) - 2L_1(X) + 2a(a - x)\big] + \big[L_1(X)^2 - 2a(a - x)L_1(X) - a^2 y^2\big].
\end{align*}
定义辅助多项式 $Q(X) = F_N(X) - 2L_1(X) + 2a(a - x)$，以及剩余项 $R(X) = L_1(X)^2 - 2a(a - x)L_1(X) - a^2 y^2$。\\
将 $L_1(X) = 2x_0(a - x) - 2y_0 y$ 代入 $R(X)$ 中展开计算：
\begin{align*}
R(X) &= \big[2x_0(a - x) - 2y_0 y\big]^2 - 2a(a - x)\big[2x_0(a - x) - 2y_0 y\big] - a^2 y^2 \\
&= 4x_0^2(a - x)^2 - 8x_0 y_0(a - x)y + 4y_0^2 y^2 - 4ax_0(a - x)^2 + 4ay_0(a - x)y - a^2 y^2 \\
&= 4x_0(x_0 - a)(a - x)^2 - 4y_0(2x_0 - a)(a - x)y + (4y_0^2 - a^2)y^2.
\end{align*}
结合初始坐标约束 $x_0(x_0 - a) = -y_0^2$，对各项系数进行代换：
关于 $(a - x)^2$ 的系数为 $4x_0(x_0 - a) = -4y_0^2$；
关于 $y^2$ 的系数为 $4y_0^2 - a^2 = -4x_0(x_0 - a) - a^2 = -4x_0^2 + 4ax_0 - a^2 = -(2x_0 - a)^2$。
代入上述系数关系，剩余项 $R(X)$ 化为完全平方差形式：
\begin{align*}
R(X) &= -4y_0^2(a - x)^2 - 4y_0(2x_0 - a)(a - x)y - (2x_0 - a)^2 y^2 \\
&= -\big[2y_0(a - x) + (2x_0 - a)y\big]^2.
\end{align*}
定义一次线性泛函 $L_T(X) = 2y_0(a - x) + (2x_0 - a)y$，由此得到曲线 $G$ 与圆 $N$ 的多项式代数恒等式：
\[ E_G(X) \equiv F_N(X) Q(X) - L_T(X)^2. \]
从该恒等关系可知，曲线 $G$（即 $E_G(X) = 0$）与圆 $N$（即 $F_N(X) = 0$）在平面内的所有公共交点必定同时满足 $L_T(X) = 0$。
设 $X_c$ 为两曲线的任意一个公共交点。对恒等式两端求取微分梯度算子 $\nabla$，得：
\[ \nabla E_G(X) = Q(X)\nabla F_N(X) + F_N(X)\nabla Q(X) - 2L_T(X)\nabla L_T(X). \]
将交点 $X_c$ 代入上式。由于在该交点处 $F_N(X_c) = 0$ 且 $L_T(X_c) = 0$，梯度表达式化简为：
\[ \nabla E_G(X_c) = Q(X_c)\nabla F_N(X_c). \]
该等式表明，在公共交点 $X_c$ 处，曲线 $G$ 的法向量 $\nabla E_G(X_c)$ 与圆 $N$ 的法向量 $\nabla F_N(X_c)$ 呈纯量倍数关系，即两者的法向量相互平行。
在解析几何中，两条曲线在交点处的法向量相互平行，等价于两曲线在该点处共用同一条切线，满足相切的几何条件。因此，圆 $N$ 必定与曲线 $G$ 相切。结论成立。
\end{solution}




\begin{example}{}{}
如图 $\triangle ABC$ 是抛物线 $E: y^2 = 2x$ 上的内接正三角形，记 $\triangle ABC$ 外接圆为圆 $G$，试求圆 $G$ 所形成的包络曲线方程；
\end{example}
\begin{solution}
为避免暴力联立所引发的多项式次幂膨胀，本题基于算子法核心的多项式结构降维与根系对称性进行纯代数推导。

首先，构建内接正三角形的根系多项式。
设抛物线 $E: y^2 = 2x$ 上的动点纵坐标为 $y$，则横坐标为 $x = \frac{1}{2}y^2$。
已知 $\triangle ABC$ 为抛物线内接正三角形，由正三角形几何性质可知，其外心与重心重合。设外接圆 $G$ 的圆心（即重心）为 $(X_0, Y_0)$，半径为 $R$。
圆 $G$ 的方程为 $x^2 + y^2 - 2X_0 x - 2Y_0 y + X_0^2 + Y_0^2 - R^2 = 0$。
将抛物线方程代入圆方程中，得到关于 $y$ 的四次交点多项式：
\[
\frac{1}{4}y^4 + y^2 - X_0 y^2 - 2Y_0 y + X_0^2 + Y_0^2 - R^2 = 0 \implies y^4 + 4(1 - X_0)y^2 - 8Y_0 y + 4(X_0^2 + Y_0^2 - R^2) = 0.
\]
该多项式的四个根分别对应交点 $A, B, C$ 与第四交点 $D$。由于三次项系数为 $0$，由韦达定理知四根之和为零，即 $y_A + y_B + y_C + y_D = 0$。
因 $(X_0, Y_0)$ 为 $\triangle ABC$ 重心，满足 $\frac{y_A + y_B + y_C}{3} = Y_0 \implies y_A + y_B + y_C = 3Y_0$。
由此可解得第四交点的纵坐标 $y_D = -3Y_0$。
剔除根 $y_D$ 后，剩余三根 $y_A, y_B, y_C$ 必然构成一个三次多项式方程，设为：
\[
y^3 - 3Y_0 y^2 + qy - r = 0.
\]

其次，利用平移变换与多项式降次锁定外接圆参数。
为消除不对称性，作平移代换 $y = z + Y_0$，则 $z_A, z_B, z_C$ 满足缺二次项的方程：
\[
z^3 + Pz + Q = 0.
\]
其中 $P, Q$ 为待定系数。根据韦达定理有 $z_A + z_B + z_C = 0$。
考量正三角形边长的代数表达：
\[
|AB|^2 = (x_A - x_B)^2 + (y_A - y_B)^2 = \frac{1}{4}(y_A^2 - y_B^2)^2 + (y_A - y_B)^2 = (y_A - y_B)^2 \left[ 1 + \frac{1}{4}(y_A + y_B)^2 \right].
\]
将 $y$ 替换为平移后的 $z$：$y_A - y_B = z_A - z_B$，$y_A + y_B = z_A + z_B + 2Y_0 = -z_C + 2Y_0$。
同时，$(z_A - z_B)^2 = (z_A + z_B)^2 - 4z_A z_B = z_C^2 - 4z_A z_B$。
由 $z_A z_B + z_B z_C + z_C z_A = P$，得 $z_A z_B = P - z_C(z_A + z_B) = P + z_C^2$。
故 $(z_A - z_B)^2 = z_C^2 - 4(P + z_C^2) = -4P - 3z_C^2$。
代入边长平方公式，得到仅关于 $z_C$ 的四次多项式表达：
\[
|AB|^2 = (-4P - 3z_C^2) \left[ 1 + \frac{1}{4}(2Y_0 - z_C)^2 \right] = -\frac{3}{4}z_C^4 + 3Y_0 z_C^3 - (P + 3Y_0^2 + 3)z_C^2 + 4Y_0 P z_C - 4P(Y_0^2 + 1).
\]
由于 $\triangle ABC$ 为正三角形，无论选取哪个顶点作为 $z_C$，边长平方必须为同一绝对常数。这意味着该四次多项式在模掉三次方程 $z^3 + Pz + Q = 0$ 进行降次后，所得的余式必须是一个零次多项式（常数）。
利用 $z^3 = -Pz - Q$ 与 $z^4 = -Pz^2 - Qz$ 代入降次：
\begin{align*}
|AB|^2 &= -\frac{3}{4}(-Pz_C^2 - Qz_C) + 3Y_0(-Pz_C - Q) - (P + 3Y_0^2 + 3)z_C^2 + 4Y_0 P z_C - 4P(Y_0^2 + 1) \\
&= \left( -\frac{1}{4}P - 3Y_0^2 - 3 \right)z_C^2 + \left( \frac{3}{4}Q + Y_0 P \right)z_C - 3Y_0 Q - 4P(Y_0^2 + 1).
\end{align*}
令二次项与一次项系数恒等于 $0$，解得代数约束：
\[
-\frac{1}{4}P - 3Y_0^2 - 3 = 0 \implies P = -12(Y_0^2 + 1).
\]
\[
\frac{3}{4}Q + Y_0 P = 0 \implies Q = -\frac{4}{3}Y_0 P = 16Y_0(Y_0^2 + 1).
\]
此时常数项即为边长的平方：
\[
|AB|^2 = -3Y_0 Q - 4P(Y_0^2 + 1) = -48Y_0^2(Y_0^2 + 1) + 48(Y_0^2 + 1)^2 = 48(Y_0^2 + 1).
\]
由正三角形外接圆半径公式 $R^2 = \frac{1}{3}|AB|^2$，得出半径平方的绝对约束：
\[
R^2 = 16(Y_0^2 + 1).
\]
回代参数比较四次方程系数。原 $y$ 的三次方程为 $(z+Y_0)^3 + P(z+Y_0) + Q = z^3 + 3Y_0 z^2 + (3Y_0^2 + P)z + (Y_0^3 + P Y_0 + Q) = 0$。
对比前文剥离出的 $y^3 - 3Y_0 y^2 + qy - r = 0$，其一次项系数为 $q = 9Y_0^2 + 4 - 4X_0$。
而展开变换后的 $y$ 一次项系数对应 $3Y_0^2 + P = 3Y_0^2 - 12Y_0^2 - 12 = -9Y_0^2 - 12$。
两者相等即 $9Y_0^2 + 4 - 4X_0 = -9Y_0^2 - 12$，解得圆心横坐标恒等式：
\[
X_0 = \frac{9}{2}Y_0^2 + 4.
\]

探求外接圆 $G$ 形成的包络曲线。
设单参数 $v = Y_0$，则圆 $G$ 的方程族为 $F(x,y,v) = 0$，代入 $X_0$ 与 $R^2$：
\[
\left(x - \frac{9}{2}v^2 - 4\right)^2 + (y - v)^2 = 16v^2 + 16.
\]
展开并按 $v$ 的降幂排列，得到：
\[
F(x,y,v) = \frac{81}{4}v^4 + (21 - 9x)v^2 - 2yv + x^2 + y^2 - 8x = 0.
\]
包络曲线方程等价于多项式 $F$ 关于参数 $v$ 的判别式为零，即 $F=0$ 与 $F_v=0$ 具有公共根。对 $v$ 求偏导：
\[
F_v = 81v^3 + 2(21 - 9x)v - 2y = 0.
\]
为避免直接展开四次判别式的计算爆炸，利用多项式除法进行降阶。作线性组合 $F - \frac{v}{4}F_v = 0$：
\[
\frac{1}{2}(21 - 9x)v^2 - \frac{3}{2}yv + x^2 + y^2 - 8x = 0 \implies 3(7 - 3x)v^2 - 3yv + 2(x^2 + y^2 - 8x) = 0 \quad \text{(记为 } Q_1=0 \text{)}.
\]
同时，对 $F_v$ 乘上因式 $3(7 - 3x)$ 并利用 $Q_1$ 进行二次降阶：
\[
3(7 - 3x)(81v^3) + 6(7 - 3x)^2 v - 6y(7 - 3x) = 0 \implies 81v \left[ 3yv - 2(x^2 + y^2 - 8x) \right] + 6(7 - 3x)^2 v - 6y(7 - 3x) = 0.
\]
整理后得到第二个二次方程：
\[
81y v^2 + \left[ 6(7 - 3x)^2 - 162(x^2 + y^2 - 8x) \right] v - 6y(7 - 3x) = 0.
\]
化简一次项系数，并对等式同除以 $3$，得：
\[
27y v^2 + 2( -18x^2 - 27y^2 + 174x + 49 ) v - 2y(7 - 3x) = 0 \quad \text{(记为 } Q_2=0 \text{)}.
\]
提取两个二次方程的系数：$A_1 = 3(7 - 3x), B_1 = -3y, C_1 = 2(x^2 + y^2 - 8x)$ 与 $A_2 = 27y, B_2 = 2( -18x^2 - 27y^2 + 174x + 49 ), C_2 = -2y(7 - 3x)$。
两个二次方程存在公共根的充要条件为结式为零，通过交叉相乘消元可得：
\[
(A_1 C_2 - A_2 C_1)^2 = (A_1 B_2 - A_2 B_1)(B_1 C_2 - B_2 C_1).
\]
将各系数代入并展开代数运算：
\begin{align*}
A_1 C_2 - A_2 C_1 &= -6y(7 - 3x)^2 - 54y(x^2 + y^2 - 8x) \\
&= -6y \left( 9x^2 + 27x^2 + 27y^2 - 42x - 216x + 49 \right) \\
&= -6y(36x^2 + 27y^2 - 258x + 49)\\
A_1 B_2 - A_2 B_1 &= 6(7 - 3x)( -18x^2 - 27y^2 + 174x + 49 ) + 81y^2 \\
&= 3( 108x^3 + 162xy^2 - 1296x^2 - 297y^2 + 2142x + 686 ).\\
B_1 C_2 - B_2 C_1 &= 6y^2(7 - 3x) - 4( -18x^2 - 27y^2 + 174x + 49 )(x^2 + y^2 - 8x) \\
&= 2( 36x^4 + 54y^4 + 90x^2 y^2 - 636x^3 - 789xy^2 + 2686x^2 - 77y^2 + 784x )
\end{align*}
将其整体代回结式条件，消去常数因子，得到圆 $G$ 所形成的包络曲线方程为：
\begin{align*}
6 y^2 (36x^2 + 27y^2 - 258x + 49)^2 = ( 108x^3 + 162xy^2 - 1296x^2 - 297y^2 + 2142x + 686 ) \\
\times ( 36x^4 + 54y^4 + 90x^2 y^2 - 636x^3 - 789xy^2 + 2686x^2 - 77y^2 + 784x ).
\end{align*}
该六次代数曲线即为所求之包络方程。推理完备，结论闭环。
\end{solution}





\begin{example}{}{}
 $A,B,C,D$ 为二次曲线 $\tau: ax^2+by^2=1$ 上四个不同的点, 直线 $AB,BD,AC$ 分别交 $x$ 轴于定点 $M(m,0),N(n,0),N'(-n,0)$，直线 $AC,BD$ 相交于点 $P$；\\
(1) 求证：直线 $CD$ 过点 $M'(-m,0)$；\\
(2) 求证：点 $P$ 在曲线 $\sigma: am^2(am^2-1)x^2+bm^2(an^2-1)y^2=n^2(am^2-1)$ 上；\\
(3) 求证：直线 $BC$ 与曲线 $\varepsilon: a(an^2-1)(am^2-1)x^2+b(amn+1)^2y^2=(am^2-1)(an^2-1)$ 相切；\\
(4) 求证：直线 $AD$ 与曲线 $\xi: a(an^2-1)(am^2-1)x^2+b(amn-1)^2y^2=(am^2-1)(an^2-1)$ 相切；\\
(5) 记直线 $AB,CD$ 的斜率分别为 $k_1,k_2$，求证：$(an^2-1)((k_1-k_2)^2-an^2(k_1+k_2)^2)=4bn^2(am^2-1)k_1^2k_2^2$；
\end{example}
\begin{solution}
首先，应用四交点曲线系原理推导直线 $CD$ 的定点性质。
设二次曲线 $\tau$ 的方程为 $ax^2 + by^2 - 1 = 0$。
已知点 $A,B,C,D$ 均在曲线 $\tau$ 上，且直线 $AB, AC, BD$ 分别经过定点 $M(m,0), N'(-n,0), N(n,0)$。
设直线 $CD$ 交 $x$ 轴于点 $M'(m',0)$。
四条直线的方程分别可表示为：
$L_{AB}: x - m + p_1 y = 0$；
$L_{CD}: x - m' + p_4 y = 0$；
$L_{AC}: x + n + p_2 y = 0$；
$L_{BD}: x - n + p_3 y = 0$。
根据四交点曲线系原理，退化二次曲线 $L_{AB} \cdot L_{CD} = 0$ 与 $L_{AC} \cdot L_{BD} = 0$ 共点张成的曲线束必然包含曲线 $\tau$。故存在非零实数 $\lambda, \mu$，使得全平面多项式恒等式成立：
\[ (x - m + p_1 y)(x - m' + p_4 y) - \lambda (x + n + p_2 y)(x - n + p_3 y) \equiv \mu (ax^2 + by^2 - 1). \]
将该恒等式限制于 $y = 0$ 的 $x$ 轴上，代入得到关于 $x$ 的多项式恒等式：
\[ (x - m)(x - m') - \lambda (x + n)(x - n) \equiv \mu (ax^2 - 1). \]
对比等式两端 $x$ 的一次项系数：右侧不含一次项，其系数为 $0$；左侧展开后系数为 $-(m + m')$。
因此必有 $m + m' = 0$，解得 $m' = -m$。由此可得，直线 $CD$ 恒过定点 $M'(-m,0)$。\\
其次，展开全局恒等式并提取各项系数，推导对角线交点 $P$ 的轨迹方程。
将 $m' = -m$ 代回原恒等式，并在平面内完全展开两侧多项式。对比同类项系数，建立如下代数方程组：
$x^2$ 的系数：$1 - \lambda = \mu a$；
常数项：$-m^2 + \lambda n^2 = -\mu$；
$xy$ 的系数：$p_1 + p_4 - \lambda(p_2 + p_3) = 0$；
$y$ 的系数：$m(p_1 - p_4) - \lambda n(p_3 - p_2) = 0$；
$y^2$ 的系数：$p_1 p_4 - \lambda p_2 p_3 = \mu b$。
由前两式消去 $\lambda$ 可得 $1 - \mu a = \lambda$，进而 $-m^2 + (1 - \mu a)n^2 = -\mu$，化简得 $\mu(an^2 - 1) = n^2 - m^2$，即 $\mu = \frac{n^2 - m^2}{an^2 - 1}$。
将其代回得 $\lambda = \frac{am^2 - 1}{an^2 - 1}$。
设点 $P(x,y)$ 为直线 $AC, BD$ 的交点。代入直线方程解得 $p_2 = -\frac{x+n}{y}, p_3 = -\frac{x-n}{y}$。
由此计算可得：$p_2 + p_3 = -\frac{2x}{y}$，$p_3 - p_2 = \frac{2n}{y}$，$p_2 p_3 = \frac{x^2 - n^2}{y^2}$。
由系数关系知 $p_1 + p_4 = \lambda(p_2 + p_3)$ 且 $p_1 - p_4 = \frac{\lambda n}{m}(p_3 - p_2)$。
利用完全平方差公式 $4 p_1 p_4 = (p_1 + p_4)^2 - (p_1 - p_4)^2$，代入可得：
\[ 4(\mu b + \lambda p_2 p_3) = \lambda^2 (p_2 + p_3)^2 - \lambda^2 \frac{n^2}{m^2} (p_3 - p_2)^2. \]
将包含 $P$ 点坐标的表达式代入其中：
\[ 4\mu b + 4\lambda \frac{x^2 - n^2}{y^2} = \lambda^2 \frac{4x^2}{y^2} - \lambda^2 \frac{n^2}{m^2} \frac{4n^2}{y^2}. \]
等式两端同乘 $\frac{y^2}{4}$ 并展开，重组关于 $x^2, y^2$ 的项：
\[ (\lambda^2 - \lambda)x^2 - \mu b y^2 = \lambda^2 \frac{n^4}{m^2} - \lambda n^2. \]
利用已知关系 $\lambda^2 - \lambda = \lambda(\lambda - 1) = -\lambda \mu a$，以及右侧 $\lambda^2 \frac{n^4}{m^2} - \lambda n^2 = \lambda \frac{n^2}{m^2}(\lambda n^2 - m^2) = -\lambda \mu \frac{n^2}{m^2}$。
代入方程并同除以 $-\mu$，得到 $\lambda a x^2 + b y^2 = \lambda \frac{n^2}{m^2}$。
两端同乘 $m^2(an^2 - 1)$ 并代入 $\lambda = \frac{am^2 - 1}{an^2 - 1}$，即得点 $P$ 的轨迹方程：
\[ am^2(am^2 - 1)x^2 + bm^2(an^2 - 1)y^2 = n^2(am^2 - 1). \]
该方程精确等同于曲线 $\sigma$，结论成立。\\
另一方面，利用代数恒等式的对称性证明斜率关联式。
由于几何构型的完全对称性，将直线对 $(AB, CD)$ 与 $(AC, BD)$ 互换，等价于将参数 $(m, p_1, p_4)$ 与 $(-n, p_2, p_3)$ 互换。
此时曲线系参数 $\lambda$ 映射为 $\lambda' = \frac{a(-n)^2 - 1}{am^2 - 1} = \frac{1}{\lambda}$，$\mu$ 映射为 $\mu' = \frac{(-n)^2 - m^2}{am^2 - 1} = -\frac{\mu}{\lambda}$。
在前述推导中，关于 $p_2, p_3$ 的恒等式为 $4\mu b + 4\lambda p_2 p_3 = \lambda^2 (p_2 + p_3)^2 - \lambda^2 \frac{n^2}{m^2} (p_3 - p_2)^2$。
应用对称性替换后，得到关于 $p_1, p_4$ 的恒等式：
\[ -4\frac{\mu}{\lambda} b + \frac{4}{\lambda} p_1 p_4 = \frac{1}{\lambda^2} (p_1 + p_4)^2 - \frac{1}{\lambda^2} \frac{m^2}{n^2} (p_1 - p_4)^2. \]
两端同乘 $\lambda^2$ 并代入 $4\lambda p_1 p_4 = \lambda(p_1 + p_4)^2 - \lambda(p_1 - p_4)^2$，整理得：
\[ -4\mu \lambda b + \lambda(p_1 + p_4)^2 - \lambda(p_1 - p_4)^2 = (p_1 + p_4)^2 - \frac{m^2}{n^2}(p_1 - p_4)^2. \]
移项合并同类项：
\[ \left(\frac{m^2}{n^2} - \lambda\right)(p_1 - p_4)^2 - (1 - \lambda)(p_1 + p_4)^2 = 4\mu \lambda b. \]
运用关系式 $\frac{m^2}{n^2} - \lambda = \frac{m^2 - \lambda n^2}{n^2} = \frac{\mu}{n^2}$ 及 $1 - \lambda = \mu a$，代入并约去非零常数 $\mu$：
\[ \frac{1}{n^2}(p_1 - p_4)^2 - a(p_1 + p_4)^2 = 4\lambda b. \]
已知直线 $AB, CD$ 的斜率分别为 $k_1 = -\frac{1}{p_1}$， $k_2 = -\frac{1}{p_4}$。由此转化得 $p_1 - p_4 = \frac{k_1 - k_2}{k_1 k_2}$ 且 $p_1 + p_4 = \frac{-(k_1 + k_2)}{k_1 k_2}$。
代入方程可得 $\frac{1}{n^2} \frac{(k_1 - k_2)^2}{k_1^2 k_2^2} - a \frac{(k_1 + k_2)^2}{k_1^2 k_2^2} = 4b \frac{am^2 - 1}{an^2 - 1}$。
等式两端同乘 $n^2 k_1^2 k_2^2 (an^2 - 1)$，即得到目标斜率关联式：
\[ (an^2 - 1) \left[ (k_1 - k_2)^2 - an^2(k_1 + k_2)^2 \right] = 4bn^2 (am^2 - 1) k_1^2 k_2^2. \]
接着，对于相切包络问题，若坚持使用纯四交点曲线系的系数比对，求切线条件会产生高次耦合方程，且在分离参数时会产生难以界定的正负号歧义。因此，我们完善算子理论，将算子系统与二次曲线的有理参数化结合进行推导。
作坐标伸缩变换 $X = \sqrt{a}x, Y = \sqrt{b}y$，二次曲线 $\tau$ 转化为单位圆 $X^2 + Y^2 = 1$。
引入算子参数微元 $t = \tan \frac{\theta}{2}$，圆上动点坐标为 $\left(\frac{1-t^2}{1+t^2}, \frac{2t}{1+t^2}\right)$。
对于过 $X$ 轴定点 $(d, 0)$ 的任意动弦，其两端点参数必满足 $t_i t_j = \frac{d - 1}{d + 1}$。
设 $K_m = \frac{\sqrt{a}m - 1}{\sqrt{a}m + 1}$， $K_n = \frac{\sqrt{a}n - 1}{\sqrt{a}n + 1}$。
由题设直线 $AB$ 过 $M(\sqrt{a}m, 0)$，直线 $AC$ 过 $N'(-\sqrt{a}n, 0)$，直线 $BD$ 过 $N(\sqrt{a}n, 0)$，可得端点参数约束：
$t_1 t_2 = K_m$，$t_1 t_3 = \frac{-\sqrt{a}n - 1}{-\sqrt{a}n + 1} = \frac{1}{K_n}$，$t_2 t_4 = K_n$。\\
对于直线 $BC$，其端点参数为 $t_2, t_3$。计算其乘积与和：
\[ t_2 t_3 = \frac{K_m}{t_1} \frac{1}{K_n t_1} = \frac{K_m}{K_n t_1^2}, \quad t_2 + t_3 = \frac{K_m}{t_1} + \frac{1}{K_n t_1} = \frac{K_m K_n + 1}{K_n t_1}. \]
消去自由度 $t_1$，得到包络约束式 $(t_2 + t_3)^2 = \frac{(K_m K_n + 1)^2}{K_m K_n} t_2 t_3$。记常量 $C_1 = \frac{(K_m K_n + 1)^2}{K_m K_n}$。
动弦 $BC$ 的方程为 $\frac{1 - t_2 t_3}{1 + t_2 t_3} X + \frac{t_2 + t_3}{1 + t_2 t_3} Y = 1$。
该直线与中心二次曲线 $X^2 + \beta_1 Y^2 = 1$ 相切的充要条件为对应系数满足：
\[ (1 - t_2 t_3)^2 + \frac{(t_2 + t_3)^2}{\beta_1} = (1 + t_2 t_3)^2 \implies \frac{C_1 t_2 t_3}{\beta_1} = 4 t_2 t_3 \implies \beta_1 = \frac{C_1}{4}. \]
计算包络系数常数：
\[ \frac{C_1}{4} = \frac{(K_m K_n + 1)^2}{4 K_m K_n} = \frac{4(amn + 1)^2}{4(amn + 1 - \sqrt{a}(m+n))(amn + 1 + \sqrt{a}(m+n))} = \frac{(amn + 1)^2}{(amn + 1)^2 - a(m+n)^2}. \]
展开分母 $(amn + 1)^2 - a(m+n)^2 = a^2 m^2 n^2 - am^2 - an^2 + 1 = (am^2 - 1)(an^2 - 1)$。
故包络曲线在 $(X, Y)$ 坐标下为 $X^2 + \frac{(amn + 1)^2}{(am^2 - 1)(an^2 - 1)} Y^2 = 1$。
作逆变换还原至 $(x, y)$ 坐标，两端同乘分母即得直线 $BC$ 恒相切的曲线 $\varepsilon$：
\[ a(an^2 - 1)(am^2 - 1)x^2 + b(amn + 1)^2 y^2 = (am^2 - 1)(an^2 - 1). \]
特别地，对于直线 $AD$，其端点参数为 $t_1, t_4$。
计算其乘积与和：
\[ t_1 t_4 = t_1 \frac{K_n}{t_2} = t_1 \frac{K_n}{K_m / t_1} = \frac{K_n}{K_m} t_1^2, \quad t_1 + t_4 = t_1 + \frac{K_n}{K_m} t_1 = \frac{K_m + K_n}{K_m} t_1. \]
消去 $t_1$ 得到约束式 $(t_1 + t_4)^2 = \frac{(K_m + K_n)^2}{K_m K_n} t_1 t_4$。记常量 $C_2 = \frac{(K_m + K_n)^2}{K_m K_n}$。
同理，其包络曲线对应的系数 $\beta_2 = \frac{C_2}{4}$。
计算可得：
\[ \frac{C_2}{4} = \frac{(K_m + K_n)^2}{4 K_m K_n} = \frac{4(amn - 1)^2}{4(amn - \sqrt{a}(m+n) + 1)(amn + \sqrt{a}(m+n) + 1)} = \frac{(amn - 1)^2}{(am^2 - 1)(an^2 - 1)}. \]
作逆变换还原至 $(x, y)$ 坐标，即得直线 $AD$ 恒相切的曲线 $\xi$：
\[ a(an^2 - 1)(am^2 - 1)x^2 + b(amn - 1)^2 y^2 = (am^2 - 1)(an^2 - 1). \]
至此，所有代数推导均严格闭合，结论完全成立。
\end{solution}





\begin{example}{}{}
已知过点 $M(m,0)$ 的直线交二次曲线 $\tau: ax^2+by^2=1$ 于 $A_1,B_1$ 两点，延长直线 $A_1Q_1$ 交曲线 $\tau$ 于点 $A_2$，延长直线 $A_2Q_2$ 交曲线 $\tau$ 于点 $A_3$，以此类推，延长直线 $A_kQ_k,(k=1,2,\dots,n)$ 交曲线 $\tau$ 于点 $A_{k+1}$，同理延长直线 $B_1Q_1'$ 交曲线 $\tau$ 于点 $B_2$，延长直线 $B_2Q_2'$ 交曲线 $\tau$ 于点 $B_3$，以此类推，延长 $B_kQ_k',(k=1,2,\dots,n)$ 交曲线 $\tau$ 于点 $B_{k+1}$，记直线 $NA_n,N'B_n$ 相交于点 $P$，（其中 $M(m,0),N(n,0),N'(-n,0)$，点 $Q_k,Q_k',(k=1,2,\dots,n)$ 关于 $y$ 轴对称），求证：点 $P$ 在曲线 $\sigma: am^2(am^2-1)x^2+bm^2(an^2-1)y^2=n^2(am^2-1)$ 上；
\end{example}
\begin{solution}
首先，应用交点曲线系算子分析迭代过程的对称不变量，确立直线截距的代数传递法则。
设二次曲线的算子方程为 $\tau(x, y) = a x^2 + b y^2 - 1 = 0$。
初始直线 $A_1 B_1$ 经过 $x$ 轴上的定点 $M(m,0)$。设递推至第 $k$ 步时，直线 $A_k B_k$ 过 $x$ 轴上的点 $M_k(m_k, 0)$，其一次多项式方程为 $L_k(x, y) = u_k(x - m_k) + v_k y = 0$。
同理，下一代直线 $A_{k+1} B_{k+1}$ 的方程为 $L_{k+1}(x, y) = u_{k+1}(x - m_{k+1}) + v_{k+1} y = 0$。
由于曲线 $\tau$ 穿过 $A_k, B_k, A_{k+1}, B_{k+1}$ 四点，根据交点曲线系定理，必然存在非零常数 $\alpha, \beta$，使得如下多项式恒等式在全平面成立：
$$ \tau(x, y) - \alpha L_k(x, y) L_{k+1}(x, y) \equiv \beta L_{A_k A_{k+1}}(x, y) L_{B_k B_{k+1}}(x, y) $$
已知点 $Q_k(q_x, q_y)$ 与 $Q'_k(-q_x, q_y)$ 分别位于直线 $A_k A_{k+1}$ 与 $B_k B_{k+1}$ 上。将点 $(q_x, q_y)$ 与 $(-q_x, q_y)$ 分别代入上述恒等式，等式右端均归零。
由曲线 $\tau$ 关于 $y$ 轴的对称性可得 $\tau(q_x, q_y) = \tau(-q_x, q_y)$，从而导出关系式：
$$ L_k(q_x, q_y) L_{k+1}(q_x, q_y) = L_k(-q_x, q_y) L_{k+1}(-q_x, q_y) $$
代入坐标表达式并展开：
$$ \big[u_k(q_x - m_k) + v_k q_y\big] \big[u_{k+1}(q_x - m_{k+1}) + v_{k+1} q_y\big] = \big[u_k(-q_x - m_k) + v_k q_y\big] \big[u_{k+1}(-q_x - m_{k+1}) + v_{k+1} q_y\big] $$
令 $U_k = -u_k m_k + v_k q_y$，$U_{k+1} = -u_{k+1} m_{k+1} + v_{k+1} q_y$。等式化为：
$$ (u_k q_x + U_k)(u_{k+1} q_x + U_{k+1}) = (-u_k q_x + U_k)(-u_{k+1} q_x + U_{k+1}) $$
展开并移项合并后得 $2 q_x (u_k U_{k+1} + u_{k+1} U_k) = 0$。
若轨迹具有确定封闭性，对于非退化的 $q_x \neq 0$，必须恒有 $u_k U_{k+1} + u_{k+1} U_k = 0$。
将其完全代入展开可得：
$$ -u_k u_{k+1} (m_k + m_{k+1}) + q_y (u_k v_{k+1} + u_{k+1} v_k) = 0 $$
由于 $u_k, v_k, u_{k+1}, v_{k+1}$ 依赖于动点在曲线上的位置而不断变化，要使该递推关系独立于前驱状态，从而保证存在确定的定点 $m_{k+1}$，必须在代数上满足 $q_y = 0$。
此结论表明，为了使最终轨迹客观存在，点 $Q_k$ 必须位于 $x$ 轴上。在此约束下，递推关系严格坍缩为 $m_{k+1} = -m_k$。
因此，经过 $n-1$ 次迭代后，直线 $A_n B_n$ 恒过 $x$ 轴上的定点 $M^*(m^*, 0)$，其中 $m^* = (-1)^{n-1}m$。

其次，运用四点曲线系多项式，证明交点投影在 $x$ 轴上的对称对应性。
设直线 $P N$ 交 $\tau$ 于点 $A'$（即 $A' = A_n$），直线 $P N'$ 交 $\tau$ 于点 $B'$。我们需要确定直线 $A' B'$ 过 $x$ 轴的定点位置。
考虑 $\tau$ 上的四点 $A_n, B_n, A', B'$。构建相应的交点曲线系恒等式：
$$ \tau(x, y) - \alpha L_{A_n A'}(x, y) L_{B_n B'}(x, y) \equiv \beta L_{A_n B_n}(x, y) L_{A' B'}(x, y) $$
将该多项式恒等式限制在 $x$ 轴上（即令 $y=0$），此时 $\tau(x, 0) = a x^2 - 1$。
已知直线 $A_n A'$ 即为 $P N$，过点 $N(n,0)$，其 $x$ 轴截距式为 $c_1(x-n)$；
直线 $B_n B'$ 即为 $P N'$，过点 $N'(-n,0)$，其 $x$ 轴截距式为 $c_2(x+n)$；
直线 $A_n B_n$ 过定点 $M^*(m^*, 0)$，其 $x$ 轴截距式为 $c_3(x-m^*)$。
设直线 $A' B'$ 在 $x$ 轴上的截距式为 $c_4 x + c_5$。代入恒等式得：
$$ a x^2 - 1 - \alpha c_1 c_2 (x^2 - n^2) \equiv \beta c_3 (x - m^*) (c_4 x + c_5) $$
观察等式左端，仅含有 $x^2$ 项与常数项，一次项 $x$ 的系数为 $0$。
展开等式右端，其一次项系数为 $\beta c_3 (c_5 - c_4 m^*)$。由于等式两端系数必须绝对对应相等，故 $c_5 - c_4 m^* = 0$，即 $c_5 = c_4 m^*$。
因此，$L_{A' B'}(x, 0) = c_4(x + m^*)$。这严格证明了直线 $A' B'$ 必然经过定点 $M^{**}(-m^*, 0)$。

接下来，建立全域四点曲线系方程，通过系数量化提取交点 $P(x_0, y_0)$ 的代数关系。
构建经过 $A_n, A', B_n, B'$ 的退化二次曲线对：
线对一（由点 $P(x_0, y_0)$ 和 $N, N'$ 确定的两条直线）：
$L_1(x, y) = y_0 x - (x_0-n) y - n y_0$；
$L_2(x, y) = y_0 x - (x_0+n) y + n y_0$。
线对二（由定点 $M^*, -M^*$ 确定的两条弦）：
$L_3(x, y) = y - k_3 (x - m^*)$；
$L_4(x, y) = y - k_4 (x + m^*)$。
存在常数 $\lambda, \mu$ 使得下述多项式恒等：
$$ \tau(x, y) - \lambda L_1(x, y) L_2(x, y) \equiv \mu L_3(x, y) L_4(x, y) $$
等式左端代数展开（将动点坐标记为 $x, y$ 以免混淆）：
$L_1 L_2 = y_0^2 x^2 - 2 x_0 y_0 x y + (x_0^2 - n^2) y^2 + 2 n^2 y_0 y - n^2 y_0^2$。
等式右端代数展开：
$\mu L_3 L_4 = \mu k_3 k_4 x^2 - \mu (k_3 + k_4) x y + \mu y^2 + \mu m^*(k_3 - k_4) y - \mu k_3 k_4 (m^*)^2$。
严格比较两端各项系数：
\begin{itemize}
\item 二次项 $x^2$ 系数：$a - \lambda y_0^2 = \mu k_3 k_4$；
\item 二次项 $y^2$ 系数：$b - \lambda (x_0^2 - n^2) = \mu$；
\item 交叉项 $x y$ 系数：$2 \lambda x_0 y_0 = \mu (k_3 + k_4)$；
\item 一次项 $y$ 系数：$-2 \lambda n^2 y_0 = \mu m^* (k_3 - k_4)$；
\item 常数项：$-1 + \lambda n^2 y_0^2 = -\mu k_3 k_4 (m^*)^2$。
\end{itemize}
最后，执行代数消元，坍缩生成最终的封闭轨迹。
由等式1和5联立：
$-1 + \lambda n^2 y_0^2 = -(a - \lambda y_0^2) (m^*)^2 = -a (m^*)^2 + \lambda (m^*)^2 y_0^2$。
移项解得核心常数 $\lambda$：
$$ \lambda = \frac{a (m^*)^2 - 1}{y_0^2 \big((m^*)^2 - n^2\big)} $$
由等式2可得 $\mu = b - \lambda (x_0^2 - n^2)$。
由等式1可得 $\mu k_3 k_4 = a - \lambda y_0^2 = a - \frac{a (m^*)^2 - 1}{(m^*)^2 - n^2} = \frac{1 - a n^2}{(m^*)^2 - n^2}$。
利用代数恒等式 $(k_3 + k_4)^2 - (k_3 - k_4)^2 = 4 k_3 k_4$，消除未知斜率 $k_3, k_4$：
$$ \left( \frac{2 \lambda x_0 y_0}{\mu} \right)^2 - \left( \frac{-2 \lambda n^2 y_0}{\mu m^*} \right)^2 = \frac{4(1 - a n^2)}{\mu \big((m^*)^2 - n^2\big)} $$
等式两端同乘 $\mu^2 (m^*)^2 / 4$，并利用 $(m^*)^2 = m^2$ 代换：
$$ \lambda^2 y_0^2 (m^2 x_0^2 - n^4) = \mu m^2 \frac{1 - a n^2}{m^2 - n^2} $$
将 $\lambda$ 与 $\mu$ 表达式代入：
$$ \frac{(a m^2 - 1)^2}{y_0^2 (m^2 - n^2)^2} (m^2 x_0^2 - n^4) = \left[ b - \frac{a m^2 - 1}{y_0^2 (m^2 - n^2)} (x_0^2 - n^2) \right] m^2 \frac{1 - a n^2}{m^2 - n^2} $$
等式两端同乘 $y_0^2 (m^2 - n^2)^2$ 消除所有分母：
$$ (a m^2 - 1)^2 (m^2 x_0^2 - n^4) = m^2 (1 - a n^2) \big[ b y_0^2 (m^2 - n^2) - (a m^2 - 1)(x_0^2 - n^2) \big] $$
将含有 $x_0^2$ 的项全部移至等式左端：
$$ (a m^2 - 1) \big[ (a m^2 - 1)(m^2 x_0^2 - n^4) + m^2 (1 - a n^2)(x_0^2 - n^2) \big] = b m^2 y_0^2 (m^2 - n^2) (1 - a n^2) $$
对左端中括号内部展开合并：
$= a m^4 x_0^2 - a m^2 n^4 - m^2 x_0^2 + n^4 + m^2 x_0^2 - m^2 n^2 - a m^2 n^2 x_0^2 + a m^2 n^4$
$= a m^2 x_0^2 (m^2 - n^2) - n^2 (m^2 - n^2) = (m^2 - n^2) (a m^2 x_0^2 - n^2)$。
将此结果代回主方程，并约去等式两端非零的公因子 $(m^2 - n^2)$：
$$ (a m^2 - 1) (a m^2 x_0^2 - n^2) = b m^2 y_0^2 (1 - a n^2) $$
展开左端并将包含坐标变量的项归集整理：
$$ a m^2 (a m^2 - 1) x_0^2 + b m^2 (a n^2 - 1) y_0^2 = n^2 (a m^2 - 1) $$
将定点 $P(x_0, y_0)$ 泛化为轨迹动点坐标 $(x,y)$，得到最终方程：
$$ a m^2 (a m^2 - 1) x^2 + b m^2 (a n^2 - 1) y^2 = n^2 (a m^2 - 1) $$
此纯代数推演结果与题设曲线 $\sigma$ 的方程完全一致。命题得证。
\end{solution}


\begin{example}{}{}
 $P$ 是圆 $O: x^2 + y^2 = 1$ 上一动点，$A(a,0),(a>1)$ 为 $x$ 轴上一定点，以 $PA$ 为直径的圆记为圆 $M$；求证：圆 $M$ 与曲线 $G: (x-a)^2 + y^2 = (x^2 + y^2 - ax)^2$ 相切；
\end{example}
\begin{solution}
首先，利用内积与外积算子重构曲线的代数方程。
已知动点 $P(x_0, y_0)$ 位于圆 $O: x^2 + y^2 = 1$ 上，其坐标满足 $x_0^2 + y_0^2 = 1$。
以定点 $A(a,0)$ 与动点 $P(x_0, y_0)$ 为直径端点的圆记为 $M$。平面内任意一点 $(x,y)$ 在圆 $M$ 上，等价于其与两端点连线的向量相互垂直，内积为零：
\[ (x - a)(x - x_0) + y(y - y_0) = 0. \]
将其展开并重新组合，定义圆 $M$ 的内积多项式算子 $F(x,y)$：
\[ F(x,y) = x^2 + y^2 - (a + x_0)x - y_0 y + a x_0. \]
圆 $M$ 的方程即为 $F(x,y) = 0$。
其次，寻找圆 $M$ 的算子与曲线 $G$ 的代数联系。
曲线 $G$ 的方程为 $(x - a)^2 + y^2 = (x^2 + y^2 - ax)^2$。
观察等式右侧的底数部分 $x^2 + y^2 - ax$，将其分离出圆 $M$ 的算子 $F(x,y)$：
\[ x^2 + y^2 - ax = \left[x^2 + y^2 - (a + x_0)x - y_0 y + a x_0\right] + x_0 x + y_0 y - a x_0 = F(x,y) + x_0(x - a) + y_0 y. \]
定义一次线性算子 $L(x,y) = x_0(x - a) + y_0 y$。
曲线 $G$ 等号右侧可化为 $(F(x,y) + L(x,y))^2$。
对于等号左侧的 $(x - a)^2 + y^2$，定义外积算子 $H(x,y) = x_0 y - y_0(x - a)$。
根据算子的拉格朗日恒等式展开 $L(x,y)^2 + H(x,y)^2$：
\[ L(x,y)^2 + H(x,y)^2 = \left[x_0(x - a) + y_0 y\right]^2 + \left[x_0 y - y_0(x - a)\right]^2 = (x_0^2 + y_0^2)\left[(x - a)^2 + y^2\right]. \]
代入约束条件 $x_0^2 + y_0^2 = 1$，得到：
\[ L(x,y)^2 + H(x,y)^2 = (x - a)^2 + y^2. \]
这恰好是曲线 $G$ 等号左侧的表达式。
将左右两侧的等效表达式代回曲线 $G$ 的方程，得到：
\[ L(x,y)^2 + H(x,y)^2 = (F(x,y) + L(x,y))^2. \]
展开完全平方式，并消去两侧的 $L(x,y)^2$，得到曲线 $G$ 的全局等效算子恒等式：
\[ J(x,y) \equiv F(x,y)^2 + 2F(x,y)L(x,y) - H(x,y)^2 = 0. \]
另一方面，结合重构的恒等式，求解圆 $M$ 与曲线 $G$ 的公共交点。
两曲线的交点必同时满足圆 $M$ 的方程 $F(x,y) = 0$ 与曲线 $G$ 的方程 $J(x,y) = 0$。
将 $F(x,y) = 0$ 代入 $J(x,y) = 0$ 中，得出 $-H(x,y)^2 = 0$，即 $H(x,y) = 0$。
方程 $H(x,y) = x_0 y - y_0(x - a) = 0$ 表示一条恒过定点 $A(a,0)$ 且方向向量为 $(x_0, y_0)$ 的直线。
设该直线上动点的参数化坐标为 $x - a = t x_0$ 且 $y = t y_0$（$t$ 为伸缩比例参数）。
将其代入圆 $M$ 的算子方程 $F(x,y) = (x - a)(x - x_0) + y(y - y_0) = 0$ 中：
\[ t x_0(t x_0 + a - x_0) + t y_0(t y_0 - y_0) = 0. \]
提取公因式 $t$ 并合并关于 $t$ 的同类项：
\[ t \left[ t(x_0^2 + y_0^2) + a x_0 - (x_0^2 + y_0^2) \right] = 0. \]
再次代入 $x_0^2 + y_0^2 = 1$，方程化简为：
\[ t(t + a x_0 - 1) = 0. \]
解得参数根 $t = 0$ 或 $t = 1 - a x_0$。
当 $t = 0$ 时，对应的交点为定点 $A(a,0)$。
当 $t = 1 - a x_0$ 时，对应的另一交点为 $Q(x_Q, y_Q)$，其相对坐标满足 $x_Q - a = (1 - a x_0)x_0$ 且 $y_Q = (1 - a x_0)y_0$。\\
针对 $1 - a x_0 \neq 0$ 的情形，通过代数多项式的平移展开提取一次齐次项，论证两曲线在交点 $Q$ 处的相切属性。
在多项式代数曲线中，将坐标系原点平移至曲线上某一点后，其展开方程的一次齐次多项式即为曲线在该点处的切线方程。
引入以交点 $Q$ 为原点的平移局部坐标 $u = x - x_Q, v = y - y_Q$。
将圆的代数算子 $F(x,y)$ 在 $Q$ 处展开。由于 $Q$ 位于圆上，$F(Q) = 0$，常数项为零，展开式记为 $F(x,y) = T_F(u,v) + H_F(u,v)$。其中 $T_F(u,v)$ 为交点处的一次切线方程多项式，$H_F(u,v) = u^2 + v^2$ 为二次项。
将一次线性算子 $L(x,y)$ 在 $Q$ 处展开：
\[ L(x,y) = L(x_Q, y_Q) + L(u,v). \]
计算其中的常数项 $L(Q) = x_0(x_Q - a) + y_0 y_Q = x_0(1 - a x_0)x_0 + y_0(1 - a x_0)y_0 = (1 - a x_0)(x_0^2 + y_0^2) = 1 - a x_0$。
将外积算子 $H(x,y)$ 在 $Q$ 处展开。由于 $Q$ 满足直线方程 $H(Q) = 0$，其展开式仅含有一次项 $T_H(u,v)$。
将这些展开式代回曲线 $G$ 的恒等式中：
\[ J(u,v) = (T_F + H_F)^2 + 2(T_F + H_F)(1 - a x_0 + L(u,v)) - T_H(u,v)^2 = 0. \]
提取展开式中关于 $u, v$ 的全体一次齐次项，记为曲线 $G$ 在 $Q$ 点的切线多项式 $T_J(u,v)$。
由于 $T_F^2$ 与 $T_H^2$ 的最低次数均为二次，一次项仅由常数与一次项的交叉乘积产生：
\[ T_J(u,v) = 2(1 - a x_0) T_F(u,v). \]
由前提 $1 - a x_0 \neq 0$ 知常数因子非零。这表明曲线 $G$ 在交点 $Q$ 处的一次切线多项式严格等于圆 $M$ 切线多项式的非零常数倍。
两个多项式的一次项成比例，代数上严格等价于两曲线在该点共享完全相同的切线。\\
特别地，讨论 $1 - a x_0 = 0$ 时的退化情形。
此时 $x_0 = \frac{1}{a}$。因给定条件 $a > 1$，动点纵坐标 $y_0 = \pm\frac{\sqrt{a^2 - 1}}{a} \neq 0$。
交点 $Q$ 退化重合于定点 $A(a,0)$。
由于常数项 $L(A) = 1 - a(1/a) = 0$，曲线 $G$ 在点 $A$ 的平移展开式中一次项 $T_J(u,v) \equiv 0$。这表明点 $A$ 为曲线 $G$ 的自交点。
自交点处的分支切线由展开式中的二次齐次部分 $Q_J(u,v) = 0$ 决定：
\[ Q_J(u,v) = T_F(u,v)^2 + 2 T_F(u,v) L(u,v) - T_H(u,v)^2. \]
其中 $T_F(u,v) = (a - x_0)u - y_0 v$ 为圆 $M$ 在点 $A$ 的一次切线多项式。
注意到 $T_F(u,v) + L(u,v) = (a - x_0)u - y_0 v + x_0 u + y_0 v = a u$，
且由拉格朗日恒等式， $L(u,v)^2 + T_H(u,v)^2 = (x_0^2 + y_0^2)(u^2 + v^2) = u^2 + v^2$。
代入二次部分进行恒等变形：
\begin{align*}
Q_J(u,v) &= (T_F + L)^2 - L^2 - T_H^2 \\
&= (a u)^2 - (u^2 + v^2) = (a^2 - 1)u^2 - v^2.
\end{align*}
曲线 $G$ 在自交点 $A$ 的切线分支由方程 $(a^2 - 1)u^2 - v^2 = 0$ 给出，即 $v = \pm\sqrt{a^2 - 1}u$。
将参数 $x_0 = \frac{1}{a}$ 与 $y_0 = \pm\frac{\sqrt{a^2 - 1}}{a}$ 代入圆 $M$ 在点 $A$ 的切线方程 $T_F(u,v) = 0$ 中：
\[ \left(a - \frac{1}{a}\right)u \mp \frac{\sqrt{a^2 - 1}}{a} v = 0 \implies \frac{a^2 - 1}{a}u = \pm\frac{\sqrt{a^2 - 1}}{a} v. \]
因 $a > 1$，约去非零系数化简得到 $v = \pm\sqrt{a^2 - 1}u$。
这证明即使在交点退化为自交点的情况下，圆 $M$ 的切线仍等价于高次曲线 $G$ 在该自交点处的一条分支切线，符合几何相切的代数准则。
综上所述，所有情形下圆 $M$ 均与曲线 $G$ 相切，结论成立。
\end{solution}

\begin{example}{}{}
已知椭圆 $O: \frac{x^2}{6} + \frac{y^2}{3} = 1$ 和曲线 $C_1: y = x - \frac{2}{x}$，椭圆 $O$ 与曲线 $C_1$ 在第一象限的交点为 $P$，圆 $G: (x+2)^2 + \left(y-\frac{9}{4}\right)^2 = \frac{25}{16}$ 在点 $Q$ 处的切线 $L$ 交曲线 $C_1$ 于 $A,B$ 两点，延长 $AP,BP$ 分别交椭圆 $O$ 于 $M,N$ 两点；\\
求证：直线 $MN$ 与椭圆 $E: 25x^2 + 8y^2 = 12$ 相切；
\end{example}
\begin{solution}
首先，求解椭圆 $O$ 与曲线 $C_1$ 在第一象限的交点 $P$ 的坐标。
已知椭圆 $O$ 的方程为 $\frac{x^2}{6} + \frac{y^2}{3} = 1$，化简为 $x^2 + 2y^2 = 6$。
曲线 $C_1$ 的方程为 $y = x - \frac{2}{x}$，等价于 $x^2 - xy - 2 = 0$。
将 $y = \frac{x^2 - 2}{x}$ 代入椭圆 $O$ 的方程中：
$$ x^2 + 2\left(\frac{x^2 - 2}{x}\right)^2 = 6 $$
等式两端同乘 $x^2$ 并展开：
$$ x^4 + 2(x^4 - 4x^2 + 4) = 6x^2 \Leftrightarrow 3x^4 - 14x^2 + 8 =(3x^2 - 2)(x^2 - 4) = 0 $$
解得 $x^2 = \frac{2}{3}$ 或 $x^2 = 4$。
由于点 $P$ 位于第一象限，故 $x > 0$ 且 $y > 0$。
若 $x^2 = \frac{2}{3}$，则 $x = \sqrt{\frac{2}{3}}$，此时 $y = \sqrt{\frac{2}{3}} - \sqrt{6} < 0$，不合题意。
若 $x^2 = 4$，则 $x = 2$，此时 $y = 2 - 1 = 1 > 0$，符合题意。
因此，交点 $P$ 的坐标为 $(2, 1)$。\\
其次，通过坐标平移应用齐次化方法，推导经过点 $P$ 的射线对的联合多项式方程。
将坐标系平移至点 $P(2, 1)$，引入局部坐标变换 $X = x - 2, Y = y - 1$。
在新的坐标系下，椭圆 $O$ 的方程转化为：
$$ (X + 2)^2 + 2(Y + 1)^2 = 6 \implies X^2 + 2Y^2 + 4X + 4Y = 0 $$
曲线 $C_1$ 的方程转化为：
$$ (X + 2)^2 - (X + 2)(Y + 1) - 2 = 0 \implies X^2 - XY + 3X - 2Y = 0 $$
设切线 $L$ 在该局部坐标系下的方程为 $uX + vY + 1 = 0$。由于直线 $L$ 交曲线 $C_1$ 于 $A, B$ 两点，利用 $1 = -uX - vY$ 对曲线 $C_1$ 的一次项进行齐次化，构造出经过原点（即点 $P$）且经过 $A, B$ 的射线对 $PA$ 与 $PB$ 的联合二次齐次方程：
$$ X^2 - XY + (3X - 2Y)(-uX - vY) = 0 $$
展开并合并同类项得：
$$ (1 - 3u)X^2 - (1 - 2u + 3v)XY + 2vY^2 = 0 $$
同理，推导椭圆 $O$ 上对应射线对的联合方程。
设直线 $MN$ 在局部坐标系下的方程为 $UX + VY + 1 = 0$。
利用 $1 = -UX - VY$ 对椭圆 $O$ 进行齐次化，构造出经过原点且经过 $M, N$ 的射线对 $PM$ 与 $PN$ 的联合二次齐次方程：
$$ X^2 + 2Y^2 + (4X + 4Y)(-UX - VY) = 0 $$
展开并合并同类项得：
$$ (1 - 4U)X^2 - 4(U + V)XY + (2 - 4V)Y^2 = 0 $$
接着，建立恒等映射并解算直线 $MN$ 的系数。
由于点 $A, P, M$ 共线且 $B, P, N$ 共线，射线对 $PA, PB$ 与射线对 $PM, PN$ 在几何上重合，其对应的齐次多项式系数必定成比例。
存在非零实数 $\lambda$ 使得如下比例关系成立：
$$ \frac{1 - 4U}{1 - 3u} = \frac{-4(U + V)}{-(1 - 2u + 3v)} = \frac{2 - 4V}{2v} = \lambda $$
由第一式与第三式可得：
$$ 4U = 1 - \lambda(1 - 3u), \quad 4V = 2 - 2\lambda v $$
将上述两式相加并代入第二式的变形 $4(U + V) = \lambda(1 - 2u + 3v)$ 中：
$$ 1 - \lambda(1 - 3u) + 2 - 2\lambda v = \lambda(1 - 2u + 3v) $$
展开并整理：
$$ 3 - \lambda + 3\lambda u - 2\lambda v = \lambda - 2\lambda u + 3\lambda v $$
移项合并得：
$$ \lambda(2 - 5u + 5v) = 3 \implies \lambda = \frac{3}{2 - 5u + 5v} $$
将 $\lambda$ 代回 $U, V$ 的表达式中：
$$ 4U = 1 - \frac{3(1 - 3u)}{2 - 5u + 5v} = \frac{2 - 5u + 5v - 3 + 9u}{2 - 5u + 5v} = \frac{4u + 5v - 1}{2 - 5u + 5v} $$
$$ 4V = 2 - \frac{6v}{2 - 5u + 5v} = \frac{4 - 10u + 10v - 6v}{2 - 5u + 5v} = \frac{4 - 10u + 4v}{2 - 5u + 5v} $$
将解得的参数代入局部方程 $UX + VY + 1 = 0$，再利用平移关系 $X = x - 2, Y = y - 1$ 还原至全局坐标系：
$$ \frac{4u + 5v - 1}{4(2 - 5u + 5v)}(x - 2) + \frac{4 - 10u + 4v}{4(2 - 5u + 5v)}(y - 1) + 1 = 0 $$
等式两端同乘 $4(2 - 5u + 5v)$：
$$ (4u + 5v - 1)(x - 2) + (4 - 10u + 4v)(y - 1) + 4(2 - 5u + 5v) = 0 $$
展开各项并提取 $x, y$ 的系数，整理得直线 $MN$ 的全局方程 $Ax + By + C = 0$：
$$ (4u + 5v - 1)x + (-10u + 4v + 4)y + (-18u + 6v + 6) = 0 $$
另一方面，计算圆 $G$ 与切线 $L$ 相切的代数约束条件。
已知圆 $G$ 的方程为 $(x+2)^2 + \left(y-\frac{9}{4}\right)^2 = \frac{25}{16}$，圆心为 $C_G\left(-2, \frac{9}{4}\right)$，半径 $r = \frac{5}{4}$。
直线 $L$ 在全局坐标系下的方程为 $u(x - 2) + v(y - 1) + 1 = 0$，即 $ux + vy + (1 - 2u - v) = 0$。
根据点到直线的距离公式，圆心到切线的距离等于半径：
$$ \frac{\left| -2u + \frac{9}{4}v + 1 - 2u - v \right|}{\sqrt{u^2 + v^2}} = \frac{5}{4} $$
化简分子并在等式两端同乘 $4\sqrt{u^2 + v^2}$：
$$ |-16u + 5v + 4| = 5\sqrt{u^2 + v^2} $$
两端平方并展开：
$$ 256u^2 + 25v^2 + 16 - 160uv - 128u + 40v = 25u^2 + 25v^2 $$
移项并合并同类项，得到参数 $u, v$ 必须满足的代数约束：
$$ 231u^2 - 160uv - 128u + 40v + 16 = 0 $$
最后，验证直线 $MN$ 与椭圆 $E$ 相切。
目标椭圆 $E: 25x^2 + 8y^2 = 12$ 可化为标准相切判定形式 $\frac{x^2}{12/25} + \frac{y^2}{3/2} = 1$。
直线 $Ax + By + C = 0$ 与该椭圆相切的代数等效条件为 $\frac{12}{25}A^2 + \frac{3}{2}B^2 = C^2$。
将直线 $MN$ 的系数 $A = 4u + 5v - 1$，$B = -10u + 4v + 4$，$C = -18u + 6v + 6$ 代入判定式：
$$ \frac{12}{25}(4u + 5v - 1)^2 + \frac{3}{2}(-10u + 4v + 4)^2 = (-18u + 6v + 6)^2 $$
从各项中提取公因子，$B = 2(-5u + 2v + 2)$，$C = 6(-3u + v + 1)$。
等式两端同乘 $50$ 以消除分母：
$$ 24(4u + 5v - 1)^2 + 75\big[2(-5u + 2v + 2)\big]^2 = 50\big[6(-3u + v + 1)\big]^2 $$
$$ 24(4u + 5v - 1)^2 + 300(-5u + 2v + 2)^2 = 1800(-3u + v + 1)^2 $$
等式两端同除以 $12$：
$$ 2(4u + 5v - 1)^2 + 25(-5u + 2v + 2)^2 = 150(-3u + v + 1)^2 $$
完全展开等式左端：
$$ 2(16u^2 + 25v^2 + 1 + 40uv - 8u - 10v) + 25(25u^2 + 4v^2 + 4 - 20uv - 20u + 8v) $$
合并后为 $657u^2 + 150v^2 - 420uv - 516u + 180v + 102$。
完全展开等式右端：
$$ 150(9u^2 + v^2 + 1 - 6uv - 6u + 2v) = 1350u^2 + 150v^2 - 900uv - 900u + 300v + 150 $$
令右端减去左端等于零，合并同类项：
$$ (1350 - 657)u^2 + (150 - 150)v^2 + (-900 + 420)uv + (-900 + 516)u + (300 - 180)v + (150 - 102) = 0 $$
$$ 693u^2 - 480uv - 384u + 120v + 48 = 0 $$
等式两端同除以 $3$，得到最终等价判定多项式：
$$ 231u^2 - 160uv - 128u + 40v + 16 = 0 $$
该等价判定多项式与前述推导的切线 $L$ 与圆 $G$ 的相切约束多项式完全一致。
由于参数约束客观存在，该判定恒等式必然成立。
因此，直线 $MN$ 恒与椭圆 $E: 25x^2 + 8y^2 = 12$ 相切。结论成立。
\end{solution}

$$ k_1(x - 2) + 1 = x - \frac{2}{x}\Longleftrightarrow k_1(x - 2) = x - 1 - \frac{2}{x} = \frac{x^2 - x - 2}{x} = \frac{(x - 2)(x + 1)}{x}  $$
因为点 $M$ 异于点 $P$，所以其横坐标 $x_1 \neq 2$。两边约去 $(x - 2)$得 $k_1 = \frac{x_1 + 1}{x_1} = 1 + \frac{1}{x_1}$。

设经过定点 $Q(-2, -2)$ 的直线 $L$ 的方程，当其斜率不存在时，代入椭圆方程 $x^2 + 2y^2 = 6$ 得 $x^2 + 8 = 6$，无实数解。因此可以设直线 $L$ 的方程为 $x = m(y + 2) - 2$。联立：
$$ (my + 2m - 2)^2 + 2y^2 = 6 \Longleftrightarrow (m^2 + 2)y^2 + 4m(m - 1)y + 4m^2 - 8m - 2 = 0$$
展开并按 $y$ 的降幂排列，得一元二次方程：
$$ (m^2 + 2)y^2 + 4m(m - 1)y + 4m^2 - 8m - 2 = 0 $$
设直线 $L$ 与椭圆的两个交点 $A, B$ 的坐标分别为 $(x_A, y_A), (x_B, y_B)$。
由韦达定理可得：
$$ y_A + y_B = \frac{-4m(m - 1)}{m^2 + 2}, \quad y_A y_B = \frac{4m^2 - 8m - 2}{m^2 + 2} $$
对于直线 $AP$，由斜率公式有 $k_1 = \frac{y_A - 1}{x_A - 2}$。将 $x_A = m y_A + 2m - 2$ 代入，得：
$$ k_1 - 1 = \frac{y_A - 1 - (x_A - 2)}{x_A - 2} = \frac{y_A - (my_A + 2m - 2) + 1}{my_A + 2m - 4} = \frac{(1 - m)y_A + 3 - 2m}{my_A + 2m - 4} $$
同理，对于直线 $BP$ 有 $k_2 - 1 = \frac{(1 - m)y_B + 3 - 2m}{my_B + 2m - 4}$。
将两式相乘，可得 $(k_1 - 1)(k_2 - 1) = \frac{N}{D}$，其中分子 $N$ 与分母 $D$ 展开如下：
分子 $N = (1 - m)^2 y_A y_B + (1 - m)(3 - 2m)(y_A + y_B) + (3 - 2m)^2$。
分母 $D = m^2 y_A y_B + m(2m - 4)(y_A + y_B) + (2m - 4)^2$。
将韦达定理的结果代入并同乘公分母 $m^2 + 2$，对分子进行计算：
第一项为 $(1 - m)^2 (4m^2 - 8m - 2) = (m^2 - 2m + 1)(4m^2 - 8m - 2) = 4m^4 - 16m^3 + 18m^2 - 4m - 2$；
第二项为 $(1 - m)(3 - 2m)[-4m(m - 1)] = 4m(m - 1)^2(3 - 2m) = 4m(m^2 - 2m + 1)(3 - 2m) = -8m^4 + 28m^3 - 32m^2 + 12m$；
第三项为 $(3 - 2m)^2 (m^2 + 2) = (4m^2 - 12m + 9)(m^2 + 2) = 4m^4 - 12m^3 + 17m^2 - 24m + 18$。
三项合并同类项，得 $N(m^2 + 2) = 3m^2 - 16m + 16$。
同理，对分母进行计算：
第一项为 $m^2 (4m^2 - 8m - 2) = 4m^4 - 8m^3 - 2m^2$；
第二项为 $m(2m - 4)[-4m(m - 1)] = -4m^2(2m^2 - 6m + 4) = -8m^4 + 24m^3 - 16m^2$；
第三项为 $(2m - 4)^2 (m^2 + 2) = (4m^2 - 16m + 16)(m^2 + 2) = 4m^4 - 16m^3 + 24m^2 - 32m + 32$。
三项合并同类项，得 $D(m^2 + 2) = 6m^2 - 32m + 32 = 2(3m^2 - 16m + 16)$。
若 $3m^2 - 16m + 16 = 0$，则 $m = 4$ 或 $m = \frac{4}{3}$。
当 $m = \frac{4}{3}$ 时，直线 $L$ 的方程为 $3x - 4y - 2 = 0$。经验证点 $P(2, 1)$ 在该直线上，这会导致交点 $A$ 或 $B$ 中必有一点与 $P$ 重合，从而无法作割线并向外延长，与题意中“延长 $AP$”矛盾；
当 $m = 4$ 时，直线 $L$ 的方程为 $x - 4y - 6 = 0$，代入椭圆方程解得交点之一为 $(2, -1)$。此时过该点与 $P(2, 1)$ 的直线垂直于 $x$ 轴，与曲线 $C_1$ 仅有唯一交点 $P$，不存在异于 $P$ 的交点 $M$，同样与题设矛盾。
因此 $3m^2 - 16m + 16 \neq 0$。作商可得：
$$ (k_1 - 1)(k_2 - 1) = \frac{3m^2 - 16m + 16}{2(3m^2 - 16m + 16)} = \frac{1}{2} $$
最后，利用上述定值证明目标不等式。
将 $x_1 = \frac{1}{k_1 - 1}$ 与 $x_2 = \frac{1}{k_2 - 1}$ 代入，得到：
$$ x_1 x_2 = \frac{1}{(k_1 - 1)(k_2 - 1)} = 2 $$
根据基本不等式，有 $x_1^2 + x_2^2 \ge 2|x_1 x_2| = 4$。
探讨等号成立的条件：若 $x_1^2 + x_2^2 = 4$，结合 $x_1 x_2 = 2 > 0$ 可知必有 $x_1 = x_2$。
若 $x_1 = x_2$，则推出 $k_1 = k_2$。这意味着直线 $AP$ 与直线 $BP$ 的斜率相等。由于这两条直线均经过点 $P$，故直线 $AP$ 与 $BP$ 重合，即点 $A, B, P$ 三点共线。
因为 $A, B$ 是直线 $L$ 与椭圆的两个交点，若 $A, B, P$ 共线，则直线 $L$ 必定经过点 $P$。
由前述分析已知，若直线 $L$ 过点 $P$，则其与椭圆的两个交点中必有一个为 $P$。不妨设 $A = P$，由于要求作割线延长线，此时 $AP$ 只能视为切线（此时切线斜率由导数可得 $k_1 = -1$），而 $BP$ 为直线 $L$（斜率 $k_2 = \frac{3}{4}$）。显然 $k_1 \neq k_2$，导致矛盾。
因此，$k_1 = k_2$ 不可能成立，从而 $x_1 \neq x_2$。
基本不等式中的等号无法取得，严格不等式成立：
$$ x_1^2 + x_2^2 > 4 $$
综上所述，$x_1^2 + x_2^2 > 4$ 证明完毕。
\end{solution}
