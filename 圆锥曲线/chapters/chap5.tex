\begin{example}{}{}
设$F$ 为椭圆$C_1:\frac{x^2}4+\frac{y^2}3=1$ 的右焦点，$l$为$F$ 关于$C_{_1}$的准线. 设$B(1,\frac32),A$为$l$上一动点。设直线 $AB$ 与$C_{_1}$ 再次交于点$C$ ,$\triangle BCF$ 的 外接 圆 与 直 线 $AF$ 再次交于点$D$，设$D$ 的轨迹为$C_2.\\
(1)证明：C_2$是与$y$轴相切的椭圆\\
(2)求$C_2$的离心率
\end{example}
\begin{solution}
椭圆内积算子定义为 $E(M, N) = \frac{x_M x_N}{4} + \frac{y_M y_N}{3}$。动点 $A$ 在准线 $l: x = \frac{a^2}{c} = 4$ 上，设 $A(4, t)$。已知 $B(1, \frac{3}{2})$，极易验证 $E(B,B) = \frac{1}{4} + \frac{9/4}{3} = 1$，即 $B$ 在椭圆上。我们引入向量 $\vec{v} = \overrightarrow{BA} = (3, t - \frac{3}{2})$。直线 $AB$ 上的动点可以参数化为 $X = B + \lambda \vec{v}$。将它代入椭圆极化算子 $E(X,X) - 1 = 0$ 中，得到讲义里的直线参数多项式：
$$\lambda^2 E(\vec{v}, \vec{v}) + 2\lambda E(B, \vec{v}) + E(B,B) - 1 = 0$$
因为 $E(B,B) = 1$，常数项归零！这意味方程必有一根 $\lambda = 0$（对应点 $B$）。另一个非零根（对应点 $C$）直接由韦达定理爆出：$\lambda_C = \frac{-2E(B, \vec{v})}{E(\vec{v}, \vec{v})}$。我们计算这两个算子零件（不跳步）：
\begin{itemize}
\item $E(B, \vec{v}) = \frac{1 \cdot 3}{4} + \frac{\frac{3}{2}(t - \frac{3}{2})}{3} = \frac{3}{4} + \frac{1}{2}(t - \frac{3}{2}) = \frac{3}{4} + \frac{1}{2}t - \frac{3}{4} = {\frac{1}{2}t}$
\item $E(\vec{v}, \vec{v}) = \frac{3^2}{4} + \frac{(t - \frac{3}{2})^2}{3} = \frac{9}{4} + \frac{t^2 - 3t + \frac{9}{4}}{3} = \frac{9}{4} + \frac{t^2}{3} - t + \frac{3}{4} = {\frac{t^2}{3} - t + 3}$
\item 代入得到 $\lambda_C = \frac{-t}{\frac{t^2}{3} - t + 3} = \frac{-3t}{t^2 - 3t + 9}$。
\end{itemize}
点 $C$ 的坐标瞬间解出
\begin{itemize}
\item $x_C = 1 + 3\lambda_C = 1 - \frac{9t}{t^2 - 3t + 9} ={\frac{t^2 - 12t + 9}{t^2 - 3t + 9}}$
\item $y_C = \frac{3}{2} + (t - \frac{3}{2})\lambda_C = \frac{3}{2} - \frac{3t(t - \frac{3}{2})}{t^2 - 3t + 9} = \frac{3(t^2 - 3t + 9) - 3t^2 + \frac{9}{2}t}{2(t^2 - 3t + 9)} ={\frac{3(9 - t^2)}{2(t^2 - 3t + 9)}}$
\end{itemize}
焦点 $F(1,0)$，点 $B(1, \frac{3}{2})$。注意到它们的横坐标都是 1。设 $\triangle BCF$ 外接圆方程为 $x^2 + y^2 + K_x x + K_y y + G = 0$。
\begin{itemize}
    \item 代入 $F(1,0) \implies 1 + K_x + G = 0 \implies G = -K_x - 1$
    \item 代入 $B(1, \frac{3}{2}) \implies 1 + \frac{9}{4} + K_x + \frac{3}{2}K_y + G = 0$
    \item 把 $G$ 塞进去，$K_x$ 直接抵消：$\frac{9}{4} + \frac{3}{2}K_y = 0 \implies \mathbf{K_y = -\frac{3}{2}}$
\end{itemize}
现在，圆方程变成了 $x^2 + y^2 + K_x x - \frac{3}{2}y - K_x - 1 = 0$。
将点 $C(x_C, y_C)$ 代入，合并 $K_x$：
\begin{align*}
K_x(x_C - 1) =& 1 + \frac{3}{2}y_C - (x_C^2 + y_C^2)\\
=&1 + \frac{3}{2}y_C - (4 - \frac{4}{3}y_C^2 + y_C^2) = \frac{1}{3}y_C^2 + \frac{3}{2}y_C - 3\\
=& \frac{1}{3} \cdot \frac{9(9-t^2)^2}{4(t^2-3t+9)^2} + \frac{3}{2} \cdot \frac{3(9-t^2)}{2(t^2-3t+9)} - 3\\
=&\frac{3(81 - 18t^2 + t^4) + 9(9-t^2)(t^2-3t+9) - 12(t^2-3t+9)^2}{4(t^2-3t+9)^2}
\end{align*}
分子逐项展开合并：
\begin{itemize}
\item $t^4$ 系数：$3 - 9 - 12 = -18$
\item $t^3$ 系数：$0 + 27 + 72 = 99$
\item $t^2$ 系数：$-54 - 0 - 324 = -378$
\item $t$ 系数：$0 - 243 + 648 = 405$
\item 常数项：$243 + 729 - 972 = \mathbf{0}$ （常数项奇迹般地消失了！）
\end{itemize}
所以分子式是$-18t^4 + 99t^3 - 378t^2 + 405t = -9t(2t^3 - 11t^2 + 42t - 45)$，等式左边 $x_C - 1 = \frac{t^2 - 12t + 9}{t^2 - 3t + 9} - 1 = \frac{-9t}{t^2 - 3t + 9}$。相除解出 $K_x$，令人极度舒适的对消发生了（$-9t$ 完美约去）：$$\mathbf{K_x = \frac{2t^3 - 11t^2 + 42t - 45}{4(t^2 - 3t + 9)}}$$
直线 $AF$ 过 $F(1,0)$ 和 $A(4,t)$，方向向量为 $(3, t)$。设直线 $AF$ 上动点 $X = F + u(3, t) \implies x = 3u+1, y = tu$。将参数式代入刚求出的圆方程：
$$(3u+1)^2 + (tu)^2 + K_x(3u+1) - \frac{3}{2}(tu) - K_x - 1 = 0$$
展开并提取公因式 $u$（因为 $F$ 对应 $u=0$，必然有一个根为 0）：
\begin{align*}
&9u^2 + 6u + 1 + t^2 u^2 + 3K_x u + K_x - \frac{3}{2}tu - K_x - 1 = 0\\
\Leftrightarrow&(t^2 + 9)u^2 + \left( 6 + 3K_x - \frac{3}{2}t \right)u = 0
\end{align*}
非零根 $u$ （对应点 $D$）直接算出：$$u = \frac{\frac{3}{2}t - 6 - 3K_x}{t^2 + 9}$$把 $K_x$ 塞进分子，通分（公分母为 $4(t^2 - 3t + 9)$）：
\begin{align*}
 \text{分子}&= \frac{6t(t^2 - 3t + 9) - 24(t^2 - 3t + 9) - 3(2t^3 - 11t^2 + 42t - 45)}{4(t^2 - 3t + 9)}\\
 &= \frac{6t^3 - 18t^2 + 54t - 24t^2 + 72t - 216 - 6t^3 + 33t^2 - 126t + 135}{4(t^2 - 3t + 9)}
\end{align*}
\begin{itemize}
\item $t^3$: $6 - 6 = 0$
\item $t^2$: $-18 - 24 + 33 = -9$
\item $t$: $54 + 72 - 126 = 0$
\item 常数：$-216 + 135 = -81$
\end{itemize}
分子竟然变成了 $-9t^2 - 81 = -9(t^2 + 9)$ ，与外面的分母 $(t^2 + 9)$ 完美约分
$$u = \frac{-9(t^2 + 9)}{4(t^2 - 3t + 9)(t^2 + 9)} = \mathbf{\frac{-9}{4(t^2 - 3t + 9)}}$$
点 $D$ 的参数坐标彻底净化：$$x_D = 1 + 3u = \mathbf{1 - \frac{27}{4(t^2 - 3t + 9)}},~~y_D = tu = \mathbf{\frac{-9t}{4(t^2 - 3t + 9)}}$$
观察结构，我们发现：$$\frac{y_D}{1 - x_D} = \frac{\frac{-9t}{4(\dots)}}{\frac{27}{4(\dots)}} = -\frac{t}{3} \implies \mathbf{t = \frac{-3y_D}{1 - x_D}}$$把 $t$ 代回 $1 - x_D = \frac{27}{4(t^2 - 3t + 9)}$ 中：$$1 - x_D = \frac{27}{4 \left( \frac{9y_D^2}{(1-x_D)^2} + \frac{9y_D}{1-x_D} + 9 \right)}$$分母提出 $9$ 并与分子约分：$$1 - x_D = \frac{3}{4 \left( \frac{y_D^2}{(1-x_D)^2} + \frac{y_D}{1-x_D} + 1 \right)}$$将右侧的分母乘过去，并同时乘以 $(1-x_D)$ 以清理内层分母：$$4 \left[ y_D^2 + y_D(1-x_D) + (1-x_D)^2 \right] = 3(1-x_D)$$展开一切：$$4y_D^2 + 4y_D - 4x_D y_D + 4 - 8x_D + 4x_D^2 = 3 - 3x_D$$移项整理，得到动点 $D$ 的终极轨迹 $C_2$：$$\mathbf{4x^2 - 4xy + 4y^2 - 5x + 4y + 1 = 0}$$
(1) 证明与 $y$ 轴相切：令 $x = 0$，方程化为 $4y^2 + 4y + 1 = 0 \implies (2y + 1)^2 = 0$。方程有且仅有一个实数重根 $y = -\frac{1}{2}$。判别式 $\Delta = 0$，绝对相切！切点为 $(0, -\frac{1}{2})$。证毕。\\
(2) 求离心率：方程二次项为 $4x^2 - 4xy + 4y^2$。要找椭圆的离心率，我们只需消除交叉项。使用高中最基础的换元法（引入中间变量，相当于坐标轴旋转 $45^\circ$）：令 $x = \frac{\sqrt{2}}{2}(u - v), y = \frac{\sqrt{2}}{2}(u + v)$。$x^2 + y^2 = u^2 + v^2,~xy = \frac{1}{2}(u^2 - v^2)$，代入二次项结构中：$$4(x^2 + y^2) - 4xy = 4(u^2 + v^2) - 2(u^2 - v^2) = \mathbf{2u^2 + 6v^2}$$轨迹方程化为了 $\frac{(u-u_0)^2}{\frac{1}{2}} + \frac{(v-v_0)^2}{\frac{1}{6}} = R^2$ 的标准形状！离心率 $e$ 为：$$e = \sqrt{\frac{c'^2}{a'^2}} = \sqrt{\frac{1/3}{1/2}} = \sqrt{\frac{2}{3}} = \mathbf{\frac{\sqrt{6}}{3}}$$
\end{solution}




本章将挑选三道典型的高考压轴题，严格使用“四段式”（题干翻译$\Rightarrow$算子作用$\Rightarrow$代数计算$\Rightarrow$结论输出）展现算子的统治力。

\subsection{实战一：定点弦斜率调整问题（昊天原创题）}
\textbf{题目}：过点 $M(2,1)$ 的直线交椭圆 $\mathcal{C}: \frac{x^2}{2} + y^2 = 1$ 于 $A, B$ 两点。点 $P(0,3)$，直线 $PA$ 交直线 $l: x+y=1$ 于点 $Q$。记直线 $QA$ 和 $QB$ 的斜率分别为 $k_1, k_2$。求证：$\frac{1}{k_1+1} - \frac{1}{k_2+1}$ 为定值。

\textbf{步骤1：题干翻译} \\
将椭圆方程转化为极化算子：$T(U,V) = \frac{x_U x_V}{2} + y_U y_V - 1$。
代入定点 $M(2,1)$ 求极化函数：$T(M, X) = \frac{2 x_X}{2} + 1 \cdot y_X - 1 = x_X + y_X - 1$。
\textit{题干几何条件神级翻译}：由于直线 $l$ 的方程为 $x+y-1=0$，这恰好等价于方程 $T(M, X) = 0$。因为动点 $Q$ 在直线 $l$ 上，故条件被翻译为绝对代数约束：$T(M,Q) = 0$。

\textbf{步骤2：算子作用} \\
目标是转化非对称的斜率表达式 $k_1+1 = k_{QA}+1$。
根据斜率定义 $k_{QA} = \frac{y_A - y_Q}{x_A - x_Q}$。代入表达式中并通分：
$$ k_{QA} + 1 = \frac{y_A - y_Q + x_A - x_Q}{x_A - x_Q} = \frac{(x_A + y_A - 1) - (x_Q + y_Q - 1)}{x_A - x_Q} $$
利用步骤1的翻译结论：
$$ k_{QA} + 1 = \frac{T(M,A) - T(M,Q)}{x_A - x_Q} $$
由于 $T(M,Q) = 0$，上式坍缩为 $\frac{T(M,A)}{x_A - x_Q}$。
同理，$k_{QB} + 1 = \frac{T(M,B)}{x_B - x_Q}$。
所求目标变为算子形式：
$$ \text{目标} = \frac{x_A - x_Q}{T(M,A)} - \frac{x_B - x_Q}{T(M,B)} $$

\textbf{步骤3：代数计算} \\
因为 $P, Q, A$ 三点共线，设 $Q = \alpha P + \beta A$（$\alpha+\beta=1$）。
将 $Q$ 代入 $T(M,Q) = 0$ 中，应用分配律（第一部分性质2）：
$$ T(M, \alpha P + \beta A) = \alpha T(M,P) + \beta T(M,A) = 0 $$
代入 $P(0,3)$ 到算子得 $T(M,P) = 0 + 3 - 1 = 2$。
得到方程：$2\alpha + \beta T(M,A) = 0$。
解得 $\alpha = \frac{-T(M,A)}{2 - T(M,A)}$，$\beta = \frac{2}{2 - T(M,A)}$。
计算横坐标分子 $x_A - x_Q = x_A - (\alpha x_P + \beta x_A)$。因为 $x_P = 0$，所以 $x_A - x_Q = x_A(1-\beta) = \alpha x_A$。
代入目标式第一项：
$$ \frac{\alpha x_A}{T(M,A)} = \frac{ \frac{-T(M,A)}{2 - T(M,A)} x_A }{T(M,A)} = \frac{-x_A}{2 - T(M,A)} $$
代入第二项所需 $x_Q = \beta x_A = \frac{2x_A}{2-T(M,A)}$。目标式通分：
$$ \text{目标} = \frac{-x_A}{2 - T(M,A)} - \frac{x_B - \frac{2x_A}{2 - T(M,A)}}{T(M,B)} = \frac{-x_A T(M,B) - 2x_B + x_B T(M,A) + 2x_A}{T(M,B)(2 - T(M,A))} $$
分析分子核心项：$x_B T(M,A) - x_A T(M,B) = x_B(x_A+y_A-1) - x_A(x_B+y_B-1) = x_B y_A - x_A y_B + x_A - x_B$。
由于 $A, M, B$ 共线，应用外积算子 \textbf{【性质1绝对清零】}：
$W(\overrightarrow{MA}, \overrightarrow{MB}) = 0 \Rightarrow (x_A-2)(y_B-1) - (y_A-1)(x_B-2) = 0$
展开移项得到：$x_B y_A - x_A y_B = x_B - x_A + 2y_A - 2y_B$。
将此外积结果代入分子，分子变为：
$$ (x_B - x_A + 2y_A - 2y_B) + x_A - x_B - 2x_B + 2x_A = 2x_A + 2y_A - 2x_B - 2y_B = 2(T(M,A) - T(M,B)) $$
利用 \textbf{【定理1的第3点：调和极化律】} $T(M,M)T(A,B) = 2T(M,A)T(M,B)$。由于 $T(M,M) = \frac{4}{2}+1-1 = 2$，代入变形：
$\frac{1}{T(M,A)} + \frac{1}{T(M,B)} = \frac{2}{T(M,M)} = 1 \Rightarrow T(M,A) + T(M,B) = T(M,A)T(M,B)$。
代入分母 $T(M,B)(2 - T(M,A)) = 2T(M,B) - T(M,A)T(M,B) = 2T(M,B) - (T(M,A) + T(M,B)) = -(T(M,A) - T(M,B))$。

\textbf{步骤4：结论输出} \\
目标式 $= \frac{2(T(M,A) - T(M,B))}{-(T(M,A) - T(M,B))} = -2$。原命题得证。

\subsection{实战二：极化参数解决阿波罗尼斯圆问题}
\textbf{题目}：已知抛物线 $\mathcal{C}: y^2 = 2x$，点 $P(1,2)$。若过点 $P$ 的动弦 $l$ 交抛物线于 $A, B$。问是否存在异于 $P$ 的点 $Q$ 使得 $\frac{PA}{PB} = \frac{QA}{QB}$ 恒成立？

\textbf{步骤1：题干翻译} \\
抛物线的极化算子定义为 $T(U,V) = y_U y_V - (x_U + x_V)$。自身极化 $T(P,P) = 2^2 - 2(1) = 2$。
题干要求：$\frac{PA^2}{PB^2} = \frac{QA^2}{QB^2}$ 恒成立。设未知向量 $\vec{u} = Q - P$，需通过算子找到 $\vec{u}$ 的值。

\textbf{步骤2：算子作用} \\
设直线 $l$ 方向向量为 $\vec{v} = (v_x, v_y)$，参数方程 $X = P + r\vec{v}$。
应用 \textbf{【定理2：坐标平移下的算子行为】}，将 $X$ 代入 $T(X,X) = 0$：
$$ T(\vec{v},\vec{v}) r^2 + 2T(P, \vec{v}) r + T(P,P) = 0 $$
由一元二次方程根与系数关系得到参数倒数和：
$$ \frac{1}{r_A} + \frac{1}{r_B} = -\frac{2T(P, \vec{v})}{T(P,P)} $$

\textbf{步骤3：代数计算} \\
利用欧氏内积展开距离平方：$QA^2 = |Q-A|^2 = |\vec{u} - r_A\vec{v}|^2 = |\vec{u}|^2 - 2r_A I(\vec{u}, \vec{v}) + r_A^2 |\vec{v}|^2$。
代入题干条件并除以各自的分母 $r_A^2$ 与 $r_B^2$：
$$ \frac{|\vec{u}|^2}{r_A^2} - \frac{2I(\vec{u}, \vec{v})}{r_A} + |\vec{v}|^2 = \frac{|\vec{u}|^2}{r_B^2} - \frac{2I(\vec{u}, \vec{v})}{r_B} + |\vec{v}|^2 $$
两边同时减去 $|\vec{v}|^2$ 并移项：
$$ |\vec{u}|^2 \left(\frac{1}{r_A^2} - \frac{1}{r_B^2}\right) - 2I(\vec{u}, \vec{v}) \left(\frac{1}{r_A} - \frac{1}{r_B}\right) = 0 $$
由于 $A, B$ 是相异两点，$r_A \neq r_B$，两边同时除以 $(\frac{1}{r_A} - \frac{1}{r_B})$：
$$ |\vec{u}|^2 \left(\frac{1}{r_A} + \frac{1}{r_B}\right) - 2I(\vec{u}, \vec{v}) = 0 $$
代入步骤2得到的倒数和：
$$ |\vec{u}|^2 \left(-\frac{2T(P, \vec{v})}{T(P,P)}\right) - 2I(\vec{u}, \vec{v}) = 0 $$
两边同除以 $-2$。根据算子定义 $T(P, \vec{v}) = 2v_y - v_x = I( (-1, 2), \vec{v} )$。令特征向量 $\nabla P = (-1, 2)$。
方程变为纯内积的恒等式：
$$ I \left( \frac{|\vec{u}|^2}{T(P,P)} \nabla P + \vec{u}, \vec{v} \right) = 0 $$
由于对任意弦方向 $\vec{v}$ 恒成立，括号内向量必为零向量：
$$ \vec{u} = -\frac{|\vec{u}|^2}{2} (-1, 2) $$

\textbf{步骤4：结论输出} \\
设 $\vec{u} = m(-1, 2)$，则 $|\vec{u}|^2 = 5m^2$。
代入得 $m(-1,2) = -\frac{5m^2}{2}(-1,2)$。解得 $m = -\frac{2}{5}$（舍去 $m=0$ 因为 $Q \neq P$）。
$\vec{u} = (\frac{2}{5}, -\frac{4}{5})$。
$Q = P + \vec{u} = (1,2) + (\frac{2}{5}, -\frac{4}{5}) = (\frac{7}{5}, \frac{6}{5})$。存在。

\subsection{实战三：双切线交点轨迹问题}
\textbf{题目}：已知圆 $C_1: x^2+y^2-3x+1=0$。点 $A(2,-1)$ 在椭圆 $C_2: x^2+4y^2-8=0$ 上且在 $C_1$ 上。过 $A$ 的直线交两曲线于 $B,C$。设 $C_1$ 在 $B$ 的切线与 $C_2$ 在 $C$ 的切线交于 $D$。求 $D$ 的轨迹方程。

\textbf{步骤1：题干翻译} \\
建立双算子系统：$T_1(U,V) = x_U x_V + y_U y_V - \frac{3}{2}(x_U+x_V) + 1$；$T_2(U,V) = x_U x_V + 4y_U y_V - 8$。
几何条件“$D$ 是 $B$ 点处关于 $C_1$ 的切线上的点”翻译为：$T_1(D, B) = 0$。
同理，条件翻译为：$T_2(D, C) = 0$。

\textbf{步骤2：算子作用} \\
应用 \textbf{【定理2：坐标平移下的算子行为】}。设直线参数方程 $X = A + t\vec{v}$，$\vec{v}=(m,1)$。
代入圆 $T_1(A+t\vec{v}, A+t\vec{v}) = 0$，因 $A$ 在圆上 $T_1(A,A)=0$，解得 $B$ 的参数 $t_B = -\frac{2T_1(A,\vec{v})}{T_1(\vec{v},\vec{v})}$。
代入切线方程 $T_1(D, A + t_B\vec{v}) = 0$，利用第一部分的【分配律】：
$$ T_1(D, A) + t_B T_1(D, \vec{v}) = 0 \Rightarrow T_1(D, A) \cdot T_1(\vec{v}, \vec{v}) - 2T_1(A, \vec{v}) \cdot T_1(D, \vec{v}) = 0 $$
同理作用于 $C_2$ 得到 $C$ 的参数方程：
$$ T_2(D, A) \cdot T_2(\vec{v}, \vec{v}) - 2T_2(A, \vec{v}) \cdot T_2(D, \vec{v}) = 0 $$

\textbf{步骤3：代数计算} \\
设 $D(x,y)$。计算各项算子值：
$T_1(\vec{v},\vec{v}) = m^2+1$；$T_1(A,\vec{v}) = 2m-1-\frac{3}{2}(m) = \frac{1}{2}m-1$。
$T_1(D,A) = 2x-y-\frac{3}{2}(x+2)+1 = \frac{1}{2}x-y-2$；$T_1(D,\vec{v}) = mx+y-\frac{3}{2}m$。
代入化简得圆切线展开：$m^2(x+2y+1) - 2m(2x-y-3) - (x+2y-4) = 0$。
同理，椭圆切线展开：$m^2(x+2y+4) - 4m(x-2y) - 4(x+2y-4) = 0$。
令 $S = x+2y$，利用加减消元法，将上式乘 $4$ 减下式，得 $3S m^2 - 12(x-2)m = 0 \Rightarrow m = \frac{4(x-2)}{S}$。
代回原式：$(\frac{4x-8}{S})^2(S+1) - 2\frac{4x-8}{S}(2x-y-3) - (S-4) = 0$。

\textbf{步骤4：结论输出} \\
两边同乘 $S^2$ 展开，化简后得到唯一有效解 $S = 4$。
所以点 $D$ 的轨迹方程为 $x + 2y - 4 = 0$。

$k_{AB} = 1/2$。椭圆共轭直径斜率积 $k_{AB} \cdot k_{MN} = -b^2/a^2 = -3/6 = -1/2$。即 $\frac{1}{2} k_{MN} = -1/2 \implies k_{MN} = -1$。

\textbf{10. 资料二 题10：} 椭圆 $3x^2+4y^2=12, P(-2,1), A(-2,0)$。割线 $BC$ 过 $P$。$E = AD \cap \{x=x_B\}, BE=EF$。证 $CF$ 过定点。\\
\textbf{【算子秒杀】} 点 $F$ 由等距条件 $BE=EF$ 给出，即 $F = 2E-B$。这只是 $B$ 点关于给定直线的线性仿射组合！既然 $B \to C$ 是由极化对合 $\mathcal{I}_P$ 生成的射影变换，那么 $C \to F$ 也是保射影比的 $1D$ 同态。射影变换生成的连线 $CF$ 构成射影线束，根据交比守恒，必过定点。

\subsection{第四战区：彭赛列大闭合与退化曲线系}

\textbf{11. 资料二 例题8：} 椭圆 $x^2/9+y^2/3=1$，$PT, QT$ 切内部圆 $M:(x-1)^2+y^2=1$。证 $PQ$ 切圆 $M$。\\
\textbf{12. 资料二 习题9：} 双曲线 $x^2-y^2=18, AB$ 过 $F(6,0), AC$ 过 $Q(4,0)$，证 $BC$ 切定圆。\\
\textbf{13. 资料三 例14：} 六点连线交替切于内部椭圆，证此六点共外接椭圆。\\
\textbf{【算子群体秒杀】} 以上三题均为高等几何“彭赛列大闭合定理（Poncelet's Porism）”的推论。
在我们的算子理论中，切线条件直接等价于“二次曲线系的极化判别式归零”。
由《外切线对算子方程》：从点 $P$ 引切线，退化方程为 $[T_{内}(P,X)]^2 = T_{内}(P,P)T_{内}(X,X)$。
连续的极化相切操作意味着双二次型矩阵迹满足 Cayley 算子不变量守恒（$\Theta^2 = 4 \Delta \Theta'$）。算子群的封闭性直接保证了闭合多边形最后一条边必然回归满足 $\Delta = 0$ 的相切恒等式，完全免去了长达数页的参数联立死算。

\subsection{第五战区：射影矩阵抽象证明题（12题横扫）}

资料三中剩余的 12 道极度抽象的纯字母射影题，在算子法面前如同儿戏。我们只需建立词典：\textbf{矩阵连乘 $\iff \mathcal{I}_A(B)$，迹归零 $\iff T(A,B)=0$ 或 $W=0$}。

\begin{itemize}
    \item \textbf{【矩阵例 1, 2, 8】（奇数次复合对合定点）：} 原文利用 Pauli 矩阵乘法展开。在算子法中，只需连用对合引理 $\mathcal{I}_A \circ \mathcal{I}_B \circ \mathcal{I}_A = \mathcal{I}_K$，定点 $K = 2T(A,B)A - T(A,A)B$ 一步写出。
    \item \textbf{【矩阵例 3, 4, 15】（共轭点与极线共点）：} 原文推导矩阵交换律。算子法中：复合映射退化 $\iff T(A,B) = 0$。这本身就是我们在前一章证明的《完全四边形极线定理》。点 $A,B,C$ 皆与 $D$ 极化对偶 $\implies$ 均满足动点泛函 $T(X,D)=0$，该泛函为直线极线，必共线。
    \item \textbf{【矩阵例 6, 7, 12, 13】（梅涅劳斯 / 塞瓦 / 卡诺定理多边形乘积）：} 原文：$\text{tr}([A][B][C])=0$ 且展开庞杂迹项。算子法：直接应用第二章推导的\textbf{算子相交弦定理}：$\frac{KA \cdot KB}{MA \cdot MB} = \frac{T(K,K)}{T(M,M)}$。环状相乘时，所有的 $T(V_i, V_i)$ 就像首尾相接的多米诺骨牌，瞬间连环对消为 1。
    \item \textbf{【矩阵例 10】（交比乘积 $XP \cdot RQ = XQ \cdot PR$）：} 原文：利用四点不变量。算子法：$R$ 为极线与弦的交点。由\textbf{《调和极化律》} $T(X,X)T(P,Q) = 2T(X,P)T(X,Q)$，自然保证了交点与极点构成调和点列 $(P,Q; X,R) = -1$。展开交比定义，立刻得到有向线段比值等式。
\end{itemize}

\begin{example}{费尔巴哈，反演}{}
$\Omega:\frac{x^{2}}{a^{2}}+\frac{y^{2}}{b^{2}}=1$，$F(c,0)$，$\triangle ABC$ 内接于 $\Omega$ 且垂心为 $F$．\\
$(i)$ 证明：$\triangle ABC$ 与定圆相切．\\
$(ii)$ 证明：$\triangle ABC$ 外接圆与定圆相切．\\
$(iii)$ $A,B,C$ 到三边的投影 $I,J,K$，证明：$AF\cdot FI=BF\cdot FJ=CF\cdot FK $为定值．\\
$(iv)$ $A,B,C$ 三处切线围成 $\triangle A'B'C'$，求证：$\triangle A'B'C'$ 垂心 $H$ 为定点．
\end{example}
\begin{solution}
为使计算方便，定义$I(U,V) = x_U x_V + y_U y_V$。$T(U,V) = \frac{x_U x_V}{a^2} + \frac{y_U y_V}{b^2} - 1$。 所以 $T(F,X) = \frac{c x_X}{a^2} - 1$。既然 $F$ 是关键，先把含有焦点 $F$ 的普通点乘化成含有$T,I$的形式：
\[
\begin{cases}I(U-F, V-F) = (x_U - c)(x_V - c) + y_U y_V = x_U x_V - c(x_U + x_V) + c^2 + y_U y_V \\y_U y_V = b^2 T(U,V) + b^2 - \frac{b^2}{a^2}x_U x_V
\end{cases}
\]
解得：
\begin{align*}
I(U-F, V-F) =& x_U x_V - c(x_U + x_V)+c^2 +b^2 T(U,V) + b^2 - \frac{b^2}{a^2} x_U x_V \\
=&\frac{c^2}{a^2} x_U x_V - c(x_U + x_V) + a^2 + b^2 T(U,V)\\
=&a^2 \left( \frac{c x_U}{a^2} - 1 \right) \left( \frac{c x_V}{a^2} - 1 \right)+ b^2 T(U,V)
\end{align*}
得到核心转换公式：\[{I(U-F, V-F) = b^2 T(U,V) + a^2 T(F,U) T(F,V)}\]
特别地，如果 $A$ 在椭圆上，$T(A,A)=0$，焦半径平方 $AF^2 = I(A-F, A-F) = a^2 T(F,A)^2 \implies AF = |a T(F,A)| = a - \frac{c}{a}x_A$。这就是课本里的焦半径公式。

题目说 $F$ 是 $\triangle ABC$ 的垂心，说明 $AF \perp BC$。由于三角形 $A, B, C$ 地位对称，这三个点向 $F$ 连线的点乘一定等于同一个常数。用点乘写就是：
\[I(A-F, B-C) = 0 \implies I(A-F, B-F) = I(A-F, C-F) = k\]
代入$I(U-F, V-F) = b^2 T(U,V) + a^2 T(F,U) T(F,V)$，这意味着， $B$ 和 $C$ 两点同时满足下面这个包含变量 $X$ 的等式，即直线 $BC$ 的方程。$$ b^2 T(A,X) + a^2 T(F,A) T(F,X) = k $$
为了把常数 $k$ 算出来，必要性探路：让 $A$ 在椭圆右顶点 $(a,0)$。因为 $F$ 是垂心，由对称性 $BC$ 是一条垂直于 $x$ 轴的直线，设 $B$ 的横坐标为 $m$，坐标 $(m, y_B)$。因为 $BF \perp AC$，写出向量 $\overrightarrow{BF} = (c-m, -y_B)$，$\overrightarrow{AC} = (m-a, -y_B)$。点乘等于 0：$(c-m)(m-a) + y_B^2 = 0$。把椭圆方程 $y_B^2 = b^2(1 - \frac{m^2}{a^2}) = \frac{b^2}{a^2}(a-m)(a+m)$ 代入，约掉 $(a-m)$：$c-m + \frac{b^2}{a^2}(a+m) = 0$。稍微化简解得 $m = \frac{a(ac-b^2)}{a^2+b^2}$。算常数 $k = I(A-F, B-F) = (a-c)(m-c) + 0$。把 $m$ 塞进去：
\[k = (a-c) \left( \frac{a^2 c - a b^2 - a^2 c - c b^2}{a^2+b^2} \right) = (a-c) \frac{-b^2(a+c)}{a^2+b^2} = -\frac{b^4}{a^2+b^2}\]
把 $k$ 代回算子方程，完全展开：
\[b^2(\frac{x_A x}{a^2} + \frac{y_A y}{b^2} - 1) + a^2(\frac{cx_A}{a^2} - 1)(\frac{cx}{a^2} - 1) = -\frac{b^4}{a^2+b^2}\]
整理得到直线 $BC$： \[(x_A - c)x + y_A y - M = 0,~~\boxed{M = cx_A - \frac{a^4}{a^2+b^2}}\]
解答 (iii)：证明 $AF\cdot FI = \text{常数}$\\
线段 $FI$ 的物理意义，单纯就是焦点 $F(c,0)$ 到直线 $BC$ 的距离。
\begin{align*}
FI=d=&\frac{|(x_A - c)c + y_A \cdot 0 - M|}{\sqrt{(x_A - c)^2 + y_A^2}}=\frac{|c x_A - c^2 - \left(cx_A - \frac{a^4}{a^2+b^2}\right)|}{|AF|}=\frac{|-c^2 + \frac{a^4}{a^2+b^2}|}{|AF|}\\
=& \frac{\frac{b^4}{a^2+b^2}}{AF} \implies AF \cdot FI = \frac{b^4}{a^2+b^2}
\end{align*}
由轮换对称性，$BF \cdot FJ$ 和 $CF \cdot FK$ 自然也等于它。(iii) 证毕。\\
解答 (i)：证明 $\triangle ABC$ 与定圆相切，就是找一个定点 $I_0(d,0)$（由对称性圆心在 $x$ 轴），让它到直线 $BC$ 的距离恒为常数 $r$。
\begin{align*}
r=\frac{|(x_A - c)d - M|}{\sqrt{(x_A - c)^2 + y_A^2}}=\frac{ |d x_A - cd - cx_A + \frac{a^4}{a^2+b^2}|}{|AF|}=\frac{ |d x_A - cd - cx_A + \frac{a^4}{a^2+b^2}|}{a - \frac{c}{a}x_A}
\end{align*}
分子分母对应系数成比例：
$$ \frac{d - c}{-c/a} = \frac{-cd + \frac{a^4}{a^2+b^2}}{a} \implies d = \frac{c a^2}{a^2+b^2}$$
把圆心代回比例求半径 $r$：$r = \frac{a(c - d)}{c} = a\left(1 - \frac{a^2}{a^2+b^2}\right) = {\frac{a b^2}{a^2+b^2}}$。无论三角形怎么动，它永远与这个圆完美相切！(i) 证毕！\\
解答 (iv)：证明 $\triangle A'B'C'$ 垂心 $H$ 为定点\\
$A'$ 是切线 $B$ 和切线 $C$ 的交点，这意味着 $A'$ 就是动弦 $BC$ 的极点。所以 $BC$ 方程是 $\frac{x_{A'} x}{a^2} + \frac{y_{A'} y}{b^2} - 1 = 0$，我们将 $BC$ 方程除以常数项 $M$，即 $\frac{x_A - c}{M}x + \frac{y_A}{M}y - 1 = 0$。对比系数得出 $A'$ 的坐标：$x_{A'} = \frac{a^2(x_A - c)}{M}, y_{A'} = \frac{b^2 y_A}{M}$。现在找垂心 $H(x_H, 0)$。高线 $A'H$ 必定垂直于对边 $B'C'$。
对边 $B'C'$ 就是点 $A$ 的极线，法向量是 $\vec{n} = (\frac{x_A}{a^2}, \frac{y_A}{b^2})$。
垂直说明向量 $\overrightarrow{HA'} = (x_{A'} - x_H, y_{A'})$ 和法向量平行。利用外积算子交叉相乘为 0：$$ (x_{A'} - x_H)\frac{y_A}{b^2} - y_{A'}\frac{x_A}{a^2} = 0 \implies x_H = x_{A'} - y_{A'} \frac{b^2 x_A}{a^2 y_A} $$
把刚刚求出的 $A'$ 坐标代入：
\begin{align*}
x_H =& \frac{a^2(x_A - c)}{M} - \frac{b^2 y_A}{M} \cdot \frac{b^2 x_A}{a^2 y_A} = \frac{a^4(x_A - c) - b^4 x_A}{a^2 M}=\frac{(a^4-b^4)x_A - a^4 c}{a^2 M}\\
=&\frac{c^2(a^2+b^2)x_A - a^4 c}{a^2 \left(cx_A - \frac{a^4}{a^2+b^2}\right)}=\frac{c(a^2+b^2)}{a^2}
\end{align*}
垂心 $H\left(\frac{c(a^2+b^2)}{a^2},0\right)$ 。(iv) 得证！\\
解答 (ii)：假设未知定圆是 $x^2+y^2=R^2$（由对称性，圆心必在原点）。求出外接圆和它的“交线”（根轴），只要这根轴与定圆相切就行！外接圆必定由椭圆 $\Omega=0$ 和两条相交直线拼成，$L_{BC}$ 已知。为了消除 $xy$ 项构成圆，过 $A$ 的弦 $L_{AD}$ 斜率必须与 $BC$ 相反。
\[C_{circ}: L_{BC} \cdot L_{AD} + \lambda \Omega = 0,L_{AD}: (x_A-c)x - y_A y - N = 0,L_{BC}:(x_A - c)x + y_A y - M = 0\]
已知$M = cx_A - \frac{a^4}{a^2+b^2}$，代入 $A$ 算出 $N = \frac{a^2+b^2}{a^2}x_A^2 - cx_A - b^2$。要让 $x^2, y^2$ 系数相等：$(x_A-c)^2 + \frac{\lambda}{a^2} = -y_A^2 + \frac{\lambda}{b^2}$，解出 $\lambda = \frac{a^2 b^2}{c^2} AF^2$。直接代入 $y^2 = R^2 - x^2$ 到外接圆方程中：
先看 $\lambda \Omega$ 部分：
$$ \lambda \Omega = \lambda \left( \frac{x^2}{a^2} + \frac{R^2-x^2}{b^2} - 1 \right) = \lambda \left[ -x^2\left(\frac{1}{b^2}-\frac{1}{a^2}\right) + \frac{R^2-b^2}{b^2} \right] $$
代入 $\lambda = \frac{a^2 b^2}{c^2} AF^2$，括号里的 $x^2$ 系数刚好约掉，变成了 $-AF^2 x^2$！再看 $L_{BC} \cdot L_{AD}$ 部分，展开后含有 $x^2$ 和 $y^2$ 的项是 $(x_A-c)^2 x^2 - y_A^2 y^2$。代入 $y^2 = R^2 - x^2$：
$$ \left[(x_A-c)^2 + y_A^2\right]x^2 - y_A^2 R^2 = AF^2 x^2 - y_A^2 R^2 $$
把这两部分相加求根轴：$AF^2 x^2$ 和 $-AF^2 x^2$ 奇迹般地完全抵消！ 方程真的降阶成了一条直线。把直线方程整理好，常数项是这样的：
$$ \text{常数项} = MN - y_A^2 R^2 + \lambda \frac{R^2-b^2}{b^2} = MN - y_A^2 R^2 + AF^2 \frac{a^2(R^2-b^2)}{c^2} $$
把 $AF^2 = (x_A-c)^2 + y_A^2$ 拆进去，把包含 $y_A^2$ 的项合并，发现 $y_A^2$ 的系数是：
$$ -R^2 + \frac{a^2(R^2-b^2)}{c^2} = \frac{-c^2 R^2 + a^2 R^2 - a^2 b^2}{c^2} = \frac{b^2}{c^2}(R^2 - a^2) $$
要证明相切，原点到根轴的距离公式里会用到常数项。如果 $R \neq a$，常数项里就会残留 $y_A^2$，根本不可能恒等。
尝试让 $\frac{b^2}{c^2}(R^2 - a^2) = 0 \implies {R = a}$。猜想定圆必然是 $x^2 + y^2 = a^2$。 $y_A^2$ 项消失。令 $D = a(x_A-c)$，常数项变成 $MN + D^2$。此时根轴方程为：
$$ -(M+N)(x_A-c)x + (M-N)y_A y + MN + D^2 = 0 $$
为了证明它与定圆相切，原点到它的距离必须等于 $a$：
$$ \frac{(MN + D^2)^2}{(M+N)^2(x_A-c)^2 + (M-N)^2 y_A^2} = a^2 $$
交叉相乘并展开左边和右边第一项，移项相减：
\[(M^2 N^2 + 2D^2 MN + D^4) - (D^2 M^2 + 2D^2 MN + D^2 N^2) = a^2 y_A^2 (M-N)^2\]
中间的 $2D^2 MN$ 再次完美湮灭！提取左边，全题化归为证明下面这最后一行因式分解：
$$ \mathbf{(D^2 - M^2)(D^2 - N^2) = a^2 y_A^2 (M-N)^2} $$
在草稿纸上强拆这三个式子时，我发现它们都在疯狂吐出一个共同的核心二次因式（我称之为“黄金因子 $P$”）：$$ P(x_A) = x_A^2 - \frac{2a^2c}{a^2+b^2}x_A + \frac{a^6 - a^4 b^2 - a^2 b^4}{(a^2+b^2)^2} $$
多项式硬碰硬的展开坍缩如下（纯初中代数，绝无跳步）：
\begin{itemize}
\item $D^2 - M^2 = a^2(x_A-c)^2 - (cx_A - \frac{a^4}{a^2+b^2})^2 = b^2 \cdot P(x_A)$
\item $M - N = (cx_A - \frac{a^4}{a^2+b^2}) - (\frac{a^2+b^2}{a^2}x_A^2 - cx_A - b^2) =-\frac{a^2+b^2}{a^2} \cdot P(x_A)$
\item $D^2 - N^2 = a^2(x_A-c)^2 - (\frac{a^2+b^2}{a^2}x_A^2 - cx_A - b^2)^2$
\end{itemize}
利用因式定理易知，当 $x_A = a$ 和 $x_A = -a$ 时该式均等于 0。除掉必含的因式 $(a^2 - x_A^2)$ 后，它正好等于：$\frac{(a^2+b^2)^2}{a^4} (a^2 - x_A^2) \cdot P(x_A)$把这三斩结果代回要证的等式：
左边 
\[= \big[ b^2 P(x_A) \big] \times \left[ \frac{(a^2+b^2)^2}{a^4} (a^2 - x_A^2) P(x_A) \right] = {b^2(a^2 - x_A^2) \frac{(a^2+b^2)^2}{a^4} [P(x_A)]^2}\]
右边 \[= a^2 y_A^2 \times (M-N)^2\]
利用 $a^2 y_A^2 = b^2(a^2 - x_A^2)$，代入：$= b^2(a^2 - x_A^2) \times \left[ -\frac{a^2+b^2}{a^2} P(x_A) \right]^2 = \mathbf{b^2(a^2 - x_A^2) \frac{(a^2+b^2)^2}{a^4} [P(x_A)]^2}$
所以原点到根轴的距离严格等于 $a$ 恒成立。外接圆必定与 $x^2+y^2=a^2$ 相切。
\end{solution}

\begin{example}{乐乐猴}{}
已知抛物线 $C_1: y^2 = 4cx\ (c > 0)$ 和双曲线 $C_2: \frac{x^2}{a^2} - \frac{y^2}{b^2} = 1\ (a > 0, b > 0)$ 共用一个焦点 $F(c, 0)$。设直线 $l$ 是 $C_1$ 和 $C_2$ 的公切线，$l$ 与 $C_1$ 相切于点 $A$，与 $C_2$ 相切于点 $B$。已知 $C_1$ 经过点 $P(1, 2)$。
\begin{enumerate}
    \item 求 $c$ 的值；
    \item 求公切线 $l$ 的方程（用参数 $a$ 表示）；
    \item 求 $\triangle ABF$ 的外接圆半径 $R$ 的最小值。
\end{enumerate}
\end{example}
\begin{solution}
（1）将点 $P(1,2)$ 代入抛物线 $C_1: y^2 = 4cx$，直接得到 $2^2 = 4c \times 1 \implies c = 1$。因此两曲线共用焦点 $F(1, 0)$，抛物线方程为 $y^2 = 4x$。\\
（2）对于双曲线 $C_2$有：
$$ a^2 + b^2 = c^2 = 1 \implies b^2 = 1 - a^2 \quad (0 < a < 1) $$
写出抛物线和双曲线各自的极化算子（二次型）：
\begin{itemize}
    \item $C_1$ 的极化算子：$T_1(U, V) = y_U y_V - 2(x_U + x_V)$
    \item $C_2$ 的极化算子：$T_2(U, V) = \frac{x_U x_V}{a^2} - \frac{y_U y_V}{b^2} - 1$
\end{itemize}
设公切线 $l$ 方程为 $ux + vy - 1 = 0$。既然 $l$ 与 $C_1$ 切于点 $A$，那么 $l$ 必须与极线 $T_1(A, X) = 0$ 等价：
$$ -2x + y_A y - 2x_A = 0 \implies -\frac{1}{x_A}x + \frac{y_A}{2x_A}y - 1 = 0 $$
对比系数得：
\[u = -\frac{1}{x_A}, v = \frac{y_A}{2x_A}\implies x_A = -\frac{1}{u}, y_A = -\frac{2v}{u}\]
代入 $T_1(A,A) = 0 \implies y_A^2 = 4x_A$：
$$ \left(-\frac{2v}{u}\right)^2 = 4\left(-\frac{1}{u}\right) \implies \frac{4v^2}{u^2} = -\frac{4}{u} \implies u = -v^2 $$
同理，既然 $l$ 与 $C_2$ 切于点 $B$，那么 $l$ 必须与极线 $T_2(B, X) = 0$ 等价：
$$ \frac{x_B}{a^2}x - \frac{y_B}{b^2}y - 1 = 0 \implies u = \frac{x_B}{a^2}, v = -\frac{y_B}{b^2}\implies x_B = a^2 u, y_B = -b^2 v $$
代入 $T_2(B,B) = 0 \implies \frac{x_B^2}{a^2} - \frac{y_B^2}{b^2} = 1$：
$$ \frac{(a^2 u)^2}{a^2} - \frac{(-b^2 v)^2}{b^2} = 1 \implies a^2 u^2 - b^2 v^2 = 1$$
将 $b^2 = 1 - a^2$ 和 $v^2 = -u$ 联合代入消去 $v$：
$$ a^2 u^2 - (1-a^2)(-u) = 1 \implies a^2 u^2 + (1-a^2)u - 1 = 0 \implies (u - 1)(a^2 u + 1) = 0$$
因为抛物线切点 $x_A \ge 0$，故 $u = -\frac{1}{x_A} < 0$（且 $u = -v^2 \le 0$），必定取唯一解 $u = -\frac{1}{a^2}$。
从而 $v^2 = -u = \frac{1}{a^2} \implies v = \pm \frac{1}{a}$。
代回 $ux + vy = 1$，公切线 $l$ 的方程
$$ -\frac{1}{a^2}x \pm \frac{1}{a}y = 1 \implies x \mp ay + a^2 = 0 $$
（3）为了求 $\triangle ABF$ 的外接圆，将 $u, v$ 代回切点坐标公式：
\begin{itemize}
\item 对于抛物线切点 $A$：$x_A = -\frac{1}{u} = a^2$，$y_A = -\frac{2v}{u} = \pm 2a \implies A(a^2, \pm 2a)$。$ \overrightarrow{FA} = A - F = (a^2-1, \pm 2a) $
\item 对于双曲线切点 $B$：$x_B = a^2 u = -1$，$y_B = -b^2 v = -(1-a^2)(\pm \frac{1}{a}) = \mp \frac{1-a^2}{a}$。$ \overrightarrow{FB} = B - F = \left(-2, \mp \frac{1-a^2}{a}\right) $
\end{itemize}
由
\begin{align*}
I(\overrightarrow{FA}, \overrightarrow{FB}) &= (a^2-1)(-2) + (\pm 2a)\left(\mp \frac{1-a^2}{a}\right) \\
&= -2(a^2-1) - 2(1-a^2) \\
&= -2a^2 + 2 - 2 + 2a^2 \equiv 0 !
\end{align*}
于是$\overrightarrow{FA} \perp \overrightarrow{FB}$，即 $\triangle ABF$ 是一个以 $F$ 为直角顶点的直角三角形。
它的外接圆圆心必定是斜边 $AB$ 的中点，外接圆半径 $R$ 恰好等于斜边长度的一半，即 $4R^2 = |AB|^2$。
利用内积算子计算斜边模长的平方，由于两向量正交，满足完美的勾股定理 $|AB|^2 = |\overrightarrow{FA}|^2 + |\overrightarrow{FB}|^2$：
$$ |\overrightarrow{FA}|^2 = (a^2-1)^2 + 4a^2 = (a^2+1)^2 $$
$$ |\overrightarrow{FB}|^2 = 4 + \left(\frac{1-a^2}{a}\right)^2 = \frac{4a^2 + 1 - 2a^2 + a^4}{a^2} = \frac{(a^2+1)^2}{a^2} $$
相加并提取公因式，得到半径平方的绝对目标函数：
$$ 4R^2 = (a^2+1)^2 + \frac{(a^2+1)^2}{a^2} = (a^2+1)^2 \left( 1 + \frac{1}{a^2} \right) = \frac{(a^2+1)^3}{a^2} $$
令 $t = a^2$。因为双曲线有 $a^2 + b^2 = 1$ 且 $b>0$，故定义域为 $t \in (0, 1)$。
构造核心极值函数 $f(t) = \frac{(t+1)^3}{t}$。对其求导寻找极小值：
$$ f'(t) = \frac{3(t+1)^2 \cdot t - (t+1)^3 \cdot 1}{t^2} = \frac{(t+1)^2 (3t - (t+1))}{t^2} = \frac{(t+1)^2 (2t - 1)}{t^2} $$
令 $f'(t) = 0$，在区间 $(0,1)$ 内唯一解为 $t = \frac{1}{2}$（即 $a^2 = \frac{1}{2}$）。
且导数在 $t = 1/2$ 处左负右正，故取得全局最小值。将 $t = \frac{1}{2}$ 代入，得到最小半径的平方：
$$ 4R_{\min}^2 = f\left(\frac{1}{2}\right) = \frac{(\frac{1}{2} + 1)^3}{\frac{1}{2}} = 2 \times \frac{27}{8} = \frac{27}{4} \implies R_{\min}^2 = \frac{27}{16} $$
开根号，无情输出最终答案：
$$ R_{\min} = \frac{3\sqrt{3}}{4} $$
\end{solution}

\begin{example}{定弦长面积问题}{}
 己知椭圆$C:\frac{x^2}{a^2}+\frac{y^2}{b^2}=1(a>b>0),A,B$ 为椭圆$C$ 上的两点，且
$\left|AB\right|=l(0<l<2a)$,试求$\Delta OAB$ 面积的最大值与最小值。
\end{example}
\begin{solution}
我们有 $\triangle OAB$ 面积的纯算子公式：$$S_{\triangle OAB} = \frac{1}{2}ab\sqrt{-T(A,B)\big(T(A,B)+2\big)}$$让我们定义一个干净的变量 $x = -T(A,B)$。因为 $A$ 和 $B$ 都在椭圆上，它们的内积 $E(A,B)$ 的范围是 $[-1, 1]$。因为 $T(A,B) = E(A,B) - 1$，所以我们知道 $T(A,B) \in [-2, 0]$。这完美地将我们的新变量界定在 $x \in [0, 2]$ 的范围内。我们的面积函数现在变成了一个绝美的、纯代数的抛物线（在根号内）：$$S = \frac{1}{2}ab\sqrt{x(2-x)} = \frac{1}{2}ab\sqrt{1 - (x-1)^2}$$现在，我们需要将几何长度 $l = |AB|$ 与算子 $x$ 绑定。利用资料中向量内积的展开（拉格朗日恒等式框架）：
$$E(\overrightarrow{AB}, \overrightarrow{AB}) = E(A-B, A-B) = E(A,A) - 2E(A,B) + E(B,B)$$
因为 $A$ 和 $B$ 在椭圆上，$E(A,A) = E(B,B) = 1$。
$$E(\overrightarrow{AB}, \overrightarrow{AB}) = 2 - 2E(A,B) = -2T(A,B) = 2x$$
根据标准定义，$E(\overrightarrow{AB}, \overrightarrow{AB}) = \frac{\Delta x^2}{a^2} + \frac{\Delta y^2}{b^2}$。我们同时也知道真实的欧氏长度平方为 $l^2 = \Delta x^2 + \Delta y^2$。因为 $\Delta x^2 \geq 0$ 且 $\Delta y^2 \geq 0$，假设 $a > b > 0$，我们可以利用长短半轴对算子的值进行放缩夹逼：
$$\frac{l^2}{a^2} \leq \frac{\Delta x^2}{a^2} + \frac{\Delta y^2}{b^2} \leq \frac{l^2}{b^2}$$
将中间项替换为 $2x$，我们就得到了能够解决这两个问题的核心不等式：$$\frac{l^2}{2a^2} \leq x \leq \frac{l^2}{2b^2}$$
问题 1：定弦长面积问题我们需要在区间 $x \in [x_{\min}, x_{\max}]$ 内求面积函数 $S = \frac{1}{2}ab\sqrt{1 - (x-1)^2}$ 的最大值和最小值，其中 $x_{\min} = \frac{l^2}{2a^2}$，$x_{\max} = \min\left(2, \frac{l^2}{2b^2}\right)$。根号下的二次函数是一个开口向下的抛物线，其绝对顶点恰好在 $x=1$ 处。\\
最大面积：我们只需要让 $x$ 尽可能靠近顶点 $1$。
\begin{itemize}
\item 情况 1： 如果 $x_{\min} \geq 1 \implies l \geq \sqrt{2}a$，我们能最靠近 $1$ 的方式就是取左边界 $x_{\min}$。代入 $x = \frac{l^2}{2a^2}$：$$S = \frac{1}{2}ab\sqrt{\frac{l^2}{2a^2}\left(2 - \frac{l^2}{2a^2}\right)} = \frac{bl}{4a}\sqrt{4a^2 - l^2}$$
\item 情况 2： 如果 $x_{\max} \leq 1 \implies l \leq \sqrt{2}b$，能最靠近 $1$ 的方式是取右边界 $x_{\max}$。代入 $x = \frac{l^2}{2b^2}$：$$S = \frac{1}{2}ab\sqrt{\frac{l^2}{2b^2}\left(2 - \frac{l^2}{2b^2}\right)} = \frac{al}{4b}\sqrt{4b^2 - l^2}$$
\item 情况 3： 如果 $\sqrt{2}b < l < \sqrt{2}a$，顶点 $x=1$ 完美地落在我们的区间内！代入 $x=1$：$$S = \frac{1}{2}ab\sqrt{1} = \frac{ab}{2}$$
\end{itemize}
最小面积：我们希望 $x$ 尽可能远离顶点 $1$，所以我们要比较两个极端边界。让我们通过平方来找到让两个边界产生相同面积的临界点：$$\frac{b^2l^2}{16a^2}(4a^2 - l^2) = \frac{a^2l^2}{16b^2}(4b^2 - l^2)$$解这个方程，极其漂亮地坍缩为 $l = \frac{2ab}{\sqrt{a^2+b^2}}$。
\begin{itemize}
\item 情况 1： 如果 $0 < l \leq \frac{2ab}{\sqrt{a^2+b^2}}$，左端点 $x_{\min}$ 离顶点更远，所以最小面积是 $\frac{bl}{4a}\sqrt{4a^2 - l^2}$。
\item 情况 2： 如果 $\frac{2ab}{\sqrt{a^2+b^2}} < l < 2b$，右端点 $x_{\max}$ 离顶点更远，得到 $\frac{al}{4b}\sqrt{4b^2 - l^2}$。
\item 情况 3： 如果 $l \geq 2b$，我们的 $x_{\max}$ 达到了绝对极限 $2$。在 $x=2$ 时，弦穿过原点（或退化），面积完美归零。
\end{itemize}
\begin{align*}&S_{\mathrm{max}}=\begin{cases}\frac{al\sqrt{4b^2-l^2}}{4b},0<l\leq\sqrt{2}b;\\\frac{ab}{2},\sqrt{2}b<l<\sqrt{2}a;\\\frac{bl\sqrt{4a^2-l^2}}{4a},\sqrt{2}a\leq l<2a.&\end{cases}\\&S_{\mathrm{min}}=\begin{cases}\frac{bl\sqrt{4a^2-l^2}}{4a},0<l\leq\frac{2ab}{\sqrt{a^2+b^2}};\\\frac{al\sqrt{4b^2-l^2}}{4b},\frac{2ab}{\sqrt{a^2+b^2}}<l<2b;\\0,2b\leq l<2a.&\end{cases}\end{align*}
\end{solution}

\begin{example}{定距离面积问题}{}
 已知椭圆 $C: \frac{x^2}{a^2} + \frac{y^2}{b^2} = 1 (a > b > 0)$, $A, B$ 为椭圆 $C$ 上的两点，且原点 $O$ 到直线 $AB$ 距离为 $d (0 < d < a)$，试求 $\triangle OAB$ 面积的最大值与最小值
\end{example}
\begin{solution}
这就是算子法展现真正威力的地方。我们根本不需要从头再来！只需利用基础的三角形面积公式将距离 $d$ 和弦长 $l$ 联系起来：
$$S = \frac{1}{2}ld \implies l = \frac{2S}{d}$$
我们拿出问题 1 中的黄金不等式 $\frac{l^2}{2a^2} \leq x \leq \frac{l^2}{2b^2}$，并将 $l$ 替换掉：
$$\frac{2S^2}{a^2d^2} \leq x \leq \frac{2S^2}{b^2d^2}$$
等等，我们已经有了 $S^2$ 的纯算子表达式！它是 $S^2 = \frac{1}{4}a^2b^2x(2-x)$。让我们把它们拼在一起：
\begin{itemize}
\item 左侧： $\frac{2}{a^2d^2}\left(\frac{1}{4}a^2b^2x(2-x)\right) \leq x \implies \frac{b^2}{2d^2}(2-x) \leq 1 \implies x \geq 2 - \frac{2d^2}{b^2}$
\item 右侧： $x \leq \frac{2}{b^2d^2}\left(\frac{1}{4}a^2b^2x(2-x)\right) \implies 1 \leq \frac{a^2}{2d^2}(2-x) \implies x \leq 2 - \frac{2d^2}{a^2}$
\end{itemize}
看哪！面积公式直接把 $x$ 的全新区间端到了我们面前，完美地交换了 $a$ 和 $b$ 的位置：
$$x \in \left[\max\left(0, 2 - \frac{2d^2}{b^2}\right), 2 - \frac{2d^2}{a^2}\right]$$
让我们用这些新边界来计算面积函数 $S = \frac{1}{2}ab\sqrt{x(2-x)}$：
\begin{itemize}
\item 在 $x_{\max} = 2 - \frac{2d^2}{a^2}$ 处： $S_a = \frac{1}{2}ab\sqrt{\left(2 - \frac{2d^2}{a^2}\right)\left(\frac{2d^2}{a^2}\right)} = \frac{bd}{a}\sqrt{a^2 - d^2}$
\item 在 $x_{\min} = 2 - \frac{2d^2}{b^2}$ 处： $S_b = \frac{1}{2}ab\sqrt{\left(2 - \frac{2d^2}{b^2}\right)\left(\frac{2d^2}{b^2}\right)} = \frac{ad}{b}\sqrt{b^2 - d^2}$
\end{itemize}
最大面积：我们再次检查 $x=1$ 是否落在我们的新区间内：$$2 - \frac{2d^2}{b^2} \leq 1 \leq 2 - \frac{2d^2}{a^2} \implies \frac{\sqrt{2}b}{2} \leq d \leq \frac{\sqrt{2}a}{2}$$
\begin{itemize}
\item 如果 $0 < d \leq \frac{\sqrt{2}b}{2}$，$x=1$ 严格在左侧。最大面积在 $x_{\min}$ 处取得，结果为 $S_b = \frac{ad}{b}\sqrt{b^2 - d^2}$。
\item 如果 $\frac{\sqrt{2}b}{2} < d < \frac{\sqrt{2}a}{2}$，顶点 $1$ 在区间内！最大面积为 $\frac{ab}{2}$。
\item 如果 $\frac{\sqrt{2}a}{2} \leq d < a$，$x=1$ 严格在右侧。最大面积在 $x_{\max}$ 处取得，结果为 $S_a = \frac{bd}{a}\sqrt{a^2 - d^2}$。
\end{itemize}
最小面积：令 $S_a = S_b$ 来寻找端点反转的临界值，得到 $d = \frac{ab}{\sqrt{a^2+b^2}}$。
\begin{itemize}
\item 如果 $0 < d \leq \frac{ab}{\sqrt{a^2+b^2}}$，离 $1$ 最远的端点是 $x_{\max}$。最小面积是 $S_a = \frac{bd}{a}\sqrt{a^2 - d^2}$。
\item 如果 $\frac{ab}{\sqrt{a^2+b^2}} < d < b$，最远的端点是 $x_{\min}$。最小面积是 $S_b = \frac{ad}{b}\sqrt{b^2 - d^2}$。
\item 如果 $b \leq d < a$，$x_{\min}$ 触底到 $0$，意味着面积完全归零。
\end{itemize}
    $$S_{\max} = \begin{cases} \dfrac{ad\sqrt{b^2 - d^2}}{b}, & 0 < d \leq \dfrac{\sqrt{2}b}{2}; \\ \dfrac{ab}{2}, & \dfrac{\sqrt{2}b}{2} < d < \dfrac{\sqrt{2}a}{2}; \\ \dfrac{bd\sqrt{a^2 - d^2}}{a}, & \dfrac{\sqrt{2}a}{2} \leq d < a. \end{cases}$$
$$S_{\min} = \begin{cases} \dfrac{bd\sqrt{a^2 - d^2}}{a}, & 0 < d \leq \dfrac{ab}{\sqrt{a^2 + b^2}}; \\ \dfrac{ad\sqrt{b^2 - d^2}}{b}, & \dfrac{ab}{\sqrt{a^2 + b^2}} < d < b; \\ 0, & b \leq d < a. \end{cases}$$
\end{solution}

\begin{example}{中心三相切}{}
已知点$P$为椭圆$C_1:\frac{x^2}{a^2}+\frac{y^2}{b^2}=1$上的一动点，过点$P$作圆$C_2:x^2+y^2=(\frac{ab}{a+b})^2$的两条切线分别交椭圆于$M,N$两点，求证：直线$MN$与圆$C_2$相切；
\end{example}
\begin{solution}
题目里有两个二次曲线，我们先把它们的算子写得明明白白：
\begin{itemize}
\item 外侧的椭圆 $C_1$：$T_1(X,X) = \frac{x^2}{a^2} + \frac{y^2}{b^2} - 1 = 0$。动点 $P(x_P, y_P)$ 在椭圆上，所以它自身的极化能量归零：$T_1(P,P) = 0$。
\item 内侧的圆 $C_2$：它的方程是 $x^2+y^2 = R^2$（这里 $R = \frac{ab}{a+b}$）。内积算子可以直接写为 $E_2(X,X) = \frac{x^2+y^2}{R^2}$，极化算子就是 $T_2(X,X) = \frac{x^2+y^2}{R^2} - 1 = 0$。
\end{itemize}
从点 $P$ 引向圆 $C_2$ 的两条切线 $PM$ 和 $PN$，它们的总方程可以直接一秒写出：
$$\big[T_2(P,X)\big]^2 = T_2(P,P) \cdot T_2(X,X)$$
把坐标代进去：
$$\left( \frac{x_P x + y_P y}{R^2} - 1 \right)^2 = \left( \frac{x_P^2 + y_P^2}{R^2} - 1 \right) \left( \frac{x^2 + y^2}{R^2} - 1 \right)$$
两边同乘 $R^4$ 把恶心的分母干掉，然后像初中生一样暴力展开并移项：
$$(x_P x + y_P y - R^2)^2 = (x_P^2 + y_P^2 - R^2)(x^2 + y^2 - R^2)$$
合并同类项，我们得到了这对切线的纯二次方程 $F(X) = 0$：$$F(X) = (y_P^2 - R^2)x^2 + (x_P^2 - R^2)y^2 - 2x_P y_P xy + 2R^2 x_P x + 2R^2 y_P y - R^2(x_P^2 + y_P^2) = 0$$
这两条切线交椭圆于 $M, N$ 两点，同时它们俩原本就相交于点 $P$（点 $P$ 也在椭圆上）。这意味着，外切线对 $F(X)=0$ 和椭圆 $T_1(X,X)=0$ 有四个交点：$P, P, M, N$（点 $P$ 是双重交点）。
过这四个交点的任何曲线，都可以写成：外切线对 $+ \lambda \cdot$ 椭圆 $= 0$。既然点 $P$ 处是双重相交（相切），那么过这四个点的“退化二次曲线”就必定包含两条直线：一条是椭圆在点 $P$ 处的切线 $L_P$，另一条就是我们梦寐以求的直线 $MN$！所以，必定存在一个常数 $\lambda$，使得：
$$F(X) + \lambda \cdot T_1(X,X) \equiv L_P(X) \cdot L_{MN}(X)$$
椭圆在 $P$ 点的切线算子方程闭着眼就能写出：$L_P(X) = \frac{x_P x}{a^2} + \frac{y_P y}{b^2} - 1$。
我们设未知的直线 $MN$ 方程为 $L_{MN}(X) = ux + vy - w$。
恒等式左边加上 $\lambda \cdot \left(\frac{x^2}{a^2} + \frac{y^2}{b^2} - 1\right)$ 后，必须严格等于右边展开的：$$\left( \frac{x_P x}{a^2} + \frac{y_P y}{b^2} - 1 \right) (ux + vy - w)$$
提取系数与完美坍缩我们根本不用管 $\lambda$ 是多少，只盯住那些一次项和交叉项（左右两边系数必须绝对相等）！把右边展开，去找对应的系数：
\begin{itemize}
\item 看 $x$ 项系数：右边是 $-u - \frac{w x_P}{a^2}$，左边 $F(X)$ 里是 $2R^2 x_P$。$\implies u = -x_P \left( 2R^2 + \frac{w}{a^2} \right)$
\item 看 $y$ 项系数：右边是 $-v - \frac{w y_P}{b^2}$，左边 $F(X)$ 里是 $2R^2 y_P$。$\implies v = -y_P \left( 2R^2 + \frac{w}{b^2} \right)$
\end{itemize}
现在，看最核心的 $xy$ 交叉项系数！左边 $F(X)$ 里的 $xy$ 系数是纯粹的 $-2x_P y_P$。右边的 $xy$ 系数是 $\frac{v x_P}{a^2} + \frac{u y_P}{b^2}$。把刚刚求出的 $u, v$ 塞进去：
$$-2x_P y_P = \frac{x_P}{a^2}\left[ -y_P \left( 2R^2 + \frac{w}{b^2} \right) \right] + \frac{y_P}{b^2}\left[ -x_P \left( 2R^2 + \frac{w}{a^2} \right) \right]$$
等式两边都有 $-x_P y_P$，直接潇洒地约掉，剩下：
$$2 = 2R^2 \left( \frac{1}{a^2} + \frac{1}{b^2} \right) + \frac{2w}{a^2 b^2} \implies 1 = R^2 \frac{a^2+b^2}{a^2 b^2} + \frac{w}{a^2 b^2}$$
见证奇迹的时刻到了！题目已知 $R = \frac{ab}{a+b}$，所以 $\frac{1}{R} = \frac{a+b}{ab} = \frac{1}{a} + \frac{1}{b}$。对它平方：$\frac{1}{R^2} = \frac{1}{a^2} + \frac{1}{b^2} + \frac{2}{ab} = \frac{a^2+b^2}{a^2 b^2} + \frac{2}{ab}$。
把它代入上面的等式：$$\frac{w}{a^2 b^2} = 1 - R^2 \frac{a^2+b^2}{a^2 b^2} = R^2 \left( \frac{1}{R^2} - \frac{a^2+b^2}{a^2 b^2} \right) = R^2 \left( \frac{2}{ab} \right)$$瞬间解出完美常数：$$\mathbf{w = 2ab R^2}$$拿着这个 $w$ 回去把 $u$ 和 $v$ 彻底解脱：$$u = -x_P \left( 2R^2 + \frac{2ab R^2}{a^2} \right) = -2R^2 x_P \left( \frac{a+b}{a} \right)$$因为 $\frac{a+b}{a} = \frac{ab/R}{a} = \frac{b}{R}$，所以 $u = -2R^2 x_P \left( \frac{b}{R} \right) = \mathbf{-2b R x_P}$。同理，对称地算出 $v = \mathbf{-2a R y_P}$。我们直接把直线 $MN$ 的方程 $ux + vy - w = 0$ 拼出来了：$$-2b R x_P x - 2a R y_P y - 2ab R^2 = 0$$两边同除以 $-2R$，直线方程清爽出炉：$$\mathbf{b x_P x + a y_P y + ab R = 0}$$
直线 $MN$ 已经有了，要证明它和圆 $C_2: x^2+y^2=R^2$ 相切，只需要算原点 $(0,0)$ 到这条直线的距离 $d$ 是不是等于 $R$。距离公式直接甩上去：$$d = \frac{|ab R|}{\sqrt{(b x_P)^2 + (a y_P)^2}} = \frac{ab R}{\sqrt{b^2 x_P^2 + a^2 y_P^2}}$$千万别忘了，动点 $P(x_P, y_P)$ 在我们最初的椭圆 $C_1$ 上！把 $\frac{x_P^2}{a^2} + \frac{y_P^2}{b^2} = 1$ 两边同乘 $a^2b^2$，立刻得到：$$b^2 x_P^2 + a^2 y_P^2 = a^2 b^2$$所以分母 $\sqrt{b^2 x_P^2 + a^2 y_P^2} = \sqrt{a^2 b^2} = ab$。代入距离公式：$$d = \frac{ab R}{ab} = \mathbf{R}$$原点到直线 $MN$ 的距离严丝合缝地等于圆的半径 $R$！直线 $MN$ 必然与圆 $C_2$ 相切！证毕！
\end{solution}

\begin{example}{一般三相切}{}
    已知圆$G:(x-m)^2+(y-n)^2=r^2$为曲线$C_1:\frac{x^2}{a^2}+\frac{y^2}{b^2}=1(a>0,b>0)$内接三角形$ABC$的内切圆，过曲线$C_1$上一动点$P$作圆$G$的两条切线分别交椭圆于$M,N$两点，求证：直线$MN$与圆$G$相切；
\end{example}
\begin{solution}
设圆 $G$ 的极化算子为 $T_G$，椭圆 $C_1$ 的极化算子为 $T_E$。点 $P$ 在椭圆上引出圆 $G$ 的两条切线，其联合方程为：
$$\big[T_G(P,X)\big]^2 = T_G(P,P) \cdot T_G(X,X)$$
用“仿新齐次化”的思想，把坐标系的原点瞬间平移到动点 $P$ 上！设直线上任意一点 $X = P + \vec{v}$。我们把这个局部坐标代入上述方程，利用算子的分配律展开。
\begin{itemize}
\item 左边切线算子展开：$$T_G(P, P+\vec{v}) = T_G(P,P) + E_G(P,\vec{v})$$
\item 右边圆的算子展开：$$T_G(P+\vec{v}, P+\vec{v}) = T_G(P,P) + 2E_G(P,\vec{v}) + E_G(\vec{v},\vec{v})$$
\end{itemize}
把它们塞回联合方程中，像初中生一样展开完全平方：$$\big[T_G(P,P) + E_G(P,\vec{v})\big]^2 = T_G(P,P) \big[T_G(P,P) + 2E_G(P,\vec{v}) + E_G(\vec{v},\vec{v})\big]$$
\begin{itemize}
\item 展开左边：$T_G(P,P)^2 + 2T_G(P,P)E_G(P,\vec{v}) + [E_G(P,\vec{v})]^2$
\item 展开右边：$T_G(P,P)^2 + 2T_G(P,P)E_G(P,\vec{v}) + T_G(P,P)E_G(\vec{v},\vec{v})$
\end{itemize}
等号两边的常数项和一次项竟严丝合缝地完美抵消！方程瞬间坍缩为一个极度纯粹的纯二次型（斜率算子）：$$\underbrace{\big[E_G(P,\vec{v})\big]^2 - T_G(P,P) E_G(\vec{v},\vec{v})}_{H(\vec{v})} = 0$$这就代表了在以 $P$ 为原点的新坐标系下，那两条相交直线的纯粹方向方程 $H(\vec{v}) = 0$。
接下来，我们来看看椭圆 $C_1$ 在新坐标系下长什么样。因为点 $P$ 在椭圆上，必定满足 $T_E(P,P) = 0$。将动点 $X = P + \vec{v}$ 代入椭圆方程 $T_E(X,X)=0$，常数项直接归零，化简为：$$\underbrace{E_E(\vec{v},\vec{v})}_{\text{二次齐次项}} + 2\underbrace{E_E(P,\vec{v})}_{\text{一次线性项}} = 0$$
设我们要找的割线 $MN$ 在新坐标系下的方程为 $L(\vec{v}) + 1 = 0$（即 $-L(\vec{v}) = 1$）。我们将它与椭圆方程进行仿新齐次化配凑：$$E_E(\vec{v},\vec{v}) + 2E_E(P,\vec{v}) \cdot \big[-L(\vec{v})\big] = 0 \implies E_E(\vec{v},\vec{v}) - 2E_E(P,\vec{v}) L(\vec{v}) = 0$$在几何意义上，这个纯二次型代表了连接原点 $P$ 与交点 $M, N$ 的两条射线。等等！连接 $P$ 与 $M, N$ 的射线，不正是我们第一步求出的那两条切线 $PM$ 和 $PN$ 吗？！
既然这两个纯二次型代表的是同一对几何直线，那么它们在代数上必须是严格成比例的！设缩放倍数为 $\mu$，我们立刻得到了一个绝对恒等式：$$E_E(\vec{v},\vec{v}) - 2E_E(P,\vec{v}) L(\vec{v}) \equiv \mu \Big( \big[E_G(P,\vec{v})\big]^2 - T_G(P,P) E_G(\vec{v},\vec{v}) \Big)$$按照你笔记里“解释仿新齐次化”的套路，我们把未知量（直线 $L$）留在一边：$$\underbrace{E_E(\vec{v},\vec{v}) - \mu H(\vec{v})}_{\text{已知曲线减去缩放切线系}} \equiv \underbrace{2E_E(P,\vec{v})}_{\text{椭圆在P点的切线}} \cdot L(\vec{v})$$这行代数式宣告了绝对的胜利：割线 $MN$ 的方程 $L(\vec{v})$，完完全全被椭圆的二次项、圆的二次项以及常数 $\mu$ 通过多项式整除唯一确定！
如果我们把动点 $P$ 移动到顶点 $A$ 的位置，生成的切线就是 $AB$ 和 $AC$，而生成的割线 $MN$ 就是底边 $BC$。由于 $G$ 是内切圆，此时的割线 $BC$ 天然必定与圆 $G$ 相切！而在我们刚才无懈可击的算子代数推演中，椭圆的算子 $E_E$ 和圆的算子 $E_G$ 都是全局常数算子。缩放因子 $\mu$ 的提取只依赖于这两个二次型本身，与点 $P$ 在椭圆上的具体位置毫无关系！既然这套固定不变的代数生成法则在 $P=A$ 时吐出了一条相切的直线，那么根据算子结构的高度对称性与同构性，对于椭圆上任意一个动点 $P$，它吐出的割线 $MN$ 必然继承这一代数属性，绝对与圆 $G$ 相切！
\end{solution}

\begin{example}{$k_{OP}\cdot k_{AB}$}{}
    已知 $P$ 为椭圆 $\frac{x^2}{a^2}+\frac{y^2}{b^2}=1$ 上一动点，过点 $P$ 作圆 $x^2+y^2=r^2 (0<r<b)$ 的两条切线分别交椭圆于 $A,B$ 两点，求证： $k_{OP}\cdot k_{AB}=-\frac{b^2}{a^2}\cdot \frac{a^2b^2+r^2c^2}{a^2b^2-r^2c^2}$
\end{example}
\begin{solution}
设动点 $P$ 的坐标为 $(x_0, y_0)$，既然它在椭圆上，那天然就有 $\frac{x_0^2}{a^2} + \frac{y_0^2}{b^2} = 1$ 这个条件兜底。我们以 $P$ 为新原点，设新坐标系下的局部向量 $\vec{v} = (X, Y)$。根据咱们笔记里的外切线对联合方程的坍缩公式，过 $P$ 点作圆 $x^2+y^2=r^2$ 的两条切线，纯二次型方程 $H(X,Y)=0$ 是：
$$[E_G(P,\vec{v})]^2 - T_G(P,P) \cdot E_G(\vec{v},\vec{v}) = 0$$
咱们把圆的算子老老实实代进去：
\begin{itemize}
\item 交叉项：$E_G(P,\vec{v}) = \frac{x_0 X + y_0 Y}{r^2}$
\item 圆的二次项：$E_G(\vec{v},\vec{v}) = \frac{X^2 + Y^2}{r^2}$
\item 极化值（其实就是点到圆的相对能量）：$T_G(P,P) = \frac{x_0^2 + y_0^2 - r^2}{r^2}$
\end{itemize}
塞进去，两边同乘 $r^4$ 拔掉讨厌的分母，展开完全平方：
$$(x_0 X + y_0 Y)^2 - (x_0^2 + y_0^2 - r^2)(X^2 + Y^2) = 0$$
就像初中做因式分解一样，把左边拆开，你会发现 $x_0^2 X^2$ 和 $y_0^2 Y^2$ 竟然完美对消了！重新合并同类项，我们得到了极其清爽的切线对方向方程：
$$H(X,Y) = (r^2 - y_0^2)X^2 + 2x_0 y_0 XY + (r^2 - x_0^2)Y^2 = 0$$
现在看看椭圆在新坐标系下长啥样。椭圆方程是 $\frac{X^2}{a^2} + \frac{Y^2}{b^2} = 1$。因为 $P$ 在椭圆上，常数项被极化算子直接截断了，方程变成：
$$\underbrace{\frac{X^2}{a^2} + \frac{Y^2}{b^2}}_{\text{二次项 } E_E} + 2\underbrace{\left(\frac{x_0 X}{a^2} + \frac{y_0 Y}{b^2}\right)}_{\text{一次切线项 } T_E} = 0$$
设我们要找的割线 $AB$ 在新坐标系下的方程为 $L(X,Y) = uX + vY + 1 = 0$（也就是 $-(uX+vY)=1$）。把它乘到椭圆的一次项上，强行把椭圆“齐次化”：
$$\left(\frac{X^2}{a^2} + \frac{Y^2}{b^2}\right) - 2\left(\frac{x_0 X}{a^2} + \frac{y_0 Y}{b^2}\right)(uX + vY) = 0$$
这就代表了连接 $P$ 点与交点 $A,B$ 的两条射线。等等，这两条射线不就是第一步里求出来的那两条切线吗？！
既然代表同一个图形，那椭圆凑出来的式子，必须等于圆的切线系式子 $H(X,Y)$ 乘以某个缩放常数 $\mu$！咱们摆出这个绝对恒等式：$$\frac{X^2}{a^2} + \frac{Y^2}{b^2} - \mu H(X,Y) \equiv 2\left(\frac{x_0 X}{a^2} + \frac{y_0 Y}{b^2}\right)(uX + vY)$$咱们不需要解任何联立，只要盯着方程两边提取 $X^2$ 和 $Y^2$ 的系数：
\begin{itemize}
\item 看 $X^2$ 系数：$\frac{1}{a^2} - \mu(r^2 - y_0^2) = \frac{2x_0}{a^2}u \implies 2u = \frac{1 - \mu a^2(r^2 - y_0^2)}{x_0}$
\item 看 $Y^2$ 系数：$\frac{1}{b^2} - \mu(r^2 - x_0^2) = \frac{2y_0}{b^2}v \implies 2v = \frac{1 - \mu b^2(r^2 - x_0^2)}{y_0}$
\end{itemize}
因为在新坐标系里，割线 $uX + vY + 1 = 0$ 的斜率 $k_{AB}$ 依然是 $-\frac{u}{v}$。咱们把上面算出的 $u$ 和 $v$ 除一下：
$$k_{AB} = -\frac{u}{v} = -\frac{y_0}{x_0} \cdot \frac{1 - \mu a^2(r^2 - y_0^2)}{1 - \mu b^2(r^2 - x_0^2)}$$
怎么求 $\mu$？笔记里写了大招：直接找椭圆在 $P$ 点切线的方向向量 $\vec{t}$ 代进去！切线是 $\frac{x_0 X}{a^2} + \frac{y_0 Y}{b^2} = 0$，它的方向向量咱们随便取一个，比如 $\vec{t} = (-y_0 a^2, x_0 b^2)$。代入椭圆二次项 $E_E(\vec{t},\vec{t})$ 和圆切线二次项 $H(\vec{t})$，咱们直接写出 $\frac{1}{\mu}$ 的值：
$$\frac{1}{\mu} = \frac{H(\vec{t})}{E_E(\vec{t},\vec{t})} = \frac{r^2 y_0^2 a^2}{b^2} + \frac{r^2 x_0^2 b^2}{a^2} - a^2 b^2$$
接下来，咱们把刚才 $k_{AB}$ 公式里分子和分母的括号算一下（把 $1$ 换成 $\frac{1}{\mu}$ 乘进去）：算分子部分 $\frac{1}{\mu} - a^2(r^2 - y_0^2)$：代入 $\frac{1}{\mu}$ 展开：
$$= \frac{r^2 y_0^2 a^2}{b^2} + \frac{r^2 x_0^2 b^2}{a^2} - a^2 b^2 - a^2 r^2 + a^2 y_0^2$$
利用椭圆方程的变体 $a^2 y_0^2 = a^2 b^2 - b^2 x_0^2$ 把最后一项换掉：
$$= \frac{r^2 y_0^2 a^2}{b^2} + \frac{r^2 x_0^2 b^2}{a^2} - a^2 r^2 - b^2 x_0^2$$
再利用椭圆方程的另一个变体 $\frac{y_0^2}{b^2} = 1 - \frac{x_0^2}{a^2}$ 把第一项换掉：
$$= r^2 a^2\left(1 - \frac{x_0^2}{a^2}\right) + \frac{r^2 x_0^2 b^2}{a^2} - a^2 r^2 - b^2 x_0^2$$
展开后你会惊奇地发现，常数 $r^2 a^2$ 被完美抵消了！提出公因式 $-\frac{x_0^2}{a^2}$：
$$= -\frac{x_0^2}{a^2}(r^2 a^2 - r^2 b^2 + a^2 b^2)$$
因为 $c^2 = a^2 - b^2$，括号里直接坍缩成 $(a^2 b^2 + r^2 c^2)$。分子部分最终变成：$-\frac{x_0^2}{a^2}(a^2 b^2 + r^2 c^2)$。完全同理去算分母部分 $\frac{1}{\mu} - b^2(r^2 - x_0^2)$：经过一模一样的对称代换和对消，分母部分会完美坍缩成：$-\frac{y_0^2}{b^2}(a^2 b^2 - r^2 c^2)$。
现在，把化简完的分子和分母塞回斜率 $k_{AB}$ 的公式里：$$k_{AB} = -\frac{y_0}{x_0} \cdot \frac{-\frac{x_0^2}{a^2}(a^2 b^2 + r^2 c^2)}{-\frac{y_0^2}{b^2}(a^2 b^2 - r^2 c^2)} = -\frac{x_0 b^2}{y_0 a^2} \cdot \frac{a^2 b^2 + r^2 c^2}{a^2 b^2 - r^2 c^2}$$由于 $P(x_0, y_0)$ 和原点连线的斜率 $k_{OP} = \frac{y_0}{x_0}$，把它乘过去：$$k_{OP} \cdot k_{AB} = \frac{y_0}{x_0} \cdot \left( -\frac{x_0 b^2}{y_0 a^2} \cdot \frac{a^2 b^2 + r^2 c^2}{a^2 b^2 - r^2 c^2} \right)$$
则$x_0$ 和 $y_0$ 被彻底约掉。$$k_{OP} \cdot k_{AB} = -\frac{b^2}{a^2} \cdot \frac{a^2 b^2 + r^2 c^2}{a^2 b^2 - r^2 c^2}$$
\end{solution}

\begin{example}{}{}
已知P为椭圆$\frac{x^2}{16}+y^2=1$上一动点，过点P作圆$C:(x-2)^2+y^2=\frac{4}{9}$的
两条切线分别交椭圆于 A,B两点，延长 PE(E为圆C与x轴的左交点)交椭圆于Q点：
求证：$k_{OQ}\cdot k_{AB}=-\frac{1}{4}$

\begin{solution}
定点 $E$ 在圆 $C:(x-2)^2+y^2=4/9$ 的左边缘，显然 $y=0$ 时，$(x-2)^2=4/9 \implies x=4/3$。即 $E(4/3, 0)$。直线 $PE$ 穿过椭圆 $C_1: \frac{x^2}{16}+y^2=1$ 交于 $P, Q$ 两点。我们以 $E$ 为基点，方向向量设为 $\vec{v} = P - E = (x_0 - 4/3, y_0)$。直线上任意一点 $X = E + t\vec{v}$ 代入椭圆极化算子 $T_E(X,X)=0$，必然满足一元二次多项式：$$E_E(\vec{v},\vec{v})t^2 + 2E_E(E,\vec{v})t + T_E(E,E) = 0$$因为点 $P$ 就在这条直线上，且 $P = E + 1\cdot\vec{v}$，所以 $t_P = 1$ 必然是这个方程的一个根！直接利用常数项与二次项的关系（即两根之积 $t_P t_Q = t_Q$）：$$t_Q = \frac{T_E(E,E)}{E_E(\vec{v},\vec{v})}$$
因为 $P$ 在椭圆上，把 $t=1$ 代入方程必等于 0：$$E_E(\vec{v},\vec{v}) + 2E_E(E,\vec{v}) + T_E(E,E) = 0 \implies E_E(\vec{v},\vec{v}) = -T_E(E,E) - 2E_E(E,P) + 2E_E(E,E)$$实际上，利用内积算子的平移分配律 $E_E(P-E, P-E)$ 展开，并结合 $T_E(P,P)=0$，可以直接化简分母为 $-T_E(E,E) - 2T_E(P,E)$。我们只需计算 $E$ 点的极化值和交叉项：
\begin{itemize}
\item $T_E(E,E) = \frac{(4/3)^2}{16} + 0 - 1 = \frac{1}{9} - 1 = -\frac{8}{9}$
\item $E_E(E,P) = \frac{(4/3)x_0}{16} + 0 = \frac{x_0}{12}$
\end{itemize}
$$t_Q = \frac{T_E(E,E)}{T_E(E,E) + 2 - 2E_E(E,P)}$$
代入数值：$$t_Q = \frac{-8/9}{-8/9 + 2 - 2(x_0/12)} = \frac{-8/9}{10/9 - x_0/6} = \frac{-16}{20 - 3x_0}$$参数 $t_Q$ 一出，$Q$ 点坐标瞬间拿到（$Q = E + t_Q \vec{v}$）：$$y_Q = 0 + t_Q y_0 = \frac{-16y_0}{20 - 3x_0}$$
$$x_Q = \frac{4}{3} + t_Q\left(x_0 - \frac{4}{3}\right) = \frac{4(20-3x_0) - 48x_0 + 64}{3(20-3x_0)} = \frac{144 - 60x_0}{3(20-3x_0)} = \frac{48 - 20x_0}{20 - 3x_0}$$
所以斜率 $k_{OQ}$ 直接就是纵横坐标作商：$$k_{OQ} = \frac{y_Q}{x_Q} = \frac{-16y_0}{48 - 20x_0} = \frac{4y_0}{5x_0 - 12}$$
根据“四交点曲线系”恒等式，椭圆 $E_E(X,X)$ 减去缩放后的切线对 $H(X,X)$ 必定能被切线 $E_E(P,X)$ 整除：$$\mu = \frac{T_G(P,P) E_G(\vec{t},\vec{t}) - [E_G(P-C,\vec{t})]^2}{E_E(\vec{t},\vec{t})}$$其中 $\vec{t} = (16y_0, -x_0)$ 是椭圆在 $P$ 点切线的方向向量。根据笔记里的《算子拉格朗日恒等式》，对于圆 $G$（系数 $a=b=1$），分子前两项完美等价于外积算子 $W$ 的平方：$$T_G(P,P) E_G(\vec{t},\vec{t}) - [E_G(P-C,\vec{t})]^2 = [W(P-C, \vec{t})]^2 - r^2 E_G(\vec{t},\vec{t})$$来感受一下纯多项式魔法：
\begin{itemize}
\item 外积算子：$W(P-C, \vec{t}) = (x_0-2)(-x_0) - y_0(16y_0) = -x_0^2 + 2x_0 - 16y_0^2$。因为 $x_0^2+16y_0^2=16$，瞬间化为 $2x_0 - 16$！其平方就是 $4(x_0 - 8)^2$。
\item 内积算子：$E_G(\vec{t},\vec{t}) = (16y_0)^2 + (-x_0)^2 = 256(1 - x_0^2/16) + x_0^2 = 256 - 15x_0^2$。
\item 分母：$E_E(\vec{t},\vec{t}) = \frac{(16y_0)^2}{16} + (-x_0)^2 = 16y_0^2 + x_0^2 = 16$。
\end{itemize}
把这些极度精简的块拼起来，$\mu$ 一秒算出：$$\mu = \frac{4(x_0 - 8)^2 - \frac{4}{9}(256 - 15x_0^2)}{16} = \frac{2}{9}(3x_0^2 - 18x_0 + 40)$$

\begin{itemize}
\item 算 $C_Y$：$$C_Y = \frac{4}{9} - (x_0-2)^2 + \mu = \frac{4}{9} - x_0^2 + 4x_0 - 4 + \frac{2}{3}x_0^2 - 4x_0 + \frac{80}{9}$$常数项 $4/9 - 4 + 80/9 = 16/3$；一次项 $4x_0$ 完美对消！二次项剩下 $-\frac{1}{3}x_0^2$。$$C_Y = \frac{16 - x_0^2}{3} = \frac{16y_0^2}{3}$$（连 $x_0$ 都没了，只剩一个干净的 $y_0^2$！）
\item 算 $C_X$：$$C_X = \frac{4}{9} - y_0^2 + \frac{\mu}{16} = -\frac{5}{9} + \frac{x_0^2}{16} + \frac{1}{72}(3x_0^2 - 18x_0 + 40)$$常数项 $-5/9 + 40/72 = 0$ 完美归零！$$C_X = \frac{x_0^2}{16} + \frac{1}{24}x_0^2 - \frac{1}{4}x_0 = \frac{5}{48}x_0^2 - \frac{1}{4}x_0 = \frac{x_0}{48}(5x_0 - 12)$$
\end{itemize}
把这两个极其优美的系数除一下：$$\frac{C_X}{C_Y} = \frac{\frac{x_0}{48}(5x_0 - 12)}{\frac{16y_0^2}{3}} = \frac{x_0(5x_0 - 12)}{16 \cdot 16 y_0^2}$$将其代入最开始的等式 $\frac{x_0}{16 y_0} (-k_{AB}) = \frac{C_X}{C_Y}$，两边把 $x_0$ 和 $16y_0$ 全部约掉：$$k_{AB} = -\frac{1}{16} \frac{5x_0 - 12}{y_0}$$
把纯算子导出的两个结果相乘：$$k_{OQ} \cdot k_{AB} = \left( \frac{4y_0}{5x_0 - 12} \right) \cdot \left( -\frac{1}{16} \frac{5x_0 - 12}{y_0} \right) = -\frac{4}{16} = -\frac{1}{4}$$
\end{solution}

\begin{solution}
补齐最后两块算子拼图
\begin{itemize}
\item 非对称的【共轭扭转算子 $M^*$】:讲义中提到共轭直径的斜率积是 $-\frac{b^2}{a^2}$。但本题给出了一个极其罕见的非对称条件 $k_{OM} \cdot k_{AB} = -\frac{b}{a}$。在算子视界中，任何斜率乘积都可以化为向量的内积！我们定义动点 $M(x_M, y_M)$ 的共轭扭转点为 $M^* = (\frac{a}{b}x_M, \frac{b}{a}y_M)$。根据斜率的算子展开，这个非对称条件瞬间等价于一个极度纯粹的算子正交关系：$$E(M^*, \vec{u}_{AB}) = 0$$几何意义： 这意味着扭转点 $M^*$ 所在的直线，与弦 $AB$ 在椭圆内积算子 $E$ 下严格正交！由于弦 $AB$ 的极点设为 $Q$，根据极性，$OQ$ 必定与 $AB$ 也是 $E$-正交的。因此，$M^*, O, Q$ 三点严格共线。这建立了一条从极点 $Q$ 到动点 $M$ 的绝对线性映射！
\item 基于“三交一切系”的【彭赛列曲线系恒等式】:我们要彻底避开联立圆和椭圆！动点 $P$ 引出的两条切线 $PA, PB$ 本质上是一个“退化的二次曲线（相交直线对）”。我们引入圆 $G$ 的算子 $T_G(X,X) = (x-m)^2 + y^2 - r^2$。切线对 $PA \cup PB$ 的联合算子方程为：$$[T_G(P,X)]^2 - T_G(P,P) \cdot T_G(X,X) = 0$$根据你讲义第六章的四大经典交点曲线系，这对切线与椭圆 $T_E(X,X)=0$ 的交点为 $P$（双重交点，即在此相切）以及 $A, B$。因此，我们可以直接写出包含椭圆、切线对、切线 $P$ 以及弦 $AB$ 的终极算子恒等式（三交一切系）：$$[T_G(P,X)]^2 - T_G(P,P)T_G(X,X) + \lambda T_E(X,X) \equiv \mu \cdot T_E(P,X) \cdot T_E(Q,X)$$(其中 $T_E(P,X)=0$ 是椭圆在 P 点的切线，$T_E(Q,X)=0$ 是弦 AB 的方程)几何意义： 这是一个关于任意动点 $X$ 绝对成立的多项式恒等式！通过比对一次项系数，极点 $Q$ 的坐标与点 $P$ 的坐标被完全线性死锁（仿射映射），没有任何根号和二次方程！
\end{itemize}
内切圆 $G(m,r)$ 内接于对称三角形 $A_4EE'$。此时顶点 $A_4$ 就是椭圆左顶点 $(-a,0)$。利用内切圆的极点性质（或基础角平分线），直接得到该圆半径与椭圆常量的绑定关系方程：$c^2 r^2 + 2ab^2 r - b^2(a^2-m^2) = 0$。解出纯常量：$$r = \frac{b(\sqrt{a^4-m^2c^2} - ab)}{c^2}$$
\begin{itemize}
\item 【扩展二】证明了：极点 $Q$ 与动点 $P$ 是完全的线性仿射映射。
\item 【扩展一】证明了：扭转点 $M^*$ 与极点 $Q$ 共线，导致 $M$ 与 $P$ 的纵坐标严格成正比。
\item 把这种线性关系代入外积算子共线条件 $W(PM, PL) = 0$ 中，分子分母的变量 $y_P$ 会被完美同构对消！
\end{itemize}
既然算子理论已经从代数结构上绝对证明了：截距 $x_L$ 是一个不依赖于 $P$ 位置的拓扑不变量（即定点确实存在）。
那么，根据算子的主轴截断原理，我们根本不需要展开那坨巨大的多项式去求它！我们只需要把动点 $P$ 移动到算子的对称主轴（上顶点 $P(0,b)$）上，提取这个常量即可！当 $P$ 在 $(0,b)$ 时：
\begin{itemize}
\item 切线 $PA, PB$ 的斜率通过圆切算子瞬间退化：$k_{AB} = \frac{m}{r}$。
\item 代入非对称扭转条件：$k_{OM} \cdot (\frac{m}{r}) = -\frac{b}{a} \implies k_{OM} = -\frac{br}{am}$。
\item 点 $M$ 在椭圆 $E(X,X)=1$ 上，将此斜率比例代入内积算子，直接坍缩出 $M$ 坐标：$$M = \left( \frac{am}{\sqrt{m^2+r^2}}, \frac{-br}{\sqrt{m^2+r^2}} \right)$$
\item 利用外积算子（或三点共线斜率）计算直线 $PM$ 与 $x$ 轴的交点 $L(x_0, 0)$：$$\frac{b-0}{0-x_0} = \frac{y_M-b}{x_M-0} \implies x_0 = \frac{bx_M}{b-y_M}$$把 $M$ 的坐标代进去，分子分母的 $b$ 瞬间约掉，得到了宇宙级纯净的定点坐标：$$L \left( \frac{am}{\sqrt{m^2+r^2} + r}, 0 \right)$$
\end{itemize}
\end{solution}

已知 $P$ 为椭圆 $C_1: \frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$ 上一动点，$E(m,0), E'(-m,0)$ 为 $x$ 轴上的两定点，延长 $PE, PE'$ 分别交椭圆 $C_1$ 于 $A, B$ 两点；\\
(1) 求证：$k_{OP} \cdot k_{AB}$ 为定值，并求出该定值；\\
(2) 求证：直线 $AB$ 与一定椭圆相切，并求出该椭圆的标准方程;

\begin{solution}
动点 $X$ 在椭圆上，满足 $E(X,X) = 1$。我们引入隐式平移向量 $\vec{v} = X - P$（即动点相对于基点的偏移量）。利用内积算子 $E$ 的双线性分配律，直接将 $X$ 替换为 $P + \vec{v}$：$$E(P+\vec{v}, P+\vec{v}) = E(P,P) + 2E(P, \vec{v}) + E(\vec{v}, \vec{v}) = 1$$
因为 $P$ 也在椭圆上，自身极化能量 $E(P,P) = 1$。方程瞬间截断为纯粹的向量形式：
$$E(\vec{v}, \vec{v}) + 2E(P, \vec{v}) = 0$$
设割线 $AB$ 在向量 $\vec{v}$ 视角下的线性方程为 $L(\vec{v}) + 1 = 0$。将其作为齐次化因子乘给一次项 $2E(P, \vec{v})$：
$$E(\vec{v}, \vec{v}) - 2E(P, \vec{v}) \cdot L(\vec{v}) = 0$$
另一方面，由定点 $E(m,0)$ 和 $E'(-m,0)$ 确定的斜率和 $S = \frac{2x_P y_P}{x_P^2-m^2}$ 与斜率积 $K = \frac{y_P^2}{x_P^2-m^2}$，直接构成斜率纯二次型 $H(\vec{v}) = v_y^2 - S v_x v_y + K v_x^2$。于是，我们得到了隐式翻译下的终极算子恒等式（对任意探测向量 $\vec{v}$ 均成立）：
$$E(\vec{v}, \vec{v}) - \mu H(\vec{v}) \equiv 2E(P, \vec{v}) \cdot L(\vec{v})$$
我们要干掉右边那坨未知的 $L(\vec{v})$，只需找一个能让切线算子 $E(P, \vec{v})$ 归零的切向向量 $\vec{t}$。因为 $E(P, \vec{v}) = \frac{x_P v_x}{a^2} + \frac{y_P v_y}{b^2}$，我们直接取其正交方向：$\vec{t} = (a^2 y_P, -b^2 x_P)$。将 $\vec{t}$ 代入恒等式，右侧瞬间湮灭：$$\mu = \frac{E(\vec{t}, \vec{t})}{H(\vec{t})}$$
\begin{itemize}
\item 分子计算（极度舒适）： \[E(\vec{t}, \vec{t}) = \frac{(a^2 y_P)^2}{a^2} + \frac{(-b^2 x_P)^2}{b^2} = a^2 y_P^2 + b^2 x_P^2 = a^2 b^2 \left( \frac{y_P^2}{b^2} + \frac{x_P^2}{a^2} \right) = a^2 b^2 E(P,P) = {a^2 b^2}\]
\item 分母计算： \[H(\vec{t}) = (-b^2 x_P)^2 - S(a^2 y_P)(-b^2 x_P) + K(a^2 y_P)^2 = b^4 x_P^2 + S(a^2 b^2 x_P y_P) + K(a^4 y_P^2)\]代入 $S, K$ 稍加提取（利用 $a^2 y_P^2 = b^2(a^2 - x_P^2)$），完美对消后直接得到：$H(\vec{t}) ={b^4 \frac{a^4 - m^2 x_P^2}{x_P^2 - m^2}}$
\item $\mu$ 瞬间出炉：$$\mu = \frac{a^2(x_P^2 - m^2)}{b^2(a^4 - m^2 x_P^2)}$$
\end{itemize}
设 $L(\vec{v}) = u \cdot v_x + v \cdot v_y$。怎么求 $u, v$？继续用探针！
\begin{itemize}
\item 代入横向基底 $\vec{e_1} = (1,0)$：\[E(\vec{e_1}, \vec{e_1}) - \mu H(\vec{e_1}) = 2E(P, \vec{e_1}) L(\vec{e_1}) \implies \frac{1}{a^2} - \mu K = 2\left(\frac{x_P}{a^2}\right) u \implies u = \frac{1 - a^2 \mu K}{2x_P}\]
\item 代入纵向基底 $\vec{e_2} = (0,1)$：\[E(\vec{e_2}, \vec{e_2}) - \mu H(\vec{e_2}) = 2E(P, \vec{e_2}) L(\vec{e_2}) \implies \frac{1}{b^2} - \mu \cdot 1 = 2\left(\frac{y_P}{b^2}\right) v \implies v = \frac{1 - b^2 \mu}{2y_P}\]
\end{itemize}
不需要解任何方程组，算子输入什么，就直接吐出什么！代入 $\mu$ 后，直线系数 $u, v$ 与之前的结果分毫不差：$$u = \frac{(a^2 - m^2)x_P}{2(a^4 - m^2 x_P^2)}, \quad v = \frac{a^2(a^2 + m^2)y_P}{2b^2(a^4 - m^2 x_P^2)}$$
解决第一问：弦 $AB$ 的斜率就是隐式线性泛函 $L$ 的负系数比：$k_{AB} = -\frac{u}{v}$。$$k_{OP} \cdot k_{AB} = \frac{y_P}{x_P} \cdot \left( -\frac{(a^2 - m^2) x_P b^2}{a^2(a^2 + m^2) y_P} \right) = -\frac{b^2(a^2 - m^2)}{a^2(a^2 + m^2)}$$解决第二问：直线全局方程为 $u(x-x_P) + v(y-y_P) + 1 = 0$。展开为截距式 $Ax+By=1$：$$A = -\frac{x_P}{a^2}, \quad B = -\left(\frac{a^2+m^2}{a^2-m^2}\right)\frac{y_P}{b^2}$$将其代入相切充要条件 $\alpha^2 A^2 + \beta^2 B^2 = 1$，并利用系统被动技能 $\frac{x_P^2}{a^2} + \frac{y_P^2}{b^2} = 1$ 同构比对，直接生成相切椭圆：$$\frac{x^2}{a^2} + \frac{y^2}{b^2 \left( \frac{a^2-m^2}{a^2+m^2} \right)^2} = 1$$
\end{solution}

已知 $P$ 为椭圆 $C_1: \frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$ 上一动点， $E(m,0), E'(-m,0)$ 为 $x$ 轴上的两定点，延长 $PE, PE'$ 分别交椭圆 $C_1$ 于 $A,B$ 两点， $A,B$ 两点处切线相交于点 $Q$\\
(1)求证： $\frac{k_{O P}}{k_{O Q}}$ 为定值，并求出该定值\\
(2)求证：点 $Q$ 的轨迹为椭圆，并求出该椭圆的标准方程
\end{example}
\begin{solution}
设动点 $P(x_0, y_0)$ 在椭圆上，满足 $E(P,P) = 1$。引入探测向量 $\vec{v} = (X, Y) = (x-x_0, y-y_0)$，将椭圆方程在 $P$ 点隐式展开：
$$E(P+\vec{v}, P+\vec{v}) = 1 \implies E(P,P) + 2E(P, \vec{v}) + E(\vec{v}, \vec{v}) = 1$$
截断得：$E(\vec{v}, \vec{v}) + 2E(P, \vec{v}) = 0$。设割线 $AB$ 在向量 $\vec{v}$ 视角的局部线性方程为 $L(\vec{v}) + 1 = 0$。代入构造齐次方程（代表 $PA \cup PB$）：
$$E(\vec{v}, \vec{v}) - 2E(P, \vec{v})L(\vec{v}) = 0$$
定点 $E(m,0)$ 和 $E'(-m,0)$ 对应的局部向量为 $(m-x_0, -y_0)$ 和 $(-m-x_0, -y_0)$。弦 $PA, PB$ 的斜率分别为：$$k_1 = \frac{-y_0}{m-x_0} = \frac{y_0}{x_0-m}, \quad k_2 = \frac{-y_0}{-m-x_0} = \frac{y_0}{x_0+m}$$
计算斜率之和 $S$ 与积 $K$：
$$K = k_1 k_2 = \frac{y_0^2}{x_0^2-m^2}$$
$$S = k_1 + k_2 = \frac{y_0(x_0+m) + y_0(x_0-m)}{x_0^2-m^2} = \frac{2x_0 y_0}{x_0^2-m^2}$$
构造斜率二次型 $H(\vec{v}) = Y^2 - S \cdot XY + K \cdot X^2$。它与上方的齐次方程必定只差一个常数倍 $\mu$：$$E(\vec{v}, \vec{v}) - \mu H(\vec{v}) \equiv 2E(P, \vec{v}) \cdot L(\vec{v})$$
令 $\vec{t} = (a^2 y_0, -b^2 x_0)$，使得 $E(P, \vec{t}) = 0$。代入恒等式得 $\mu = \frac{E(\vec{t}, \vec{t})}{H(\vec{t})}$。
分子：
$$E(\vec{t}, \vec{t}) = \frac{(a^2 y_0)^2}{a^2} + \frac{(-b^2 x_0)^2}{b^2} = a^2 y_0^2 + b^2 x_0^2 = a^2 b^2 E(P,P) = a^2 b^2$$
分母：$$H(\vec{t}) = (-b^2 x_0)^2 - S(a^2 y_0)(-b^2 x_0) + K(a^2 y_0)^2= b^4 x_0^2 + S a^2 b^2 x_0 y_0 + K a^4 y_0^2$$
代入 $S, K$：
$$H(\vec{t}) = b^4 x_0^2 + \frac{2x_0 y_0}{x_0^2-m^2} a^2 b^2 x_0 y_0 + \frac{y_0^2}{x_0^2-m^2} a^4 y_0^2 = b^4 x_0^2 + \frac{2a^2 b^2 x_0^2 y_0^2 + a^4 y_0^4}{x_0^2-m^2}$$
提取公因式并用 $a^2 y_0^2 = b^2(a^2-x_0^2)$ 替换：
\begin{align*}
H(\vec{t}) &= b^4 x_0^2 + \frac{a^2 y_0^2 (2b^2 x_0^2 + a^2 y_0^2)}{x_0^2-m^2} = b^4 x_0^2 + \frac{b^2(a^2-x_0^2)[2b^2 x_0^2 + b^2(a^2-x_0^2)]}{x_0^2-m^2}\\
&= b^4 x_0^2 + \frac{b^4(a^2-x_0^2)(a^2+x_0^2)}{x_0^2-m^2} = b^4 x_0^2 + \frac{b^4(a^4-x_0^4)}{x_0^2-m^2}\\
&=\frac{b^4 x_0^2(x_0^2-m^2) + b^4(a^4-x_0^4)}{x_0^2-m^2} = \frac{b^4 x_0^4 - b^4 m^2 x_0^2 + b^4 a^4 - b^4 x_0^4}{x_0^2-m^2} = \frac{b^4(a^4 - m^2 x_0^2)}{x_0^2-m^2}
\end{align*}
两式相除得 $\mu$：$$\mu = \frac{a^2 b^2}{\frac{b^4(a^4 - m^2 x_0^2)}{x_0^2-m^2}} = \frac{a^2(x_0^2 - m^2)}{b^2(a^4 - m^2 x_0^2)}$$
基底探针萃取割线系数 $u, v$
设 $L(\vec{v}) = uX + vY$。代入 $\vec{e_1} = (1,0)$：
\begin{align*}
\frac{1}{a^2} - \mu K &= 2\left(\frac{x_0}{a^2}\right)u \implies 2x_0 u = 1 - a^2 \mu \frac{y_0^2}{x_0^2-m^2} = 1 - a^2 \frac{a^2(x_0^2-m^2)}{b^2(a^4-m^2 x_0^2)} \frac{y_0^2}{x_0^2-m^2}\\
&=1 - \frac{a^4 y_0^2}{b^2(a^4-m^2 x_0^2)} = 1 - \frac{a^2(a^2-x_0^2)}{a^4-m^2 x_0^2} = \frac{a^4-m^2 x_0^2 - a^4 + a^2 x_0^2}{a^4-m^2 x_0^2} = \frac{(a^2-m^2)x_0^2}{a^4-m^2 x_0^2}
\end{align*}
所以 $u = \frac{(a^2-m^2)x_0}{2(a^4-m^2 x_0^2)}$。代入 $\vec{e_2} = (0,1)$：
\begin{align*}
\frac{1}{b^2} - \mu &= 2\left(\frac{y_0}{b^2}\right)v \implies 2y_0 v = 1 - b^2 \mu = 1 - \frac{a^2(x_0^2-m^2)}{a^4-m^2 x_0^2}\\
&= \frac{a^4-m^2 x_0^2 - a^2 x_0^2 + a^2 m^2}{a^4-m^2 x_0^2} = \frac{a^2(a^2+m^2) - x_0^2(a^2+m^2)}{a^4-m^2 x_0^2} = \frac{(a^2+m^2)(a^2-x_0^2)}{a^4-m^2 x_0^2}
\end{align*}
替换 $a^2-x_0^2 = \frac{a^2}{b^2}y_0^2$：$$2y_0 v = \frac{a^2(a^2+m^2)y_0^2}{b^2(a^4-m^2 x_0^2)} \implies v = \frac{a^2(a^2+m^2)y_0}{2b^2(a^4-m^2 x_0^2)}$$
将 $u(x-x_0) + v(y-y_0) + 1 = 0$ 展开为 $ux + vy + (1 - ux_0 - vy_0) = 0$。计算常数项：
\begin{align*}
u x_0 + v y_0 &= \frac{(a^2-m^2)x_0^2}{2(a^4-m^2 x_0^2)} + \frac{a^2(a^2+m^2)y_0^2}{2b^2(a^4-m^2 x_0^2)} = \frac{a^2 x_0^2 - m^2 x_0^2 + (a^2+m^2)(a^2-x_0^2)}{2(a^4-m^2 x_0^2)}\\
&= \frac{a^2 x_0^2 - m^2 x_0^2 + a^4 - a^2 x_0^2 + a^2 m^2 - m^2 x_0^2}{2(a^4-m^2 x_0^2)} = \frac{a^4 + a^2 m^2 - 2m^2 x_0^2}{2(a^4-m^2 x_0^2)}\\
1 - (u x_0 + v y_0) &= \frac{2a^4 - 2m^2 x_0^2 - a^4 - a^2 m^2 + 2m^2 x_0^2}{2(a^4-m^2 x_0^2)} = \frac{a^4 - a^2 m^2}{2(a^4-m^2 x_0^2)} = \frac{a^2(a^2-m^2)}{2(a^4-m^2 x_0^2)}
\end{align*}
所以方程为 $ux + vy + \frac{a^2(a^2-m^2)}{2(a^4-m^2 x_0^2)} = 0$。两边同除以负的常数项，化为截距式 $Ax + By = 1$：
\begin{align*}
A &= -u \cdot \frac{2(a^4-m^2 x_0^2)}{a^2(a^2-m^2)} = -\frac{(a^2-m^2)x_0}{2(a^4-m^2 x_0^2)} \cdot \frac{2(a^4-m^2 x_0^2)}{a^2(a^2-m^2)} = -\frac{x_0}{a^2}\\
B &= -v \cdot \frac{2(a^4-m^2 x_0^2)}{a^2(a^2-m^2)} = -\frac{a^2(a^2+m^2)y_0}{2b^2(a^4-m^2 x_0^2)} \cdot \frac{2(a^4-m^2 x_0^2)}{a^2(a^2-m^2)} = -\frac{a^2+m^2}{a^2-m^2} \frac{y_0}{b^2}
\end{align*}
割线 $AB$ 绝对解析式诞生：$$-\frac{x_0}{a^2}x - \frac{a^2+m^2}{a^2-m^2}\frac{y_0}{b^2}y = 1$$
点 $Q(x_Q, y_Q)$ 是弦 $A, B$ 的两条切线的交点。根据讲义中算子代数的核心结论（Pole-Polar Duality）：经过两切点 $A, B$ 的割线，正是交点 $Q$ 对应的极线！极线的内积算子方程放之四海而皆准：$$E(Q, X) = 1 \implies \frac{x_Q}{a^2}x + \frac{y_Q}{b^2}y = 1$$由于极线方程具有唯一性，我们直接将它与刚求出的弦 $AB$ 的系数一一比对：$$\frac{x_Q}{a^2} = -\frac{x_0}{a^2} \implies x_Q = -x_0,~\frac{y_Q}{b^2} = -\frac{a^2+m^2}{a^2-m^2}\frac{y_0}{b^2} \implies y_Q = -\frac{a^2+m^2}{a^2-m^2}y_0$$
点 $Q$ 的坐标被算子结构完全锁死！没有任何联立求解！
\begin{itemize}
\item 求定值：$P$ 点斜率 $k_{OP} = \frac{y_0}{x_0}$。$Q$ 点斜率 $k_{OQ} = \frac{y_Q}{x_Q} = \frac{-\frac{a^2+m^2}{a^2-m^2}y_0}{-x_0} = \frac{a^2+m^2}{a^2-m^2} \cdot \frac{y_0}{x_0}$。直接相除：$$\frac{k_{OP}}{k_{OQ}} = \frac{\frac{y_0}{x_0}}{\frac{a^2+m^2}{a^2-m^2} \frac{y_0}{x_0}} = \frac{a^2-m^2}{a^2+m^2}$$第一问得证，此比值为与动点位置完全无关的绝对常数 $\frac{a^2-m^2}{a^2+m^2}$。
\item 求轨迹：把 $Q$ 点的坐标反解出 $x_0, y_0$：$$x_0 = -x_Q, \quad y_0 = -\frac{a^2-m^2}{a^2+m^2} y_Q$$因为点 $P(x_0, y_0)$ 是活生生在原椭圆 $C_1$ 上的，自带防身被动：$\frac{x_0^2}{a^2} + \frac{y_0^2}{b^2} = 1$。把 $x_0, y_0$ 的反解式怼进去：$$\frac{(-x_Q)^2}{a^2} + \frac{\left( -\frac{a^2-m^2}{a^2+m^2} y_Q \right)^2}{b^2} = 1$$整理平方项：$$\frac{x_Q^2}{a^2} + \frac{y_Q^2}{b^2 \left( \frac{a^2+m^2}{a^2-m^2} \right)^2} = 1$$第二问得证。点 $Q$ 完美贴合于一个标准的定椭圆轨道上，其标准方程即为上式。
\end{itemize}
\end{solution}

\begin{example}{}{}
已知$P$为椭圆$C_1:\frac{x^2}{a_1^2}+\frac{y^2}{b_3^2}=1$上一动点，过点$P$作椭圆$C_{2}:\frac{x^2}{a_3^2}+\frac{y^2}{b_3^2}=1$的两切线分别交椭圆$C_1$于$A,B$ 两点；\\
(1)求证：$k_{OP}\cdot k_{AB}$为定值，并求出该定值；\\
(2)求证：直线$AB$ 与一定椭圆相切，并求出该椭圆的标准方程；
\end{example}
\begin{solution}
我们定义外层椭圆 $C_1$ 和内层椭圆 $C_2$ 的内积算子：
$$E_1(X,Y) = \frac{x_X x_Y}{a_1^2} + \frac{y_X y_Y}{b_3^2},~~ E_2(X,Y) = \frac{x_X x_Y}{a_3^2} + \frac{y_X y_Y}{b_3^2}$$
由于动点 $P(x_0, y_0)$ 在外椭圆 $C_1$ 上，自带绝对被动技能：
$$E_1(P,P) = \frac{x_0^2}{a_1^2} + \frac{y_0^2}{b_3^2} = 1$$
根据我们之前推导过的结论，从点 $P$ 向 $C_2$ 引出的两条切线 $PA, PB$，它们构成的“切线对联合方程”为：
$$[E_2(P,X) - 1]^2 = [E_2(P,P) - 1][E_2(X,X) - 1]$$
展开并整理，得到这对切线的纯多项式：
$$[E_2(P,X)]^2 - [E_2(P,P) - 1]E_2(X,X) - 2E_2(P,X) + E_2(P,P) = 0$$
极度核心的逻辑来了：这对切线与外层椭圆 $C_1$（方程为 $E_1(X,X) - 1 = 0$）有且仅有三个交点：切线发源点 $P$（双重交点，因为两条线都过 $P$），以及弦的端点 $A, B$。既然这对切线刚好穿过这几个点，那么在代数上，它的方程必然可以由外椭圆 $C_1$ 和经过这些点的直线“拼凑”而成！经过 $P$ 点（双重交点）的直线是 $C_1$ 在 $P$ 处的切线：$E_1(P,X) - 1 = 0$。经过 $A, B$ 的直线就是我们要找的割线：设其方程为 $ux + vy - w = 0$。所以，必定存在一个缩放常数 $\mu$，使得下面的多项式恒等式对空间中任意一点 $X$ 绝对成立：
$$[E_2(P,X)]^2 - [E_2(P,P) - 1]E_2(X,X) - 2E_2(P,X) + E_2(P,P) \equiv \mu [E_1(X,X) - 1] + [E_1(P,X) - 1](ux + vy - w)$$
我们将恒等式右边像初中乘法一样强行展开：
右边 $= \mu E_1(X,X) - \mu + E_1(P,X)(ux + vy) - w E_1(P,X) - (ux + vy) + w$
按照二次项、一次项、常数项进行归类，与等式左边进行比对：
\begin{itemize}
\item 比对常数项：左边常数 $= E_2(P,P)$，右边常数 $= w - \mu$$$\implies w = \mu + E_2(P,P)$$(记住这个式子，只要算出 $\mu$，常数 $w$ 就出来了)
\item 比对一次项：左边 $= -2E_2(P,X)$，右边 $= -w E_1(P,X) - (ux + vy)$把负号全去掉，直接解出我们要的割线表达式：$$ux + vy = 2E_2(P,X) - w E_1(P,X)$$
\item 比对二次项：左边 $= [E_2(P,X)]^2 - [E_2(P,P) - 1]E_2(X,X)$右边 $= \mu E_1(X,X) + E_1(P,X)(ux + vy)$把刚才算出来的 $ux + vy$ 代入右边，得到最终的纯二次恒等式：$$[E_2(P,X)]^2 - [E_2(P,P) - 1]E_2(X,X) \equiv \mu E_1(X,X) + 2E_1(P,X)E_2(P,X) - w[E_1(P,X)]^2$$
\end{itemize}
为了解出唯一的未知数 $\mu$，找一个特殊的向量 $\vec{t}$ 塞进恒等式，让那些复杂的项瞬间灰飞烟灭。我们构造探针向量 $\vec{t} = (a_1^2 y_0, -b_3^2 x_0)$。它与 $C_1$ 的切线绝对正交：
\[E_1(P, \vec{t}) = \frac{x_0(a_1^2 y_0)}{a_1^2} + \frac{y_0(-b_3^2 x_0)}{b_3^2} = x_0 y_0 - x_0 y_0 = \mathbf{0}\] 
把 $\vec{t}$ 塞进刚才的纯二次恒等式，右边的后两项因为含有 $E_1(P,\vec{t})$ 直接变成 0！恒等式瞬间坍缩：$$[E_2(P,\vec{t})]^2 - [E_2(P,P) - 1]E_2(\vec{t},\vec{t}) = \mu E_1(\vec{t},\vec{t})$$接下来，我们咬紧牙关，不省任何步骤，把这三个零件算出来：
\begin{itemize}
\item $\displaystyle E_1(\vec{t},\vec{t}) = \frac{(a_1^2 y_0)^2}{a_1^2} + \frac{(-b_3^2 x_0)^2}{b_3^2} = a_1^2 y_0^2 + b_3^2 x_0^2 = a_1^2 b_3^2 \left(\frac{y_0^2}{b_3^2} + \frac{x_0^2}{a_1^2}\right) = {a_1^2 b_3^2}$
\item $\displaystyle E_2(P,\vec{t}) = \frac{x_0(a_1^2 y_0)}{a_3^2} + \frac{y_0(-b_3^2 x_0)}{b_3^2} = x_0 y_0 \left(\frac{a_1^2}{a_3^2} - 1\right) = \mathbf{x_0 y_0 \left(\frac{a_1^2 - a_3^2}{a_3^2}\right)}$
\item $\displaystyle E_2(\vec{t},\vec{t}) = \frac{(a_1^2 y_0)^2}{a_3^2} + \frac{(-b_3^2 x_0)^2}{b_3^2} = \mathbf{\frac{a_1^4 y_0^2}{a_3^2} + b_3^2 x_0^2}$
\item $[E_2(P,P) - 1]$。注意，因为 $\frac{y_0^2}{b_3^2} = 1 - \frac{x_0^2}{a_1^2}$，代入得：
\[E_2(P,P) - 1 = \left(\frac{x_0^2}{a_3^2} + \frac{y_0^2}{b_3^2}\right) - 1 = \frac{x_0^2}{a_3^2} + \left(1 - \frac{x_0^2}{a_1^2}\right) - 1 = x_0^2 \left(\frac{1}{a_3^2} - \frac{1}{a_1^2}\right) = {x_0^2 \frac{a_1^2 - a_3^2}{a_1^2 a_3^2}}\]
\end{itemize}
把这四个零件塞回坍缩的恒等式，计算左边：
\begin{align*}
LHS &=\left[ x_0 y_0 \frac{a_1^2 - a_3^2}{a_3^2} \right]^2 - \left[ x_0^2 \frac{a_1^2 - a_3^2}{a_1^2 a_3^2} \right] \left( \frac{a_1^4 y_0^2}{a_3^2} + b_3^2 x_0^2 \right)\\
&= x_0^2 \frac{a_1^2 - a_3^2}{a_3^2} \left[ y_0^2 \frac{a_1^2 - a_3^2}{a_3^2} - \frac{1}{a_1^2} \left( \frac{a_1^4 y_0^2}{a_3^2} + b_3^2 x_0^2 \right) \right]\\
&= x_0^2 \frac{a_1^2 - a_3^2}{a_3^2}\left[ \frac{a_1^2 y_0^2}{a_3^2} - y_0^2 - \frac{a_1^2 y_0^2}{a_3^2} - \frac{b_3^2 x_0^2}{a_1^2} \right]=x_0^2 \frac{a_1^2 - a_3^2}{a_3^2}\left[ - y_0^2 - \frac{b_3^2 x_0^2}{a_1^2}\right]\\
&= x_0^2 \frac{a_1^2 - a_3^2}{a_3^2}(-b_3^2)\left[\frac{y_0^2}{b_3^2} + \frac{x_0^2}{a_1^2} \right]=- x_0^2 b_3^2 \frac{a_1^2 - a_3^2}{a_3^2}
\end{align*}
右边是 $\mu (a_1^2 b_3^2)$。两边一约：
$$\mu = - \frac{x_0^2}{a_1^2} \cdot \frac{a_1^2 - a_3^2}{a_3^2}$$
趁热打铁，算截距常数 $w = \mu + E_2(P,P)$：
\begin{align*}
w &= - x_0^2 \frac{a_1^2 - a_3^2}{a_1^2 a_3^2} + \left(\frac{x_0^2}{a_3^2} + \frac{y_0^2}{b_3^2}\right)= x_0^2 \left( - \frac{1}{a_3^2} + \frac{1}{a_1^2} \right) + \frac{x_0^2}{a_3^2} + 1 - \frac{x_0^2}{a_1^2}\\
&= x_0^2 \left( -\frac{1}{a_3^2} + \frac{1}{a_1^2} + \frac{1}{a_3^2} - \frac{1}{a_1^2} \right) + 1 = \mathbf{1}
\end{align*}
既然 $w=1$，我们在第二步中得到的割线 $AB$ 表达式瞬间定型：
\[ux + vy = 2E_2(P,X) - 1 \cdot E_1(P,X)= 2\left(\frac{x_0 x}{a_3^2} + \frac{y_0 y}{b_3^2}\right) - \left(\frac{x_0 x}{a_1^2} + \frac{y_0 y}{b_3^2}\right)\]
按 $x, y$ 合并同类项：$$\left( \frac{2}{a_3^2} - \frac{1}{a_1^2} \right) x_0 x + \frac{y_0}{b_3^2} y = 1$$我们将割线 $AB$ 写成标准截距式 $Ax + By = 1$，立刻读出：$$A = \left( \frac{2}{a_3^2} - \frac{1}{a_1^2} \right) x_0, \quad B = \frac{y_0}{b_3^2}$$
\begin{itemize}
\item 求证斜率乘积为定值:直线 $AB$ 的斜率 $k_{AB} = -\frac{A}{B} = -\frac{\left( \frac{2}{a_3^2} - \frac{1}{a_1^2} \right) x_0}{\frac{y_0}{b_3^2}} = - \frac{b_3^2}{y_0} \left( \frac{2}{a_3^2} - \frac{1}{a_1^2} \right) x_0$动点 $P$ 的斜率 $k_{OP} = \frac{y_0}{x_0}$二者相乘，动点坐标 $x_0, y_0$ 完美互为倒数约去：$$k_{OP} \cdot k_{AB} = \frac{y_0}{x_0} \cdot \left[ - \frac{b_3^2}{y_0} \left( \frac{2}{a_3^2} - \frac{1}{a_1^2} \right) x_0 \right] = - b_3^2 \left( \frac{2}{a_3^2} - \frac{1}{a_1^2} \right) = \mathbf{\frac{b_3^2(a_3^2 - 2a_1^2)}{a_1^2 a_3^2}}$$定值证毕！
\item 求证相切并求出定椭圆:一条直线 $Ax + By = 1$ 与标准椭圆 $\frac{x^2}{M^2} + \frac{y^2}{N^2} = 1$ 相切的充要条件是：$M^2 A^2 + N^2 B^2 = 1$。我们把读出的 $A, B$ 塞进这个条件里凑数：\[M^2 \left[ \left( \frac{2}{a_3^2} - \frac{1}{a_1^2} \right) x_0 \right]^2 + N^2 \left( \frac{y_0}{b_3^2} \right)^2 = 1\]
因为动点 $P$ 的被动技能是 $\frac{x_0^2}{a_1^2} + \frac{y_0^2}{b_3^2} = 1$，为了让上式对所有 $(x_0,y_0)$ 绝对成立，我们强行让 $x_0^2$ 的总系数等于 $\frac{1}{a_1^2}$，让 $y_0^2$ 的总系数等于 $\frac{1}{b_3^2}$：
\begin{itemize}
\item 关于 $y_0^2$：$N^2 \cdot \frac{1}{b_3^4} = \frac{1}{b_3^2} \implies \mathbf{N^2 = b_3^2}$
\item 关于 $x_0^2$：$M^2 \left( \frac{2a_1^2 - a_3^2}{a_1^2 a_3^2} \right)^2 = \frac{1}{a_1^2} \implies {M^2 = \frac{a_1^2 a_3^4}{(2a_1^2 - a_3^2)^2}}$
\end{itemize}
既然 $M^2$ 和 $N^2$ 算出来全是不带变量的常数，说明这根弦在滑动过程中，永远死死贴在同一个椭圆上！相切证毕！定椭圆的标准方程直接出炉：$${\frac{(2a_1^2 - a_3^2)^2}{a_1^2 a_3^4} x^2 + \frac{y^2}{b_3^2} = 1}$$
\end{itemize}
\end{solution}

\begin{example}{}{}
    已知 $P$ 为椭圆 $C_1: \frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$ 上一动点，$E(m,0), G(n,0)$ 为 $x$ 轴上的两定点，延长 $PG, PE$ 分别交椭圆 $C_1$ 于 $A, B$ 两点\\
(1) 求证：在 $x$ 轴存在点 $M$，使得 $k_{MP} \cdot k_{AB}$ 为定值，并求出该定值\\
(2) 求证：直线 $AB$ 与一定椭圆相切，并求出该椭圆的标准方程
\end{example}
\begin{solution}
设动点 $P(x_0, y_0)$ 在椭圆上，自带被动技能 $E(P,P) = \frac{x_0^2}{a^2} + \frac{y_0^2}{b^2} = 1$。引入探测向量 $\vec{v} = (X, Y) = (x-x_0, y-y_0)$，将椭圆方程在 $P$ 点隐式展开，截断后得到向量视角的二次方程：
$$E(\vec{v}, \vec{v}) + 2E(P, \vec{v}) = 0$$
设我们要找的割线 $AB$ 局部线性方程为 $L(\vec{v}) + 1 = 0$（即 $uX+vY+1=0$）。将其作为齐次化因子乘给一次项构造恒等结构：
$$E(\vec{v}, \vec{v}) - 2E(P, \vec{v})L(\vec{v}) = 0$$
现在处理两条弦 $PE$ 和 $PG$。在以 $P$ 为原点的局部向量下，点 $E(m,0), G(n,0)$ 对应向量为 $(m-x_0, -y_0)$ 和 $(n-x_0, -y_0)$。两弦斜率分别为：
$$k_1 = \frac{-y_0}{m-x_0} = \frac{y_0}{x_0-m}, \quad k_2 = \frac{-y_0}{n-x_0} = \frac{y_0}{x_0-n}$$
立刻算这两个斜率的积 $K$ 以及和 $S$：
$$K = k_1 k_2 = \frac{y_0^2}{(x_0-m)(x_0-n)}，~~S = k_1 + k_2 = \frac{y_0(2x_0 - m - n)}{(x_0-m)(x_0-n)}$$
构造斜率纯二次型 $H(\vec{v}) = Y^2 - S \cdot XY + K \cdot X^2$。它与上面的齐次方程只差一个全局常数倍 $\mu$：$$E(\vec{v}, \vec{v}) - \mu H(\vec{v}) \equiv 2E(P, \vec{v}) \cdot L(\vec{v})$$
令正交探针 $\vec{t} = (a^2 y_0, -b^2 x_0)$，使得切向算子 $E(P, \vec{t}) = 0$。代入恒等式得 $\mu = \frac{E(\vec{t}, \vec{t})}{H(\vec{t})}$。
\begin{align*}
    E(\vec{t}, \vec{t}) &= \frac{(a^2 y_0)^2}{a^2} + \frac{(-b^2 x_0)^2}{b^2} = a^2 b^2 \left( \frac{y_0^2}{b^2} + \frac{x_0^2}{a^2} \right) = a^2 b^2\\
    H(\vec{t}) &= (-b^2 x_0)^2 - S(a^2 y_0)(-b^2 x_0) + K(a^2 y_0)^2 = b^4 x_0^2 + S a^2 b^2 x_0 y_0 + K a^4 y_0^2\\
    &=\frac{b^4 x_0^2(x_0^2 - (m+n)x_0 + mn) + a^2 b^2 x_0 y_0^2 (2x_0 - m - n) + a^4 y_0^4}{(x_0-m)(y_0-n)}\\
    &=\frac{b^4 \left[ x_0^2(x_0^2 - (m+n)x_0 + mn) + x_0(a^2 - x_0^2)(2x_0 - m - n) + (a^2 - x_0^2)^2 \right]}{(x_0-m)(y_0-n)}\\
    &=\frac{b^4\left[x_0^4 - (m+n)x_0^3 + mn x_0^2 + 2a^2 x_0^2 - a^2(m+n)x_0 - 2x_0^4 + (m+n)x_0^3 + a^4 - 2a^2 x_0^2 + x_0^4\right]}{(x_0-m)(y_0-n)}\\
    &=\frac{b^4(a^4 - a^2(m+n)x_0 + mn x_0^2)}{(x_0-m)(y_0-n)}=\frac{b^4(a^2-mx_0)(a^2-nx_0)}{(x_0-m)(y_0-n)}\\
\implies \mu &= \frac{E(\vec{t}, \vec{t})}{H(\vec{t})} = \frac{a^2(x_0-m)(y_0-n)}{b^2(a^2-mx_0)(a^2-nx_0)}
\end{align*}
下面解出$u,v$，代 $\vec{e_1} = (1,0)$得$\frac{1}{a^2} - \mu k_1k_2 = 2\left(\frac{x_0}{a^2}\right)u $
\begin{align*}
\implies 2x_0 u &= 1 - a^2 \mu \frac{y_0^2}{(x_0-m)(x_0-n)}\\
&=1 - a^2 \frac{a^2(x_0-m)(y_0-n)}{b^2(a^2-mx_0)(a^2-nx_0)} \frac{y_0^2}{(x_0-m)(x_0-n)}\\
&=1 - \frac{a^4y_0^2}{b^2(a^2-mx_0)(a^2-nx_0)}= 1 - \frac{a^4b^2(1-\frac{x_0^2}{a^2})}{b^2(a^2-mx_0)(a^2-nx_0)}\\
&= 1 - \frac{a^2(a^2-x_0^2)}{(a^2-mx_0)(a^2-nx_0)}=\frac{a^4 - a^2(m+n)x_0 + mn x_0^2 - a^4 + a^2 x_0^2}{(a^2-mx_0)(a^2-nx_0)} \\
&= \frac{x_0[(a^2+mn)x_0 - a^2(m+n)]}{(a^2-mx_0)(a^2-nx_0)}\implies u = \frac{(a^2+mn)x_0 - a^2(m+n)}{2 (a^2-mx_0)(a^2-nx_0)}
\end{align*}
代入 $\vec{e_2} = (0,1)$：$\frac{1}{b^2} - \mu = 2\left(\frac{y_0}{b^2}\right)v $
\begin{align*}
\implies 2y_0 v =& 1 - b^2 \mu = 1 - \frac{a^2(x_0^2-(m+n)x_0+mn)}{ (a^2-mx_0)(a^2-nx_0)}\\
=&\frac{a^4-a^2(m+n)x_0+mnx_0^2-a^2(x_0^2-(m+n)x_0+mn)}{ (a^2-mx_0)(a^2-nx_0)}\\
=&\frac{a^2(a^2-x_0^2)+mn(x_0^2-a^2)}{ (a^2-mx_0)(a^2-nx_0)}=\frac{(a^2-mn)(a^2-x_0^2)}{(a^2-mx_0)(a^2-nx_0)}=\frac{(a^2-mn)\frac{a^2y_0^2}{b^2}}{(a^2-mx_0)(a^2-nx_0)}\\
\implies v =& \frac{a^2(a^2-mn)y_0}{2 b^2  (a^2-mx_0)(a^2-nx_0)}
\end{align*}
将局部线 $u(x-x_0) + v(y-y_0) + 1 = 0$ 展开为 $ux + vy + (1 - ux_0 - vy_0) = 0$。计算常数项 
\begin{align*}
1 - (ux_0 + vy_0)&=1-\left(\frac{(a^2+mn)x_0^2 - a^2(m+n)x_0}{2 (a^2-mx_0)(a^2-nx_0)} + \frac{a^2(a^2-mn)y_0^2}{2b^2 (a^2-mx_0)(a^2-nx_0)}\right)\\
&=1-\left(\frac{(a^2+mn)x_0^2 - a^2(m+n)x_0}{2 (a^2-mx_0)(a^2-nx_0)} + \frac{a^2(a^2-mn)b^2(1-\frac{x_0^2}{a^2})}{2b^2 (a^2-mx_0)(a^2-nx_0)}\right)\\
&=1-\left(\frac{(a^2+mn)x_0^2 - a^2(m+n)x_0}{2 (a^2-mx_0)(a^2-nx_0)} + \frac{(a^2-mn)(a^2-x_0^2)}{2(a^2-mx_0)(a^2-nx_0)}\right)\\
&=\frac{2(a^2-mx_0)(a^2-nx_0)-(a^2+mn)x_0^2 + a^2(m+n)x_0-(a^2-mn)(a^2-x_0^2)}{2(a^2-mx_0)(a^2-nx_0)}\\
&= \frac{a^4 - a^2(m+n)x_0 + a^2 mn}{2(a^2-mx_0)(a^2-nx_0)} = \frac{a^2[a^2 - (m+n)x_0 + mn]}{2(a^2-mx_0)(a^2-nx_0)}
\end{align*}
方程同除以这个负的常数项，得到截距式 $Ax + By = 1$：
$$A = -\frac{(a^2+mn)x_0 - a^2(m+n)}{a^2[a^2 - (m+n)x_0 + mn]}, \quad B = -\frac{(a^2-mn)y_0}{b^2[a^2 - (m+n)x_0 + mn]}$$
（1）割线斜率 $\displaystyle k_{AB} = -\frac{A}{B} = - \frac{b^2[(a^2+mn)x_0 - a^2(m+n)]}{a^2(a^2-mn)y_0}$，设 $x$ 轴上的点 $M(t,0)$，则 $k_{MP} = \frac{y_0}{x_0-t}$，乘积：$$k_{MP} \cdot k_{AB} = \frac{y_0}{x_0-t} \times \left( - \frac{b^2[(a^2+mn)x_0 - a^2(m+n)]}{a^2(a^2-mn)y_0} \right) = - \frac{b^2(a^2+mn)}{a^2(a^2-mn)} \cdot \frac{x_0 - \frac{a^2(m+n)}{a^2+mn}}{x_0-t}$$为了让这个式子不依赖于动点横坐标 $x_0$（即成为定值），分子分母关于 $x_0$ 必须严格成正比。所以直接锁死 $t$ 的值：$$t = \frac{a^2(m+n)}{a^2+mn}$$第一问得证：存在定点 $M \left( \frac{a^2(m+n)}{a^2+mn}, 0 \right)$，使得 $k_{MP} \cdot k_{AB}$ 为绝对定值 $-\frac{b^2(a^2+mn)}{a^2(a^2-mn)}$。\\
（2）直线 $Ax + By = 1$ 与标准椭圆 $\frac{x^2}{\alpha^2} + \frac{y^2}{\beta^2} = 1$ 相切的充要条件是：$\alpha^2 A^2 + \beta^2 B^2 = 1$。把我们求出来的 $A, B$ 代入，分母平方都是 $[a^2 - (m+n)x_0 + mn]^2$。我们要证明：
$$\frac{\alpha^2}{a^4} [ (a^2+mn)x_0 - a^2(m+n) ]^2 + \frac{\beta^2}{b^4} (a^2-mn)^2 y_0^2 \equiv [a^2 - (m+n)x_0 + mn]^2$$
利用替换 $y_0^2 = b^2(1 - \frac{x_0^2}{a^2})$，全式化为关于动点 $x_0$ 的恒等式：
\begin{itemize}
\item 比对一次项 $x_0$ 系数：左边 $= 2 \frac{\alpha^2}{a^4} (a^2+mn)(-a^2(m+n)) = -2 \frac{\alpha^2}{a^2} (a^2+mn)(m+n)$；右边展开的一次项 $= 2(a^2+mn)(-(m+n)x_0)$，系数为 $-2(a^2+mn)(m+n)$；两边相等，一秒爆出：$\alpha^2 = a^2$
\item 比对二次项 $x_0^2$ 系数：左边 $= \frac{a^2}{a^4}(a^2+mn)^2 - \frac{\beta^2}{b^4} (a^2-mn)^2 \frac{b^2}{a^2} = \frac{(a^2+mn)^2}{a^2} - \frac{\beta^2(a^2-mn)^2}{a^2 b^2}$右边展开的二次项 $= (m+n)^2$则：$$\frac{\beta^2(a^2-mn)^2}{a^2 b^2} = \frac{(a^2+mn)^2 - a^2(m+n)^2}{a^2} =  \frac{(a^2-m^2)(a^2-n^2)}{a^2}$$直接解出：$\beta^2 = b^2 \frac{(a^2-m^2)(a^2-n^2)}{(a^2-mn)^2}$
\item 最后验算常数项（闭环确认）：左边 $= \frac{a^2}{a^4} a^4(m+n)^2 + \frac{\beta^2}{b^2}(a^2-mn)^2 = a^2(m+n)^2 + (a^2-m^2)(a^2-n^2)$$= a^2 m^2 + 2a^2 mn + a^2 n^2 + a^4 - a^2 n^2 - a^2 m^2 + m^2 n^2 = a^4 + 2a^2 mn + m^2 n^2 = (a^2+mn)^2$右边常数项也是 $(a^2+mn)^2$。完美咬合！
\end{itemize}
第二问得证：弦 $AB$ 无论怎么跑，永远死死贴在下面这个确定的内层椭圆上相切！该定椭圆标准方程为：$$\frac{x^2}{a^2} + \frac{y^2}{b^2 \frac{(a^2-m^2)(a^2-n^2)}{(a^2-mn)^2}} = 1$$
\end{solution}

\begin{example}{}{}
    已知 $P$ 为椭圆 $C_1: \frac{x^2}{a_1^2} + \frac{y^2}{b_1^2} = 1$ 上一动点，过点 $P$ 作圆 $C_2: (x - m)^2 + y^2 = r^2$ 的两切线分别交椭圆 $C_1$ 于 $A, B$ 两点。求证：在 $x$ 轴存在点 $M$，使得 $k_{MP} \cdot k_{AB}$ 为定值，并求出该定值；
\end{example}
\begin{solution}
设动点 $P(x_0, y_0)$ 在椭圆 $C_1$ 上，自带$E_1(P,P) = \frac{x_0^2}{a_1^2} + \frac{y_0^2}{b_1^2} = 1$。引入局部探测向量 $\vec{v} = (X,Y) = (x-x_0, y-y_0)$。
将 $C_1$ 方程在 $P$ 点隐式展开得到：$$E_1(\vec{v}, \vec{v}) + 2E_1(P, \vec{v}) = 0$$
圆 $C_2: (x-m)^2 + y^2 = r^2$。在局部平移下，圆心坐标变为 $(m-x_0, -y_0)$。从原点 $(0,0)$（即全局的 $P$ 点）向该圆引出的两条切线，其方程本质上就是距离公式的纯二次齐次化展开。根据初中完全平方公式，这对切线的纯二次型为：
$$H(\vec{v}) = (r^2 - y_0^2)X^2 + 2(x_0-m)y_0 XY + [r^2 - (x_0-m)^2]Y^2 = 0$$
设我们要找的割线 $AB$ 局部线性方程为 $L(\vec{v}) + 1 = 0$（即 $uX+vY+1=0$）。将切线对 $H(\vec{v})$、椭圆 $C_1$ 和割线 $L(\vec{v})$ 进行算子缝合，必然存在一个全局缩放常数 $\mu$，使得下面的多项式对空间任意向量 $\vec{v}$ 绝对恒等：$$E_1(\vec{v}, \vec{v}) - \mu H(\vec{v}) \equiv 2E_1(P, \vec{v}) \cdot L(\vec{v})$$
令正交探针 $\vec{t} = (a_1^2 y_0, -b_1^2 x_0)$，使得切向算子 $E_1(P, \vec{t}) = 0$。代入恒等式直接得到 $\mu = \dfrac{E_1(\vec{t}, \vec{t})}{H(\vec{t})}$。
\begin{itemize}
\item 分子：$E_1(\vec{t}, \vec{t}) = \frac{(a_1^2 y_0)^2}{a_1^2} + \frac{(-b_1^2 x_0)^2}{b_1^2} = a_1^2 b_1^2 \left( \frac{y_0^2}{b_1^2} + \frac{x_0^2}{a_1^2} \right) = {a_1^2 b_1^2}$
\item 分母：将 $\vec{t}$ 塞入 $H(\vec{v})$：
\begin{align*}
H(\vec{t}) &= (r^2 - y_0^2)(a_1^4 y_0^2) - 2(x_0-m)y_0(a_1^2 y_0)(b_1^2 x_0) + [r^2 - (x_0-m)^2](b_1^4 x_0^2)\\
&= r^2(a_1^4 y_0^2 + b_1^4 x_0^2)- [a_1^4 y_0^4 + 2(x_0-m)x_0 a_1^2 b_1^2 y_0^2 + (x_0-m)^2 b_1^4 x_0^2]\\
&= r^2(a_1^2 b_1^2(a_1^2 - x_0^2) + b_1^4 x_0^2)  - [a_1^2 y_0^2 + (x_0-m)b_1^2 x_0]^2\\
&= r^2 b_1^2 [a_1^4 - (a_1^2 - b_1^2)x_0^2] - [a_1^2 b_1^2-b_1^2 x_0^2 + (x_0-m)b_1^2 x_0]^2\\
&= r^2 b_1^2 (a_1^4 - c^2 x_0^2)- [a_1^2 b_1^2-b_1^2 x_0^2 + (x_0-m)b_1^2 x_0]^2\\
&= b_1^2 [r^2(a_1^4 - c^2 x_0^2) - b_1^2(a_1^2 - m x_0)^2]
\end{align*}
为了书写极简，我们定义核心分母 $D_0 = r^2(a_1^4 - c^2 x_0^2) - b_1^2(a_1^2 - m x_0)^2$。$\mu$ 瞬间出炉：$$\mu = \frac{a_1^2}{D_0}$$
\end{itemize}
代入 $\vec{e_1} = (1,0)$ 到恒等式，看 $X^2$ 系数：
\begin{align*}
&\frac{1}{a_1^2} - \mu(r^2 - y_0^2) = \frac{2x_0 u}{a_1^2} \implies 2x_0 u = 1 - \frac{a_1^4(r^2 - y_0^2)}{D_0} = \frac{D_0 - a_1^4(r^2 - y_0^2)}{D_0}\\
=&\frac{r^2 a_1^4 - r^2 c^2 x_0^2 - b_1^2(a_1^4 - 2a_1^2 m x_0 + m^2 x_0^2) - a_1^4 r^2 + a_1^4 y_0^2}{D_0}\\
=&\frac{-r^2 c^2 x_0^2 - a_1^4 b_1^2 + 2a_1^2 b_1^2 m x_0 - b_1^2 m^2 x_0^2 + a_1^4 b_1^2 - a_1^2 b_1^2 x_0^2}{D_0}\\
=& \frac{x_0 [2a_1^2 b_1^2 m - (r^2 c^2 + b_1^2 m^2 + a_1^2 b_1^2) x_0]}{D_0} \implies u = \frac{2a_1^2 b_1^2 m - (r^2 c^2 + b_1^2 m^2 + a_1^2 b_1^2) x_0}{2 D_0}
\end{align*}
代入 $\vec{e_2} = (0,1)$ 到恒等式，看 $Y^2$ 系数：
\begin{align*}
&\frac{1}{b_1^2} - \mu[r^2 - (x_0-m)^2] = \frac{2y_0 v}{b_1^2} \implies 2y_0 v = \frac{D_0 - a_1^2 b_1^2 [r^2 - (x_0-m)^2]}{D_0}\\
=&\frac{r^2(a_1^4 - c^2 x_0^2) - b_1^2(a_1^2 - m x_0)^2-a_1^2 b_1^2 [r^2 - (x_0-m)^2]}{D_0}\\
=&\frac{r^2a_1^4-r^2c^2x_0^2-b_1^2\left[(a_1^4-2a_1^2m x_0+m^2x_0^2)+a_1^2r^2-a_1^2(x_0^2-2mx_0+m^2)\right]}{D_0}\\
=&\frac{r^2 c^2(a_1^2 - x_0^2) - b_1^2(a_1^2 - m^2)(a_1^2 - x_0^2)}{D_0}\\
=&\frac{\frac{a_1^2}{b_1^2} y_0^2 [r^2 c^2 - b_1^2(a_1^2 - m^2)]}{D_0} \implies v = \frac{a_1^2 y_0 [r^2 c^2 - b_1^2(a_1^2 - m^2)]}{2b_1^2 D_0}
\end{align*}
割线 $AB$ 在局部坐标系下的方程是 $uX + vY + 1 = 0$，它的斜率就是：
$$k_{AB} = -\frac{u}{v} = - \frac{\frac{2a_1^2 b_1^2 m - (r^2 c^2 + b_1^2 m^2 + a_1^2 b_1^2) x_0}{2 D_0}}{\frac{a_1^2 y_0 [r^2 c^2 - b_1^2(a_1^2 - m^2)]}{2b_1^2 D_0}}= - \frac{b_1^2 [2a_1^2 b_1^2 m - (r^2 c^2 + b_1^2 m^2 + a_1^2 b_1^2) x_0]}{a_1^2 y_0 [r^2 c^2 - b_1^2(a_1^2 - m^2)]}$$
要在 $x$ 轴上找一个点 $M(t, 0)$，使得 $k_{MP} \cdot k_{AB}$ 为定值。$M$ 和 $P$ 的连线斜率 $k_{MP} = \frac{y_0}{x_0 - t}$。我们把两者乘起来，动点纵坐标 $y_0$ 被完美对消！
$$k_{MP} \cdot k_{AB} = \frac{-b_1^2}{a_1^2 [r^2 c^2 - b_1^2(a_1^2 - m^2)]} \times \frac{2a_1^2 b_1^2 m - (r^2 c^2 + b_1^2 m^2 + a_1^2 b_1^2) x_0}{x_0 - t}$$为了让这个式子成为一个与动点 $x_0$ 毫无关系的绝对常数，右边那个分式的分子和分母必须完全成比例！我们提取分子的 $x_0$ 系数 $-(r^2 c^2 + b_1^2 m^2 + a_1^2 b_1^2)$，强行让它和分母 $(x_0 - t)$ 对齐，直接锁死 $t$ 的坐标：
$$t = \frac{2a_1^2 b_1^2 m}{r^2 c^2 + b_1^2 m^2 + a_1^2 b_1^2}$$
定点 $M$ 坐标就是 $\left( \frac{2a_1^2 b_1^2 m}{r^2 c^2 + b_1^2 m^2 + a_1^2 b_1^2}, 0 \right)$。当 $t$ 取这个值时，含有 $x_0$ 的分式瞬间约简为常数 $-(r^2 c^2 + b_1^2 m^2 + a_1^2 b_1^2)$。把它乘到前面的系数上，定值直接引爆：$$k_{MP} \cdot k_{AB} = {\frac{b_1^2 (r^2 c^2 + b_1^2 m^2 + a_1^2 b_1^2)}{a_1^2 [r^2 c^2 - b_1^2(a_1^2 - m^2)]}}$$
\end{solution}

\begin{example}{}{}
     $P$ 为圆 $O: x^2 + y^2 = 1$ 上一动点，过点 $P$ 作圆 $O$ 的切线交椭圆 $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1, (a > b > 1)$ 于 $A, B$ 两点，再过点 $A, B$ 分别作圆 $O$ 的另一条切线 $AQ, BQ$ 交于点 $Q$，求动点 $Q$ 的轨迹方程；
\end{example}
\begin{solution}
动点 $P(x_P, y_P)$ 在单位圆 $O$ 上：$$x_P^2 + y_P^2 = 1$$
直线 $AB$ 是圆 $O$ 在 $P$ 点的切线，其算子方程为：$$x_P x + y_P y = 1$$
点 $A, B$ 是这条切线与椭圆 $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$ 的交点。我们将直线 $AB$ 的方程代入椭圆方程进行齐次化，：
$$\frac{x^2}{a^2} + \frac{y^2}{b^2} = (x_P x + y_P y)^2$$展开并合并同类项，得到纯二次型
\[A_0 x^2 + B_0 xy + C_0 y^2 = 0,~A_0 = \frac{1}{a^2} - x_P^2,B_0 = -2x_P y_P,C_0 = \frac{1}{b^2} - y_P^2\]
点 $Q$ 是两条切线 $QA, QB$ 的交点。设这两条切线与圆 $O$ 的切点分别为 $T_1, T_2$。根据极点极线对偶定理，$Q$ 的极线就是直线 $T_1 T_2$。同时，$A$ 是切线 $AP$ 和 $AT_1$ 的交点，所以 $A$ 的极线是 $P T_1$。由于 $P, T_1$ 都在圆上，这意味着 $T_1$ 本质上是点 $P$ 关于射线 $OA$ 的对称点！同理，$T_2$ 是 $P$ 关于射线 $OB$ 的对称点。设射线 $OA, OB$ 的倾斜角分别为 $\alpha, \beta$。根据对称，切点 $T_1, T_2$ 的极角分别为 $2\alpha - \theta_P$ 和 $2\beta - \theta_P$。极线 $T_1 T_2$ 的法线方向直接指向极点 $Q$，所以 $Q$ 的极角 $\theta_Q$ 恰好是 $T_1, T_2$ 极角的平均值：
$$\theta_Q = \alpha + \beta - \theta_P \implies \theta_Q + \theta_P = \alpha + \beta$$
对两边同时取正切，利用第一步中 $OA, OB$ 的韦达定理
\[\tan\alpha + \tan\beta = -\frac{B_0}{C_0},~~\tan\alpha \tan\beta = \frac{A_0}{C_0}\]
得到：$$\tan(\theta_Q + \theta_P) = \tan(\alpha + \beta) = \frac{-\frac{B_0}{C_0}}{1 - \frac{A_0}{C_0}} = \frac{-B_0}{C_0 - A_0}$$
将其展开为坐标系下的比例式（令 $x_Q, y_Q$ 为 $Q$ 的坐标）：$$\frac{x_P y_Q + y_P x_Q}{x_P x_Q - y_P y_Q} = \frac{2x_P y_P}{\frac{1}{b^2} - \frac{1}{a^2} + x_P^2 - y_P^2}$$
交叉相乘，我们将动点 $Q$ 和 $P$ 的坐标彻底分离。左边提取 $x_Q$，右边提取 $y_Q$，利用 $x_P^2 + y_P^2 = 1$ 进行抵消：
$$x_Q \left[ \left(\frac{1}{b^2} - \frac{1}{a^2} + x_P^2 - y_P^2\right) y_P - 2x_P^2 y_P \right] + y_Q \left[ \left(\frac{1}{b^2} - \frac{1}{a^2} + x_P^2 - y_P^2\right) x_P + 2x_P y_P^2 \right] = 0$$
化简括号内的多项式，极其干净的常数系数诞生了：$$x_Q \left( \frac{1}{b^2} - \frac{1}{a^2} - 1 \right) y_P + y_Q \left( \frac{1}{b^2} - \frac{1}{a^2} + 1 \right) x_P = 0$$
为了书写绝美，我们定义两个常数：
\[c_1 = \frac{1}{b^2} - \frac{1}{a^2} + 1 = \frac{a^2 - b^2 + a^2 b^2}{a^2 b^2},~c_2 = \frac{1}{b^2} - \frac{1}{a^2} - 1 = \frac{a^2 - b^2 - a^2 b^2}{a^2 b^2}\]
于是坐标被完美解耦：
$$\frac{x_Q}{c_1 x_P} = \frac{y_Q}{-c_2 y_P} = \lambda$$
这意味着 $x_P = \frac{x_Q}{\lambda c_1}$ 且 $y_P = \frac{y_Q}{-\lambda c_2}$。那么缩放比例 $\lambda$ 是多少呢？我们计算极点 $Q$ 的模长 $OQ^2 = 1 + \tan^2(\alpha - \beta)$，经过与第一步系数的同构比对，它严格等于：$$\lambda = \frac{1}{A_0 + C_0} = \frac{1}{\frac{1}{a^2} + \frac{1}{b^2} - (x_P^2 + y_P^2)} = \frac{1}{\frac{1}{a^2} + \frac{1}{b^2} - 1}$$
定义这个绝对不变量为 $K = \frac{a^2 + b^2 - a^2 b^2}{a^2 b^2}$。
既然 $P$ 在单位圆上，我们将解耦出的 $x_P, y_P$ 直接代入被动技能 $x_P^2 + y_P^2 = 1$ 中：$$\left( \frac{K}{c_1} x_Q \right)^2 + \left( -\frac{K}{c_2} y_Q \right)^2 = 1$$将 $K, c_1, c_2$ 的表达式整体代入，分子分母的 $a^2 b^2$ 完美约去。动点 $Q$ 的轨迹瞬间定型：$$\frac{x^2}{\left( \frac{a^2 - b^2 + a^2 b^2}{a^2 + b^2 - a^2 b^2} \right)^2} + \frac{y^2}{\left( \frac{a^2 - b^2 - a^2 b^2}{a^2 + b^2 - a^2 b^2} \right)^2} = 1$$
\end{solution}

\begin{example}{}{}
点 $P$ 为圆 $O: x^2 + y^2 = 1$ 上一动点，圆 $O$ 在点 $P$ 处切线交椭圆 $C: \frac{x^2}{3} + \frac{y^2}{2} = 1$ 于 $A, C$ 两点，再过点 $A, C$ 作圆 $O$ 的另一条切线分别交椭圆 $C$ 于点 $B, D$ 两点，过点 $B, D$ 作圆 $O$ 的另一条切线相交于点 $Q$，求动点 $Q$ 的轨迹方程；
\end{example}
\begin{solution}
我们先理清这五个切点的位置关系：
\begin{itemize}
\item 动点 $P$ 在圆 $O: x^2 + y^2 = 1$ 上。过 $P$ 的切线交椭圆 $C: \frac{x^2}{3} + \frac{y^2}{2} = 1$ 于 $A, C$。
\item 再过 $A, C$ 作圆的切线 $AB$ 和 $CD$，它们的切点记为 $P_1$ 和 $P_2$。
\item 过 $B, D$ 作圆的切线 $BQ$ 和 $DQ$，它们的切点记为 $P_3$ 和 $P_4$。
\end{itemize}
根据讲义中的极点极线算子定理：
\begin{itemize}
\item $A$ 是切线 $AP$ 和 $AP_1$ 的交点。所以 $A$ 的极线就是连接切点的弦 $P P_1$。
\item 因为点 $A$ 在椭圆 $\frac{x^2}{3} + \frac{y^2}{2} = 1$ 上，所以它的极线 $P P_1$ 必定与该椭圆的对偶椭圆 $E^*: 3x^2 + 2y^2 = 1$ 相切。
\item 同理，点 $C, B, D$ 都在原椭圆上，所以它们对应的极线 $P P_2$, $P_1 P_3$, $P_2 P_4$ 全部都与对偶椭圆 $E^*: 3x^2 + 2y^2 = 1$ 相切！
\item 这构成了一个链条：在单位圆上，弦 $P P_1$、$P_1 P_3$、$P P_2$、$P_2 P_4$ 全都与 $3x^2 + 2y^2 = 1$ 相切
\end{itemize}
我们需要找出一个绝对的代数条件：单位圆上的弦，怎样才会与 $3x^2 + 2y^2 = 1$ 相切？设单位圆上两点对应的极角为 $\theta_1, \theta_2$。这条弦到原点的距离 $d = \cos\frac{\theta_1 - \theta_2}{2}$，其法线方向 $\sigma = \frac{\theta_1 + \theta_2}{2}$。因此，该弦的方程可写为：
\[x \cos\sigma + y \sin\sigma = \cos\frac{\theta_1 - \theta_2}{2}\]
两边同除以右侧化为截距式 $ux + vy = 1$，得到 $u = \frac{\cos\sigma}{\cos\delta}$， $v = \frac{\sin\sigma}{\cos\delta}$ （其中 $\delta = \frac{\theta_1 - \theta_2}{2}$）。直线与 $3x^2 + 2y^2 = 1$ 相切的充要条件是 $\frac{u^2}{3} + \frac{v^2}{2} = 1$。直接代入：
\begin{align*}
&\frac{\cos^2\sigma}{3\cos^2\delta} + \frac{\sin^2\sigma}{2\cos^2\delta} = 1\\
\Leftrightarrow & 6\cos^2\delta = 2\cos^2\sigma + 3\sin^2\sigma = 2 + \sin^2\sigma\Leftrightarrow  6 \left( \frac{1 + \cos(2\delta)}{2} \right) = 2 + \frac{1 - \cos(2\sigma)}{2}\\
\Leftrightarrow & 3 + 3\cos(\theta_1 - \theta_2) = \frac{5 - \cos(\theta_1 + \theta_2)}{2}\Leftrightarrow 6 + 6\cos(\theta_1 - \theta_2) + \cos(\theta_1 + \theta_2) - 5 = 0\\
\Leftrightarrow & 1 + 6\cos(\theta_1 - \theta_2) + \cos(\theta_1 + \theta_2) = 0\Leftrightarrow 1 + 6(x_1 x_2 + y_1 y_2) + (x_1 x_2 - y_1 y_2) = 0\\
\Leftrightarrow & 7x_1 x_2 + 5y_1 y_2 + 1 = 0
\end{align*}
单位圆上任意两点 $X, Y$ 的连线与 $3x^2 + 2y^2 = 1$ 相切的充要条件即满足 $B_1(X, Y) = 7x_X x_Y + 5y_X y_Y + 1 = 0$\\
回到我们的链条：弦 $P P_1$ 和弦 $P_3 P_1$ 都与 $E^*$ 相切。这意味着 $P_1(x_1, y_1)$ 同时满足：
$$7x_P x_1 + 5y_P y_1 + 1 = 0 \quad \text{且} \quad 7x_3 x_1 + 5y_3 y_1 + 1 = 0$$
将这视为关于 $x_1, y_1$ 的二元一次方程组，解出：
$$x_1 = \frac{5(y_P - y_3)}{35(x_P y_3 - x_3 y_P)} = \frac{y_P - y_3}{7(x_P y_3 - x_3 y_P)},~~y_1 = \frac{7(x_3 - x_P)}{35(x_P y_3 - x_3 y_P)} = \frac{x_3 - x_P}{5(x_P y_3 - x_3 y_P)}$$
因为点 $P_1$ 在单位圆上，代入 $x_1^2 + y_1^2 = 1$，并同乘公分母的平方 $1225(x_P y_3 - x_3 y_P)^2$：$$25(y_P - y_3)^2 + 49(x_P - x_3)^2 = 1225(x_P y_3 - x_3 y_P)^2$$为了继续剥离变量，我们再次使用高中极角参数代入并利用和差化积：
\[(y_P - y_3)^2 = 4\sin^2\delta \cos^2\sigma,~(x_P - x_3)^2 = 4\sin^2\delta \sin^2\sigma,~(x_P y_3 - x_3 y_P)^2 = \sin^2(2\delta)\]
再次使用二倍角公式降次展开：
\begin{align*}
& 25 \left( \frac{1 + \cos(2\sigma)}{2} \right) + 49 \left( \frac{1 - \cos(2\sigma)}{2} \right) = 1225 \left( \frac{1 + \cos(2\delta)}{2} \right)\\
\Leftrightarrow & 74 - 24\cos(\theta_P + \theta_3) = 1225 + 1225\cos(\theta_P - \theta_3)\\
\Leftrightarrow & 1225(x_P x_3 + y_P y_3) + 24(x_P x_3 - y_P y_3) + 1151 = 0\\
\Leftrightarrow & 1249 x_P x_3 + 1201 y_P y_3 + 1151 = 0
\end{align*}
点 $P_3$ 必须满足以 $P$ 为系数的线性方程！由于系统的绝对对称性，由点 $C$ 引出的另一侧链条产生的点 $P_4$，必定也满足一模一样的方程：$$1249 x_P x_4 + 1201 y_P y_4 + 1151 = 0$$既然 $P_3, P_4$ 的坐标都满足同一个一次方程，那么连接 $P_3, P_4$ 的直线方程就被唯一确定：$$1249 x_P x + 1201 y_P y + 1151 = 0$$
题目最终问的是点 $Q$ 的轨迹。点 $Q$ 是切线 $BQ$ 和 $DQ$ 的交点，而这两条切线的切点正是 $P_3$ 和 $P_4$！根据算子理论的极化定义，过切点 $P_3, P_4$ 的弦，正是交点 $Q$ 关于单位圆的极线！点 $Q(x_Q, y_Q)$ 关于单位圆 $x^2 + y^2 = 1$ 的极线方程标准形式为：$$x_Q x + y_Q y = 1$$
我们将第三阶算出的直线 $P_3 P_4$ 方程同除以 $-1151$，化为绝对标准的极线形式：
$$-\frac{1249}{1151} x_P x - \frac{1201}{1151} y_P y = 1$$
两条线是同一条线，直接进行系数对比：
$$x_Q = -\frac{1249}{1151} x_P \implies x_P = -\frac{1151}{1249} x_Q,~y_Q = -\frac{1201}{1151} y_P \implies y_P = -\frac{1151}{1201} y_Q$$
因为动点 $P$ 活生生地在单位圆上，自带 $x_P^2 + y_P^2 = 1$。将反解出的坐标毫不留情地砸进去：$$\left( -\frac{1151}{1249} x_Q \right)^2 + \left( -\frac{1151}{1201} y_Q \right)^2 = 1\implies\frac{x^2}{\left(\frac{1249}{1151}\right)^2} + \frac{y^2}{\left(\frac{1201}{1151}\right)^2} = 1$$
\end{solution}

\begin{example}{}{}
点$P$为圆$O:x^{2}+y^{2}=1$上一动点，圆$O$在点$P$处切线交椭圆$C:\dfrac{x^{2}}{a^2}+\dfrac{y^{2}}{b^2}=1$于$A_{1},B_{1}$两点，再过点$A_{1},B_{1}$作圆$O$的另一条切线分别交椭圆$C$于点$A_{2},B_{2}$两点，依次类推；设直线$A_{n}A_{n+1},B_{n}B_{n+1}$相交于点$Q$，求动点$Q$的轨迹方程：
\end{example}
\begin{solution}
首先，我们理清题目中递推切线的生成逻辑。记动点 $P$ 为 $T_0$。切线 $A_k A_{k+1}$ 是圆 $O$ 在某点 $T_k$ 处的切线。点 $A_k$ 是切线 $T_{k-1}$ 和切线 $T_k$ 的交点。因为 $A_k$ 在椭圆 $C: \frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$ 上，这意味着从椭圆上一点引出的两条圆的切线，其切点弦 $T_{k-1} T_k$ 正好是点 $A_k$ 关于圆的极线。
根据高中最基础的判别式法，我们来找这条极线满足的条件：设极线方程为 $ux + vy = 1$。它是由椭圆上一点 $A_k(u, v)$ 生成的，所以必有 $\frac{u^2}{a^2} + \frac{v^2}{b^2} = 1$。
这说明，所有相邻切点相连的弦 $T_{k-1} T_k$，都必须与一个对偶椭圆 $C^*: a^2 x^2 + b^2 y^2 = 1$ 相切。
(注：设线 $y = \frac{1-ux}{v}$ 代入 $a^2x^2+b^2y^2=1$ 展开，令判别式 $\Delta=0$，化简后恰好就是 $\frac{u^2}{a^2} + \frac{v^2}{b^2} = 1$。这里完全是初等代数。)
现在，设圆上两相邻切点为 $T_{k-1}(\cos\theta_1, \sin\theta_1)$ 和 $T_k(\cos\theta_2, \sin\theta_2)$。它们连成的弦的方程（利用和差化积）为：
$$x \cos\frac{\theta_1+\theta_2}{2} + y \sin\frac{\theta_1+\theta_2}{2} = \cos\frac{\theta_1-\theta_2}{2}$$
令 $\sigma = \frac{\theta_1+\theta_2}{2}, \delta = \frac{\theta_1-\theta_2}{2}$。将方程化为截距式 $ux+vy=1$，则 $u = \frac{\cos\sigma}{\cos\delta}, v = \frac{\sin\sigma}{\cos\delta}$。将其代入相切条件 $\frac{u^2}{a^2} + \frac{v^2}{b^2} = 1$：
$$\frac{\cos^2\sigma}{a^2 \cos^2\delta} + \frac{\sin^2\sigma}{b^2 \cos^2\delta} = 1 \implies b^2 \cos^2\sigma + a^2 \sin^2\sigma = a^2 b^2 \cos^2\delta$$
利用高中二倍角公式 $\cos^2 x = \frac{1+\cos 2x}{2}, \sin^2 x = \frac{1-\cos 2x}{2}$ 降次展开：
\begin{align*}
& b^2 \frac{1+\cos(\theta_1+\theta_2)}{2} + a^2 \frac{1-\cos(\theta_1+\theta_2)}{2} = a^2 b^2 \frac{1+\cos(\theta_1-\theta_2)}{2}\\
\Leftrightarrow & (a^2+b^2) + (b^2-a^2)\cos(\theta_1+\theta_2) = a^2 b^2 + a^2 b^2\cos(\theta_1-\theta_2)\\
\Leftrightarrow & (a^2+b^2-a^2 b^2) + (b^2-a^2 - a^2 b^2)x_1 x_2 - (b^2-a^2 + a^2 b^2)y_1 y_2 = 0\\
\Leftrightarrow & \frac{a^2 b^2 + a^2 - b^2}{a^2 b^2 - a^2 - b^2} x_1 x_2 + \frac{a^2 b^2 - a^2 + b^2}{a^2 b^2 - a^2 - b^2} y_1 y_2 + 1 = 0
\end{align*}
这是整个推导中最伟大的第一步！我们得到了任意两相邻切点 $T_{k-1}, T_k$ 必须满足的初始算子：
\[\alpha_1 x_{k-1} x_k + \beta_1 y_{k-1} y_k + 1 = 0,~\alpha_1 = \frac{a^2 b^2 + a^2 - b^2}{a^2 b^2 - a^2 - b^2}, \beta_1 = \frac{a^2 b^2 - a^2 + b^2}{a^2 b^2 - a^2 - b^2}\]
现在，我们来看第 $n$ 步的切点是如何关联的。
\begin{itemize}
\item 已知 $T_{k-1}$ 和 $T_k$ 满足 $\alpha_n x_{k-1} x_k + \beta_n y_{k-1} y_k + 1 = 0$。
\item 且 $T_k$ 和 $T_{k+1}$ 满足 $\alpha_1 x_k x_{k+1} + \beta_1 y_k y_{k+1} + 1 = 0$。
\item 我们要消去中间点 $T_k(x_k, y_k)$，找出 $T_{k-1}$ 与 $T_{k+1}$ 的关系。
\end{itemize}
将上述两式视作关于 $x_k, y_k$ 的二元一次方程组，利用克莱姆法则直接解出：
$$x_k = \frac{\beta_n y_{k-1} - \beta_1 y_{k+1}}{\alpha_n \beta_1 x_{k-1} y_{k+1} - \alpha_1 \beta_n x_{k+1} y_{k-1}}, \quad y_k = \frac{\alpha_1 x_{k+1} - \alpha_n x_{k-1}}{\alpha_n \beta_1 x_{k-1} y_{k+1} - \alpha_1 \beta_n x_{k+1} y_{k-1}}$$
(注：为了同构对称，我们令 $\alpha_n = \alpha_1, \beta_n = \beta_1$ 演示递推的本质)将解出的 $x_k, y_k$ 代入单位圆 $x_k^2 + y_k^2 = 1$，并同乘分母平方：
$$\beta_n^2 (y_{k-1} - y_{k+1})^2 + \alpha_n^2 (x_{k-1} - x_{k+1})^2 = \alpha_n^2 \beta_n^2 (x_{k-1} y_{k+1} - x_{k+1} y_{k-1})^2$$
我们再次将 $T_{k-1}, T_{k+1}$ 还原为极角参数，并利用和差化积：
\[(y_{k-1} - y_{k+1})^2 = 4 \sin^2\delta \cos^2\sigma,~(x_{k-1} - x_{k+1})^2 = 4 \sin^2\delta \sin^2\sigma,~(x_{k-1} y_{k+1} - x_{k+1} y_{k-1})^2 = \sin^2(2\delta) \]
把这些替换进去，等式两边的 $4 \sin^2\delta$ 完美约掉！方程瞬间坍缩为：
$${\beta_n^2 \cos^2\sigma + \alpha_n^2 \sin^2\sigma = \alpha_n^2 \beta_n^2 \cos^2\delta}$$
这个方程的结构，与我们上面得到的原始方程 \[b^2 \cos^2\sigma + a^2 \sin^2\sigma = a^2 b^2 \cos^2\delta\]
一模一样。唯一的区别仅仅是：$a^2$ 被替换成了 $\alpha_n^2$，$b^2$ 被替换成了 $\beta_n^2$。既然代数结构完全同构，那它展开后的算子必然也完全同构！因此，消去中间点后，$T_{k-1}$ 与 $T_{k+1}$ 的关系式必然是：
$${\alpha_{n+1} x_{k-1} x_{k+1} + \beta_{n+1} y_{k-1} y_{k+1} + 1 = 0}$$
其中系数的递推公式直接由同构生成：$${\alpha_{n+1} = \frac{\alpha_n^2 \beta_n^2 + \alpha_n^2 - \beta_n^2}{\alpha_n^2 \beta_n^2 - \alpha_n^2 - \beta_n^2}, \quad \beta_{n+1} = \frac{\alpha_n^2 \beta_n^2 - \alpha_n^2 + \beta_n^2}{\alpha_n^2 \beta_n^2 - \alpha_n^2 - \beta_n^2}}$$
题目中分为了两条链路：
\begin{itemize}
\item 从 $P$ 出发，经过 $A$ 分支的 $n$ 次相切到达 $T_{nA}$；
\item 经过 $B$ 分支的 $n$ 次相切到达 $T_{nB}$。
\item 根据我们刚才推导的绝对同构递推，无论走哪条分支，第 $n$ 个切点与初始点 $P(x_P, y_P)$ 都必须满足第 $n$ 阶算子：$$\alpha_n x_P x_{nA} + \beta_n y_P y_{nA} + 1 = 0 \quad \text{且} \quad \alpha_n x_P x_{nB} + \beta_n y_P y_{nB} + 1 = 0$$
\item 因为点 $T_{nA}$ 和 $T_{nB}$ 的坐标同时满足上面这个一次方程，这说明连接 $T_{nA}$ 和 $T_{nB}$ 的直线（也就是第 $n$ 阶的切点弦），其方程被死死锁定为：$$\alpha_n x_P x + \beta_n y_P y = -1 \implies (-\alpha_n x_P) x + (-\beta_n y_P) y = 1$$
\item 动点 $Q$ 是圆 $O$ 在 $T_{nA}$ 和 $T_{nB}$ 处的两条切线（即 $A_n A_{n+1}$ 和 $B_n B_{n+1}$）的交点。根据算子的极点极线定义，切线交点 $Q(x_Q, y_Q)$ 就是切点弦对应的极点！
\item 所以，弦的方程 $x_Q x + y_Q y = 1$ 与上面锁死的方程必须是同一条直线。直接提取系数：$$x_Q = -\alpha_n x_P \implies x_P = -\frac{x_Q}{\alpha_n},~~y_Q = -\beta_n y_P \implies y_P = -\frac{y_Q}{\beta_n}$$
\item 动点 $P$ 活生生在单位圆 $x_P^2 + y_P^2 = 1$ 上。把反解出的坐标毫不留情地砸进去：$$\left(-\frac{x_Q}{\alpha_n}\right)^2 + \left(-\frac{y_Q}{\beta_n}\right)^2 = 1$$
\end{itemize}
整理上述方程，动点 $Q$ 的轨迹标准方程显现:
$${\frac{x^2}{\alpha_n^2} + \frac{y^2}{\beta_n^2} = 1},~~\alpha_1 = \frac{a^2 b^2 + a^2 - b^2}{a^2 b^2 - a^2 - b^2}, ~~\beta_1 = \frac{a^2 b^2 - a^2 + b^2}{a^2 b^2 - a^2 - b^2}$$
其中，系数 $\alpha_n, \beta_n$ 由以下递推公式绝对定义（初值为 $n=1$）：
$$\alpha_{n} = \frac{\alpha_{n-1}^2 \beta_{n-1}^2 + \alpha_{n-1}^2 - \beta_{n-1}^2}{\alpha_{n-1}^2 \beta_{n-1}^2 - \alpha_{n-1}^2 - \beta_{n-1}^2},~~ \beta_{n} = \frac{\alpha_{n-1}^2 \beta_{n-1}^2 - \alpha_{n-1}^2 + \beta_{n-1}^2}{\alpha_{n-1}^2 \beta_{n-1}^2 - \alpha_{n-1}^2 - \beta_{n-1}^2}$$
\end{solution}

\begin{example}{}{}
已知椭圆O:$\frac{x^{2}}{2}+y^{2}=1$, 圆C: $(x-3)^{2}+(y-2)^{2}=1$, P为椭圆O上一动点，过点P作圆C的两条切线PM, PN分别交椭圆O于A,B两点，求证：存在一定点Q使得$k_{AB}+k_{PQ}$等于定值；
\end{example}
\begin{solution}
已知动点 $P(x_0, y_0)$ 在椭圆 $O: \frac{x^2}{2} + y^2 = 1$ 上。$E_O(P,P) = \frac{x_0^2}{2} + y_0^2 = 1$。引入向量 $\vec{v} = (X, Y) = (x-x_0, y-y_0)$，隐式平移原椭圆方程，得到关于 $\vec{v}$ 的算子方程：$$E_O(\vec{v}, \vec{v}) + 2E_O(P, \vec{v}) = 0$$
现在处理圆 $C: (x-3)^2 + (y-2)^2 = 1$。在以 $P$ 为原点的局部坐标系 $\vec{v}$ 中，圆心坐标变成了 $(d_x, d_y) = (3-x_0, 2-y_0)$，半径 $r = 1$。从原点（即动点 $P$）引出的两条切线。在以 $P$ 为原点的局部坐标系中，我们要找的是两条过原点 $(0,0)$ 的直线，它们与圆心为 $(d_x, d_y)$、半径为 $r=1$ 的圆相切。既然直线过原点，用最一般的一般式方程$AX + BY = 0$，则
\[\frac{|A d_x + B d_y|}{\sqrt{A^2 + B^2}} = 1\Leftrightarrow (A d_x + B d_y)^2 = A^2 + B^2\]
已知点 $(X, Y)$ 在直线 $AX + BY = 0$ 上，这意味着向量 $(X, Y)$ 与法向量 $(A, B)$ 是垂直的（点积为零）。因此，我们可以直接令直线的法向量为：$A = Y, \quad B = -X$，把这个代换关系直接砸进刚才平方后的距离公式中，原本孤立的法向量 $A, B$ 瞬间被替换成了动点坐标 $X, Y$：
$$(Y d_x - X d_y)^2 = Y^2 + (-X)^2\Leftrightarrow(d_y^2 - 1)X^2 - 2d_x d_y XY + (d_x^2 - 1)Y^2 = 0$$
设我们要找的割线 $AB$ 局部线性方程为 $L(\vec{v}) + 1 = 0$（即 $uX + vY + 1 = 0$）。将切线对 $H(\vec{v})$、椭圆 $E_O$ 和割线 $L(\vec{v})$ 进行算子缝合，必然存在一个全局缩放常数 $\mu$，使得以下多项式恒等式对空间任意探测向量 $\vec{v}$ 绝对成立：$$E_O(\vec{v}, \vec{v}) - \mu H(\vec{v}) \equiv 2E_O(P, \vec{v}) \cdot L(\vec{v})$$
为了解出 $\mu$，用切向探针 $\vec{t} = (2y_0, -x_0)$，使得 $2E_O(P, \vec{t}) = x_0(2y_0) + 2y_0(-x_0) = 0$。代入恒等式得到 $\mu = \frac{E_O(\vec{t}, \vec{t})}{H(\vec{t})}$。
\begin{itemize}
\item 分子：$$E_O(\vec{t}, \vec{t}) = \frac{(2y_0)^2}{2} + (-x_0)^2 = 2y_0^2 + x_0^2 = 2\left(\frac{x_0^2}{2} + y_0^2\right) =  2$$
\item 分母：将 $\vec{t}$ 塞入 $H(\vec{v})$ 中：$$H(\vec{t}) = (d_y^2 - 1)(2y_0)^2 - 2d_x d_y (2y_0)(-x_0) + (d_x^2 - 1)(-x_0)^2$$将含 $d_x, d_y$ 的项放在一起配成完全平方：$$= [4y_0^2 d_y^2 + 4x_0 y_0 d_x d_y + x_0^2 d_x^2] - (4y_0^2 + x_0^2) = (2y_0 d_y + x_0 d_x)^2 - (x_0^2 + 4y_0^2)$$把 $d_x = 3-x_0, d_y = 2-y_0$ 代入方括号内：$$2y_0(2-y_0) + x_0(3-x_0) = 4y_0 - 2y_0^2 + 3x_0 - x_0^2 = 3x_0 + 4y_0 - (x_0^2 + 2y_0^2)$$由于 $\frac{x_0^2}{2} + y_0^2 = 1 \implies x_0^2 + 2y_0^2 = 2$，方括号内化简为 $3x_0 + 4y_0 - 2$！另外，常数项 $x_0^2 + 4y_0^2 = (x_0^2 + 2y_0^2) + 2y_0^2 = 2 + 2y_0^2$。所以分母 $H(\vec{t}) = (3x_0 + 4y_0 - 2)^2 - 2y_0^2 - 2$。
\item 定义分母 $D_\mu = (3x_0 + 4y_0 - 2)^2 - 2y_0^2 - 2$。$\mu = \frac{2}{D_\mu}$。
\end{itemize}
代入横向基底 $\vec{e_1} = (1,0)$ 到恒等式：
$$\frac{1}{2} - \mu(d_y^2 - 1) = x_0 u \implies x_0 u = \frac{D_\mu - 4((2-y_0)^2 - 1)}{2 D_\mu} = \frac{D_\mu - 4(y_0^2 - 4y_0 + 3)}{2 D_\mu}$$
展开分子，并利用 $x_0^2 + 2y_0^2 = 2 \implies 8y_0^2 = 8 - 4x_0^2$ 降次：
\begin{align*}
\text{分子}& = (3x_0 + 4y_0 - 2)^2 - 2y_0^2 - 2 - 4y_0^2 + 16y_0 - 12 = (3x_0 + 4y_0 - 2)^2 - 6y_0^2 + 16y_0 - 14\\
&=9x_0^2 + 16y_0^2 + 4 + 24x_0 y_0 - 12x_0 - 16y_0 - 6y_0^2 + 16y_0 - 14 \\
&=9x_0^2 + 10y_0^2 + 24x_0 y_0 - 12x_0 - 10,~~\text{代入}10y_0^2 - 10 = -5x_0^2\\
&=9x_0^2 - 5x_0^2 + 24x_0 y_0 - 12x_0 = 4x_0^2 + 24x_0 y_0 - 12x_0 = 4x_0(x_0 + 6y_0 - 3)\\
\implies u &= \frac{2(x_0 + 6y_0 - 3)}{D_\mu}
\end{align*}
代入纵向基底 $\vec{e_2} = (0,1)$ 到恒等式：
$$1 - \mu(d_x^2 - 1) = 2y_0 v \implies 2y_0 v = \frac{D_\mu - 2((3-x_0)^2 - 1)}{D_\mu} = \frac{D_\mu - 2(x_0^2 - 6x_0 + 8)}{D_\mu}$$
同样展开分子，并利用 $6x_0^2 + 12y_0^2 = 12$ 降次：
\begin{align*}
\text{分子} &= (3x_0 + 4y_0 - 2)^2 - 2y_0^2 - 2 - 2x_0^2 + 12x_0 - 16\\
&= 9x_0^2 + 16y_0^2 + 4 + 24x_0 y_0 - 12x_0 - 16y_0 - 2y_0^2 - 2 - 2x_0^2 + 12x_0 - 16 \\
&= 7x_0^2 + 14y_0^2 + 24x_0 y_0 - 16y_0 - 14,~~\text{代入}7x_0^2 + 14y_0^2 - 14 = 0\\
&= 24x_0 y_0 - 16y_0 = 8y_0(3x_0 - 2)\implies v = \frac{4(3x_0 - 2)}{D_\mu}
\end{align*}
割线 $AB$ 在局部坐标系下的斜率 $k_{AB}$ 直接爆出（恐怖的 $D_\mu$ 瞬间约去）：
$$k_{AB} = -\frac{u}{v} = - \frac{2(x_0 + 6y_0 - 3)}{4(3x_0 - 2)} = {\frac{-x_0 - 6y_0 + 3}{6x_0 - 4}}$$
题目要求找到定点 $Q(x_q, y_q)$ 和定值 $C$，使得 $k_{AB} + k_{PQ} = C$。将 $k_{PQ} = \frac{y_0 - y_q}{x_0 - x_q}$ 代入，我们得到恒等关系：
$$k_{AB} = \frac{C(x_0 - x_q) - (y_0 - y_q)}{x_0 - x_q} = \frac{C x_0 - y_0 + K}{x_0 - x_q} \quad (\text{令 } K = y_q - C x_q)$$
将我们算出的 $k_{AB} = {\frac{-x_0 - 6y_0 + 3}{6x_0 - 4}}$ 强行等同过去，进行交叉相乘：
$$\frac{-x_0 - 6y_0 + 3}{6x_0 - 4} \equiv \frac{C x_0 - y_0 + K}{x_0 - x_q}\implies(-x_0 - 6y_0 + 3)(x_0 - x_q) \equiv (6x_0 - 4)(C x_0 - y_0 + K)$$
把等式两边完全展开：
\begin{itemize}
\item 左边 $= -x_0^2 - 6x_0 y_0 + 3x_0 + x_q x_0 + 6x_q y_0 - 3x_q$
\item 右边 $= 6C x_0^2 - 6x_0 y_0 + 6K x_0 - 4C x_0 + 4y_0 - 4K$
\end{itemize}
方程两边的 $x_0 y_0$ 交叉项系数相等，对比剩下的同类项系数：
\begin{itemize}
\item 比对 $x_0^2$：$-1 = 6C \implies \mathbf{C = -\frac{1}{6}}$ (定值直接浮出水面！)
\item 比对 $y_0$：$6x_q = 4 \implies \mathbf{x_q = \frac{2}{3}}$
\item 比对常数项：$-3x_q = -4K \implies -3\left(\frac{2}{3}\right) = -4K \implies \mathbf{K = \frac{1}{2}}$
\item 闭环检验 $x_0$：左边 $3 + x_q = 3 + \frac{2}{3} = \frac{11}{3}$；右边 $6K - 4C = 6\left(\frac{1}{2}\right) - 4\left(-\frac{1}{6}\right) = 3 + \frac{2}{3} = \frac{11}{3}$。相等。
\item 解出纵坐标 $y_q$：$$K = y_q - C x_q \implies \frac{1}{2} = y_q - \left(-\frac{1}{6}\right)\left(\frac{2}{3}\right) \implies y_q = \frac{1}{2} - \frac{1}{9} = \mathbf{\frac{7}{18}}$$
\end{itemize}
存在绝对定点 $Q\left(\frac{2}{3}, \frac{7}{18}\right)$，使得 $k_{AB} + k_{PQ}$ 永远等于定值 $-\frac{1}{6}$。
\end{solution}

\begin{example}{}{}
点P为椭圆O：$\frac {x^2}4+ y^2$= 1上一动点，点$M(1,\frac{\sqrt{39}-2\sqrt{3}}6)$,点$N$与点$M$关于$x$轴对称，直线$PM$和直线$PN$分别交椭圆$O$于$A，B$两点，圆$E$：$x^2+y^2-\frac32x-\frac14=0$，求证：直线$AB$与圆$E$相切
\end{example}
\begin{solution}
设动点 $P(x_0, y_0)$ 在椭圆 $O: \frac{x^2}{4} + y^2 = 1$ 上。引入探测向量 $\vec{v} = (X, Y) = (x-x_0, y-y_0)$，将椭圆方程在 $P$ 点隐式展开并截断：
$$E_O(\vec{v}, \vec{v}) + 2E_O(P, \vec{v}) = 0$$
点 $M$ 和 $N$ 关于 $x$ 轴对称，设坐标为 $M(m, n), N(m, -n)$。在以 $P$ 为原点的局部向量下，它们对应的向量为 $(m-x_0, n-y_0)$ 和 $(m-x_0, -n-y_0)$。两弦 $PM, PN$ 斜率分别为：
斜率之和 $S$ 与斜率之积 $K$：
$$S = k_1 + k_2 = \frac{-2y_0}{m-x_0} = \frac{2y_0}{x_0-m},~~K = k_1 k_2 = \frac{-(n^2 - y_0^2)}{(m-x_0)^2} = \frac{y_0^2 - n^2}{(x_0-m)^2}$$
构造斜率纯二次型 $H(\vec{v}) = Y^2 - S \cdot XY + K \cdot X^2$。它与隐式齐次方程必定只差一个全局常数倍 $\mu$。设割线 $AB$ 局部线性方程为 $L(\vec{v}) + 1 = 0$（即 $uX+vY+1=0$），得到恒等式：$$E_O(\vec{v}, \vec{v}) - \mu H(\vec{v}) \equiv 2E_O(P, \vec{v}) \cdot L(\vec{v})$$
令正交探针 $\vec{t} = (4y_0, -x_0)$，使得切向算子 $E_O(P, \vec{t}) = \frac{x_0(4y_0)}{4} + y_0(-x_0) = 0$。代入恒等式得 $\mu = \frac{E_O(\vec{t}, \vec{t})}{H(\vec{t})}$。
\begin{itemize}
\item 分子：$E_O(\vec{t}, \vec{t}) = \frac{16y_0^2}{4} + x_0^2 = 4y_0^2 + x_0^2 = 4\left(\frac{x_0^2}{4} + y_0^2\right) = \mathbf{4}$
\item 分母：将 $\vec{t}$ 塞入 $H(\vec{v})$：
\begin{align*}
H(\vec{t}) &= (-x_0)^2 - S(4y_0)(-x_0) + K(4y_0)^2 = x_0^2 + 4S x_0 y_0 + 16K y_0^2\\
&=  x_0^2 + \frac{8x_0 y_0^2}{x_0-m} + \frac{16y_0^2(y_0^2 - n^2)}{(x_0-m)^2}=x_0^2 + \frac{2x_0(4-x_0^2)}{x_0-m} + \frac{4(4-x_0^2) \left( \frac{4-x_0^2}{4} - n^2 \right)}{(x_0-m)^2}\\
&=\frac{x_0^2(x_0^2 - 2mx_0 + m^2) + (8x_0 - 2x_0^3)(x_0 - m) + (4-x_0^2)^2 - 4n^2(4-x_0^2)}{(x_0-m)^2}\\
&= \frac{x_0^4 - 2mx_0^3 + m^2 x_0^2 + 8x_0^2 - 8mx_0 - 2x_0^4 + 2mx_0^3 + 16 - 8x_0^2 + x_0^4 - 16n^2 + 4n^2 x_0^2}{(x_0-m)^2}\\
&=\frac{(m^2 + 4n^2)x_0^2 - 8mx_0 + 16 - 16n^2}{(x_0-m)^2}
\end{align*}
定义 $D_0 = (m^2 + 4n^2)x_0^2 - 8mx_0 + 16 - 16n^2$。于是$\mu = \dfrac{4(x_0-m)^2}{D_0}$。
\end{itemize}
代入横向基底 $\vec{e_1} = (1,0)$ 到恒等式：
\begin{align*}
& \frac{1}{4} - \mu K = 2\left(\frac{x_0}{4}\right)u \implies 2x_0 u = 1 - 4\mu K\\
& 4\mu K = 4 \frac{4(x_0-m)^2}{D_0} \frac{y_0^2 - n^2}{(x_0-m)^2} = \frac{16y_0^2 - 16n^2}{D_0} = \frac{4(4-x_0^2) - 16n^2}{D_0} = \frac{16 - 4x_0^2 - 16n^2}{D_0}\\
\Leftrightarrow & 2x_0 u = 1 - \frac{16 - 4x_0^2 - 16n^2}{D_0} = \frac{(m^2 + 4n^2)x_0^2 - 8mx_0 + 16 - 16n^2 - 16 + 4x_0^2 + 16n^2}{D_0}\\
\implies & u = \frac{(m^2 + 4n^2 + 4)x_0 - 8m}{2 D_0}
\end{align*}
代入纵向基底 $\vec{e_2} = (0,1)$ 到恒等式：
\begin{align*}
& 1 - \mu \cdot 1 = 2y_0 v \implies 2y_0 v = 1 - \frac{4(x_0-m)^2}{D_0} = \frac{D_0 - 4(x_0^2 - 2mx_0 + m^2)}{D_0}\\
& = \frac{(m^2 + 4n^2 - 4)x_0^2 + 16 - 16n^2 - 4m^2}{D_0}=\frac{(m^2 + 4n^2 - 4)(4 - 4y_0^2) + 16 - 16n^2 - 4m^2 }{D_0}\\
& = \frac{4m^2 + 16n^2 - 16 - 4(m^2 + 4n^2 - 4)y_0^2 + 16 - 16n^2 - 4m^2}{D_0}\\
\implies & v = \frac{-2(m^2 + 4n^2 - 4)y_0}{D_0}
\end{align*}
将局部直线 $u(x-x_0) + v(y-y_0) + 1 = 0$ 展开，计算常数项：
\begin{align*}
ux_0 + vy_0 &= \frac{(m^2 + 4n^2 + 4)x_0^2 - 8mx_0}{2 D_0} + \frac{-2(m^2 + 4n^2 - 4)y_0^2}{D_0}\\
&=  \frac{(m^2 + 4n^2 + 4)x_0^2 - 8mx_0}{2 D_0} +\frac{-(m^2 + 4n^2 - 4)(4-x_0^2)}{2 D_0}\\
&=  \frac{(m^2 + 4n^2 + 4)x_0^2 - 8mx_0}{2 D_0} +\frac{x_0^2(m^2 + 4n^2 - 4)-4(m^2 + 4n^2 - 4)}{2D_0}\\
&= \frac{2(m^2 + 4n^2)x_0^2 - 8mx_0 - 4m^2 - 16n^2 + 16}{2 D_0}\\
1 - (ux_0 + vy_0) &= \frac{-4mx_0 + 2m^2 - 8n^2 + 8}{D_0}
\end{align*}
全局直线方程为 $ux + vy + 1 - ux_0 - vy_0 = 0$。同乘 $2 D_0$ 去分母：
$$[(m^2 + 4n^2 + 4)x_0 - 8m]x - 4(m^2 + 4n^2 - 4)y_0 y + (-8mx_0 + 4m^2 - 16n^2 + 16) = 0$$
现在，把题目给定的坐标 $m = 1, n = \frac{\sqrt{39} - 2\sqrt{3}}{6} = \frac{\sqrt{3}(\sqrt{13} - 2)}{6}$ 砸进去！算一下核心常数 $n^2$：
$$n^2 = \frac{3(13 - 4\sqrt{13} + 4)}{36} = \frac{17 - 4\sqrt{13}}{12},~~m^2 + 4n^2 = 1 + \frac{17 - 4\sqrt{13}}{3} = \frac{20 - 4\sqrt{13}}{3}$$
\begin{itemize}
\item $x$ 系数块：$m^2 + 4n^2 + 4 = \frac{32 - 4\sqrt{13}}{3}$
\item $y$ 系数块：$m^2 + 4n^2 - 4 = \frac{8 - 4\sqrt{13}}{3}$
\item 常数项块：$-8x_0 + 4 - 16\left(\frac{17 - 4\sqrt{13}}{12}\right) + 16 = -8x_0 + 20 - \frac{68 - 16\sqrt{13}}{3} = -8x_0 + \frac{-8 + 16\sqrt{13}}{3}$
\end{itemize}
把这三块代回直线方程，全体同乘 3 去分母：$$[ (32 - 4\sqrt{13})x_0 - 24 ]x - 4(8 - 4\sqrt{13})y_0 y + (-24x_0 - 8 + 16\sqrt{13}) = 0$$再全体除以 4，得到最终剔透的割线 $AB$ 方程：$$A x + B y + C_0 = 0 \implies [(8 - \sqrt{13})x_0 - 6]x + 4(\sqrt{13} - 2)y_0 y + (-6x_0 - 2 + 4\sqrt{13}) = 0$$
圆 $E: x^2 + y^2 - \frac{3}{2}x - \frac{1}{4} = 0 \implies \left(x - \frac{3}{4}\right)^2 + y^2 = \frac{13}{16}$。
圆心为 $C\left(\frac{3}{4}, 0\right)$，半径 $r = \frac{\sqrt{13}}{4}$。
计算圆心到直线 $AB$ 的距离
\begin{align*}
d &= \frac{|A \cdot \frac{3}{4} + C_0|}{\sqrt{A^2 + B^2}}= \frac{|\frac{3}{4}[(8 - \sqrt{13})x_0 - 6] - 6x_0 - 2 + 4\sqrt{13}|}{\sqrt{[(8 - \sqrt{13})x_0 - 6]^2+16(13 - 4\sqrt{13} + 4)y_0^2 }}\\
&=\frac{6x_0 - \frac{3\sqrt{13}}{4}x_0 - \frac{9}{2} - 6x_0 - 2 + 4\sqrt{13}}{\sqrt{(77 - 16\sqrt{13})x_0^2 - 12(8 - \sqrt{13})x_0 + 36+ 16(17 - 4\sqrt{13}) \left(1 - \frac{x_0^2}{4}\right) }}\\
&=\frac{\frac{\sqrt{13}}{4} \left( -3x_0 + 16 - 2\sqrt{13} \right)}{\sqrt{(77 - 16\sqrt{13})x_0^2- 12(8 - \sqrt{13})x_0 + 36+(272 - 64\sqrt{13}) - 4(17 - 4\sqrt{13})x_0^2 }}\\
&=\frac{\frac{\sqrt{13}}{4} \left( -3x_0 + 16 - 2\sqrt{13} \right)}{\sqrt{ 9x_0^2 - 12(8 - \sqrt{13})x_0 + (308 - 64\sqrt{13})}}\\
&= \frac{\frac{\sqrt{13}}{4} |-3x_0 + 16 - 2\sqrt{13}|}{|3x_0 - 16 + 2\sqrt{13}|} = {\frac{\sqrt{13}}{4}}
\end{align*}
距离 $d$ 绝对等于圆 $E$ 的半径 $r$。直线 $AB$ 与圆 $E$ 相切证毕。
\end{solution}
