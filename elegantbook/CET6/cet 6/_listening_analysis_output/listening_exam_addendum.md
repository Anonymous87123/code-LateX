# 六级听力补充速记卡

这份补充基于已抽取的 433 个答案定位点和 419 道原卷听力选项。原报告讲的是整体规律；这里更偏考场临场使用。

## 1. 先别找答案，先判断题目会问什么

播放前看到选项时，优先做这 4 个判断：

1. 四个选项主语是否一样：如果都是 `he/she/they`，多半问行为、态度、原因或身份。
2. 四个选项动词是否不一样：动词就是差异点，听的时候盯动作关系。
3. 四个选项是否都有研究、实验、调查：等 `study / research / found / showed / suggest` 附近。
4. 四个选项是否都是建议或计划：等 `should / need to / plan / intend / make sure / always`。

最危险的做法是只圈名词。六级干扰项最爱复用原文名词，但把主语、动作、时间、因果改掉。

## 2. 没听全时的优先级

如果只记住几个词，按这个顺序救题：

1. 先选逻辑完整的选项：主语、动作、对象都能对上。
2. 再选限定词准确的选项：`short / chronic / moderate / only / slightly / more than` 这类词不能错。
3. 再选转折后的选项：`but / however / actually / instead / rather than` 后面权重大。
4. 最后才考虑原词复现：只因为听到某个词就选，通常风险最大。

一个实用判断：两个选项都听到过时，较可能正确的是“改写原文关系”的那个，不是“搬运原文名词”的那个。

## 3. 三个 section 的临场节奏

Section A 长对话：

- Q1/Q5：开头身份、关系、背景。
- Q4/Q8：结尾计划、结论、下一步。
- 听到对方问 `why / how / what are your plans` 后，下一句很像答案。
- 女士题跟女士立场，男士题跟男士立场；不要把另一人的评价拿来答。

Section B 短文：

- 首题常在开头主题句或争议句。
- 中间题常在研究发现、原因、数字比较处。
- 末题常在转折或建议处。
- 特别防“对象错位”：动物实验、人类结论、消费者建议、研究者观点不要互换。

Section C 讲座：

- 首题听 `Today... / focus on... / what can X tell us...`。
- 中间题听例子前后的总括句，不要只抓例子名。
- 末题听 `essentially / in short / instead / in other words / finally`。
- 抽象选项往往比具体例子更像答案，尤其是主旨题和结尾题。

## 4. 选项中这些词要警惕

从选项表面看，高频差异词包括 `more`、`only`、`all`、`not/no`、`less`、`much`。它们不是一定错，但非常容易成为陷阱。

- `only`：听原文是否真的排他。原文说“主要/略微/常常”，选项写“only”可能过强。
- `all / never / always`：绝对词通常要谨慎，除非原文明确说。
- `more / less / higher / lower`：比较对象必须一致。
- `not / no / cannot`：注意是否是说话人的否定、原文的否定，还是选项偷换否定。
- `main / major`：主旨题可以选，细节题可能范围过大。

## 5. 十类常见干扰项

1. 原词复现但关系错：听到 `proposal`，选项却把动作写错。
2. 角色错位：男士说的观点写成女士观点。
3. 时间错位：过去失败、现在成功、未来计划被混在一起。
4. 因果反转：A 导致 B 写成 B 导致 A。
5. 程度放大：`slightly higher` 写成 `excelled`。
6. 范围扩大：一个研究样本写成所有人群。
7. 例子当主旨：Boston、老鼠、医院、某个 app 只是例子。
8. 前半句陷阱：`not... but...` 前半句进干扰项。
9. 并列项偷换：利润、工资、产品、工作控制权同段出现，但题只问其中一个。
10. 相邻题串线：Q10 的原因拿去答 Q11 的结论。

## 6. 最该训练的不是全文听写，而是题点句反应

练习时可以用这个流程：

1. 只看选项，写出每题可能问什么。
2. 听一遍音频，只在听到功能词时记 1-3 个词。
3. 做题后不要先看中文解析，先回到原文找题点句。
4. 对每个错题标一个错因：角色、时间、限定、因果、范围、例子、串题。

如果目标是提高正确率，复盘错因比反复泛听更快。六级听力不是平均考每一句，而是集中考“功能句”。

## 7. 一句话临场口诀

开头听身份和主题，中间听原因、研究和数字，转折后更重要，结尾听总结和建议；选项看动词和限定，不被原词复现牵走。
