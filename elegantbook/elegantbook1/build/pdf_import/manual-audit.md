# PDF 人工审计记录

## 审计标准

- 每份 PDF 按文件名顺序处理；题面以页面渲染图为准，`pdftotext -layout` 文本仅作检索辅助。
- 每题核对三项：题干与条件、选项或小问、`solution` 中答案与主要推导。
- 完全相同题可不重复录入，但必须记录被哪一个现有标签覆盖。
- 题意相同但源卷选项排列、数值、表述不同的选择题，保留源卷版本，避免漏源。
- 每份 PDF 审计完成后编译 `elegantbook1.tex`。
