   定理 6.5 (整体 n 次方先开根)
                         √
  若通项有明显的整体 n 次方，先算 ρn = n an ，若 ρn → q < 1，则可取 q < r < 1，充分大时
  ρn ≤ r，于是 an = ρnn ≤ rn ，这说明 an 至少有几何衰减，故级数收敛。若 ρn → q > 1，则充分大
  时 an > 1，通项不趋于 0，故发散。当 ρn → 1 时，要继续把根值信息转成对数阶 ln an = n ln ρn ，
  目的是利用 ln ρn , (ρn → 1) 细化判断 ρn → 1 的速度。
