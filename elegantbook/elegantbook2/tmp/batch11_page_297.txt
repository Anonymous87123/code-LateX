         定理 3.26 (极值的充分条件：Hesse 矩阵判别法)
        设 f 在 p0 的某邻域内具有连续的二阶偏导数，且 ∇f (p0 ) = 0（即 p0 为驻点）
                                                        。若 Hf (p0 ) 正定，
        则 f 在 p0 处取严格极小值；若 Hf (p0 ) 负定，则 f 在 p0 处取严格极大值；若 Hf (p0 ) 不定，则 f
        在 p0 处没有极值（p0 为鞍点）。
                                                                                              ♥

         对于二元函数的实际计算，我们通常不需要去算特征值，而是使用基于顺序主子式的等价判据：
    性质 3.13 Hesse 矩阵的实用判别法 记 a := fxx (p0 ), b := fxy (p0 ) = fyx (p0 ), c := fyy (p0 )，以及判别式
    ∆ := ac − b2 。则：

                          情形       条件             Hf (p0 ) 性质   结论

                           (i)     a>0且∆>0        正定            严格极小值点

                           (ii)    a<0且∆>0        负定            严格极大值点

                           (iii)   ∆<0            不定            鞍点（非极值）

                           (iv)    ∆=0            退化            无法判断

                                          ，这也是为什么这个判别法被称
    的 Taylor 项或其他方法。此时三种结果均有可能出现（见下文例 3.64）
    为“充分的”而非“充分必要的”。
   练习 3.64 退化的 Hesse 判别：∆ = 0 的三种可能性 讨论函数 f1 (x, y) = x2 y 2 、f2 (x, y) = −x2 y 2 、
    f3 (x, y) = x2 y 3 在原点处的极值情况。
                                         ，且其二阶偏导数均为零：a = b = c = 0，从
    解 三个函数在 (0, 0) 处均有 fx = fy = 0（直接验证即可）
    而 ∆ = 0 ——Hesse 判别法 失效。需要逐一进行具体分析。
        1. 分析函数 f1 (x, y) = x2 y 2 ：显然 f1 (x, y) = (xy)2 ⩾ 0 对所有 (x, y) 成立，且 f1 (x, y) = 0 当且仅
          当 x = 0 或 y = 0。因此在 (0, 0) 的任何邻域内，f1 (x, y) ⩾ 0 = f1 (0, 0)。(0, 0) 是 f1 的 （非严格）
                          。
          极小值点（实际上也是全局最小值点）
        2. 分析函数 f2 (x, y) = −x2 y 2 ：与 f1 相反，f2 (x, y) = −(xy)2 ⩽ 0 对所有 (x, y) 成立，且 f2 (0, 0) = 0
          为最大值。(0, 0) 是 f2 的 （非严格）极大值点。
        3. 分析函数 f3 (x, y) = x2 y 3 ：此函数在原点附近变号！
               当 y > 0 且 x 6= 0 时：f3 = x2 y 3 > 0 > f3 (0, 0) = 0；
               当 y < 0 且 x 6= 0 时：f3 = x2 y 3 < 0 < f3 (0, 0) = 0。
          因此在原点的任何邻域内既有大于 f3 (0, 0) 的点也有小于 f3 (0, 0) 的点。(0, 0) 不是极值点（属于
          广义的鞍点）。
    总结：三者均为 ∆ = 0 的退化情形，却分别对应极小值、极大值和无极值三种截然不同的结果。这充
    分说明了高阶分析的必要性。
