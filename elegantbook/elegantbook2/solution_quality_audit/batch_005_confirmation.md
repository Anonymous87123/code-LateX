## 2. 统一修正口径

本批次正文修改时统一采用以下口径：

1. 反三角函数题统一先写 `a>0`，并用前文口径“分部积分后回到实圆根式或反正切母式”。长分部积分链必须拆为 `align*`。
2. 反三角题中若使用 `\arccos` 函数族保持一致，可以说明 `\arcsin(x/a)+\arccos(x/a)=\pi/2`，但不得使用 Markdown 星号 `*(注...)*`。
3. 指数函数题统一删除 `Operator Algebra`、`D^{-1}`、算子位移定理和 `\dots`，改为前文讲解的分部积分递推、复指数法或闭式公式。
4. `\int a^x dx`、`\int x^n a^x dx` 必须注明 `a>0, a\neq1`；`\int e^{ax}dx`、`\int x^n e^{ax}dx` 使用 `1/a` 时必须注明 `a\neq0`，并补 `a=0` 退化情形。
5. `\int e^{ax}\sin bx dx` 与 `\int e^{ax}\cos bx dx` 的闭式公式需注明 `a^2+b^2\neq0`；若 `a=b=0`，分别退化为 `0` 与 `x+C`。
6. 对数函数题统一声明 `x>0`，且对 `m=-1`、`n=-1` 的分支单独处理。
7. 双曲函数题统一使用标准命令 `\sinh,\cosh,\tanh`，并补完半角法的具体积分，不保留“展开后积分”。
8. 定积分题统一写清 `m,n` 的取值范围。若公式除以 `n`，必须先排除 `n=0`；若题面等式只在正整数条件下成立，必须显式补条件。
9. 所有教学提示类文字如果只是解释背景，不得替代实际计算；主要计算仍需给出等式链。
