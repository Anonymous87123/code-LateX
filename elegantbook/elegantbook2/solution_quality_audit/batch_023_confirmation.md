## 2. 统一修正口径

本批次正文修改时统一采用以下口径：

1. 参数曲面外法向题必须先给出 `\vec r_u\times\vec r_v`，再用与位置向量点乘或几何方向判断外侧，最后写出归一化后的单位法向量。
2. 椭球、球面、半球、半径参数题统一前置 `a,b,c>0` 或 `a>0`；径向场 `p^n\vec p` 在 `n` 为实数时应说明计算区域为 `p>0`。
3. 第二类曲面积分统一先写向量场或三个有向面积元，再代入被积式；不能把法向判断、面积元替换和积分区域压成一行。
4. 图形曲面直接投影时，要明确所选参数化的叉乘方向与题设上侧、下侧、外侧是否一致；若相反，三个面积元同时变号。
5. Gauss 公式只对封闭曲面和外法向直接成立；开曲面题必须写清补面、补面法向、补面通量贡献，再回到原曲面。
6. 三维 Green 公式证明若已经按“法向导数转梯度点乘、Gauss 公式、散度乘法法则”分块完整展示，可作为保留模板，不因短 display 自动判缺陷。
7. 长证明链、封口法总通量式和球坐标/柱坐标积分链统一拆成 display 或 `align*`。
